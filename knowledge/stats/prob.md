# 確率論・確率分布 — 統計リファレンス

> 統計検定1級・数理統計レベルの確率論知識ベース。
> 開発・データ分析・機械学習における確率論の実務参照用。

---

## §1. 測度論的確率の基礎

### 1.1 確率空間 (Ω, F, P)

- **標本空間 Ω**: 起こりうる全結果の集合
- **σ-加法族（σ-algebra）F**: Ω の部分集合の族で以下を満たす
  - Ω ∈ F
  - A ∈ F ⟹ Aᶜ ∈ F（補集合で閉じる）
  - A₁, A₂, ... ∈ F ⟹ ∪ᵢAᵢ ∈ F（可算和で閉じる）
- **確率測度 P**: F → [0,1] で以下を満たす（コルモゴロフの公理）
  - P(Ω) = 1
  - P(A) ≥ 0 （非負性）
  - 互いに排反な {Aᵢ} に対し P(∪ᵢAᵢ) = Σᵢ P(Aᵢ)（σ-加法性）

### 1.2 ボレル集合族と可測関数

- **ボレル集合族 B(R)**: R 上の開集合全体から生成されるσ-加法族
  - 開集合・閉集合・可算和/積・区間すべてボレル集合
- **可測関数**: (Ω, F) → (R, B(R)) で、任意の B ∈ B(R) に対し f⁻¹(B) ∈ F
- **確率変数**: 確率空間 (Ω, F, P) 上の可測関数 X: Ω → R

### 1.3 分布関数 F(x) の性質

- **定義**: F(x) = P(X ≤ x)
- **性質**:
  - 単調非減少: x₁ < x₂ ⟹ F(x₁) ≤ F(x₂)
  - 右連続: lim_{h→0+} F(x+h) = F(x)
  - 極限: lim_{x→-∞} F(x) = 0, lim_{x→+∞} F(x) = 1
- F(x⁻) = P(X < x)、P(X = x) = F(x) - F(x⁻)

### 1.4 離散・連続・混合分布の統一

- **離散型**: F(x) が階段関数、確率質量関数 p(x) = P(X = x)
- **連続型**: F(x) が絶対連続、確率密度関数 f(x) = dF(x)/dx
- **混合型**: 離散部分と連続部分の共存（例: ゼロ膨張モデル）
- 測度論により Lebesgue-Stieltjes 積分 E[g(X)] = ∫ g(x) dF(x) で統一的に扱える

### 1.5 実務活用

- 条件付き確率 P(A|B) を厳密に定義するには測度論が必須（P(B)=0 のケース）
- 機械学習の損失関数の期待値 = Lebesgue 積分として定式化
- 混合分布（スパムフィルタの0/1 + 連続スコア）を一つの枠組みで扱える

## §2. 離散型確率分布

### 2.1 ベルヌーイ分布 Bernoulli(p)

| 項目 | 値 |
|------|-----|
| PMF | P(X=1)=p, P(X=0)=1-p |
| 期待値 | p |
| 分散 | p(1-p) |
| MGF | (1-p) + p·e^t |

### 2.2 二項分布 Bin(n, p)

| 項目 | 値 |
|------|-----|
| PMF | C(n,k)·p^k·(1-p)^{n-k} |
| 期待値 | np |
| 分散 | np(1-p) |
| MGF | ((1-p) + p·e^t)^n |
| 再生性 | Bin(n₁,p) + Bin(n₂,p) = Bin(n₁+n₂,p) |

- **正規近似**: np ≥ 5 かつ n(1-p) ≥ 5 で Bin(n,p) ≈ N(np, np(1-p))
- **ポアソン近似**: n大・p小・np=λ 一定で Bin(n,p) ≈ Poi(λ)

### 2.3 多項分布 Multi(n; p₁,...,pₖ)

- PMF: n! / (x₁!···xₖ!) · p₁^{x₁}···pₖ^{xₖ}
- E[Xᵢ] = npᵢ, Var(Xᵢ) = npᵢ(1-pᵢ), Cov(Xᵢ,Xⱼ) = -npᵢpⱼ (i≠j)
- 各周辺分布は Bin(n, pᵢ)

### 2.4 ポアソン分布 Poi(λ)

| 項目 | 値 |
|------|-----|
| PMF | e^{-λ}·λ^k / k! |
| 期待値 | λ |
| 分散 | λ |
| MGF | exp(λ(e^t - 1)) |
| 再生性 | Poi(λ₁) + Poi(λ₂) = Poi(λ₁+λ₂) |

- 平均=分散（等分散性）が特徴。過分散なら負の二項を検討
- **ポアソン過程**: 単位時間あたりλ件のイベント発生、到着間隔は Exp(λ)

### 2.5 幾何分布 Geo(p)

- **初回成功までの試行回数** X（X=1,2,...）
  - P(X=k) = (1-p)^{k-1}·p
  - E[X] = 1/p, Var(X) = (1-p)/p²
- **無記憶性**: P(X > s+t | X > s) = P(X > t)（離散分布で唯一）

### 2.6 負の二項分布 NB(r, p)

- r回成功するまでの失敗回数 Y
- P(Y=k) = C(k+r-1, k)·p^r·(1-p)^k
- E[Y] = r(1-p)/p, Var(Y) = r(1-p)/p²
- r=1 で幾何分布と一致
- **過分散ポアソン**: Var > E のカウントデータに使用

### 2.7 超幾何分布 Hyper(N, K, n)

- N個中K個が成功品、n個抽出（非復元）で成功数 X
- P(X=k) = C(K,k)·C(N-K,n-k) / C(N,n)
- E[X] = nK/N, Var(X) = n·(K/N)·(1-K/N)·(N-n)/(N-1)
- (N-n)/(N-1) は**有限母集団修正係数**
- n/N が小さいとき二項分布で近似可

### 2.8 実務活用

- **A/Bテスト**: 二項分布 → 必要サンプルサイズ n = (z_{α/2} + z_β)² · (p₁(1-p₁) + p₂(1-p₂)) / (p₁-p₂)²
- **レアイベント**: サーバー障害回数 → ポアソン分布、過分散なら負の二項
- **品質管理**: 不良品抽出 → 超幾何（非復元抽出）
- **リトライ設計**: 成功までの試行回数 → 幾何分布で期待リトライ回数 = 1/p - 1

## §3. 連続型確率分布

### 3.1 一様分布 U(a, b)

| 項目 | 値 |
|------|-----|
| PDF | 1/(b-a),  a ≤ x ≤ b |
| 期待値 | (a+b)/2 |
| 分散 | (b-a)²/12 |
| MGF | (e^{tb} - e^{ta}) / (t(b-a)) |

- 逆関数法: U ~ U(0,1) のとき F⁻¹(U) は分布関数 F を持つ確率変数

### 3.2 指数分布 Exp(λ)

| 項目 | 値 |
|------|-----|
| PDF | λ·e^{-λx},  x ≥ 0 |
| CDF | 1 - e^{-λx} |
| 期待値 | 1/λ |
| 分散 | 1/λ² |
| MGF | λ/(λ-t),  t < λ |

- **無記憶性**: P(X > s+t | X > s) = P(X > t)（連続分布で唯一）
- ポアソン過程の到着間隔
- ハザード関数 h(t) = λ（一定） → 故障率が時間に依存しない

### 3.3 ガンマ分布 Gamma(α, β)

| 項目 | 値 |
|------|-----|
| PDF | β^α / Γ(α) · x^{α-1} · e^{-βx},  x > 0 |
| 期待値 | α/β |
| 分散 | α/β² |
| MGF | (β/(β-t))^α,  t < β |
| 再生性 | Gamma(α₁,β) + Gamma(α₂,β) = Gamma(α₁+α₂,β) |

- α=1 で Exp(β)、α=n/2, β=1/2 で χ²(n)
- ガンマ関数: Γ(α) = ∫₀^∞ t^{α-1}·e^{-t} dt, Γ(n) = (n-1)!

### 3.4 ベータ分布 Beta(α, β)

| 項目 | 値 |
|------|-----|
| PDF | x^{α-1}·(1-x)^{β-1} / B(α,β),  0 < x < 1 |
| 期待値 | α/(α+β) |
| 分散 | αβ / ((α+β)²(α+β+1)) |
| モード | (α-1)/(α+β-2),  α,β > 1 |

- B(α,β) = Γ(α)Γ(β)/Γ(α+β)（ベータ関数）
- α=β=1 で U(0,1)
- **ベイズ統計**: 二項分布の共役事前分布
  - 事前 Beta(α,β) + データ k成功/n試行 → 事後 Beta(α+k, β+n-k)

### 3.5 正規分布 N(μ, σ²)

| 項目 | 値 |
|------|-----|
| PDF | (2πσ²)^{-1/2} · exp(-(x-μ)²/(2σ²)) |
| 期待値 | μ |
| 分散 | σ² |
| MGF | exp(μt + σ²t²/2) |
| 特性関数 | exp(iμt - σ²t²/2) |
| 再生性 | N(μ₁,σ₁²) + N(μ₂,σ₂²) = N(μ₁+μ₂, σ₁²+σ₂²)（独立時） |

- **標準化**: Z = (X-μ)/σ ~ N(0,1)
- **68-95-99.7 ルール**: P(|Z| ≤ 1) ≈ 0.683, P(|Z| ≤ 2) ≈ 0.954, P(|Z| ≤ 3) ≈ 0.997
- **線形結合**: aX + bY ~ N(aμ_X + bμ_Y, a²σ²_X + b²σ²_Y + 2ab·Cov(X,Y))

### 3.6 対数正規分布 LogN(μ, σ²)

- X ~ LogN(μ, σ²) ⟺ log(X) ~ N(μ, σ²)
- E[X] = exp(μ + σ²/2), Var(X) = (exp(σ²) - 1)·exp(2μ + σ²)
- 右裾が厚い（正の歪度）、金融の株価モデル（幾何ブラウン運動）

### 3.7 ワイブル分布 Weibull(k, λ)

| 項目 | 値 |
|------|-----|
| PDF | (k/λ)·(x/λ)^{k-1}·exp(-(x/λ)^k),  x ≥ 0 |
| CDF | 1 - exp(-(x/λ)^k) |
| 期待値 | λ·Γ(1 + 1/k) |
| ハザード関数 | h(t) = (k/λ)·(t/λ)^{k-1} |

- k < 1: 故障率減少（初期不良型）、k = 1: Exp(λ)、k > 1: 故障率増加（摩耗型）
- k ≈ 3.6 で正規分布に近似

### 3.8 パレート分布 Pareto(α, x_m)

- PDF: α·x_m^α / x^{α+1},  x ≥ x_m
- E[X] = α·x_m/(α-1) (α > 1), Var(X) = x_m²·α / ((α-1)²(α-2)) (α > 2)
- べき法則（ロングテール）: P(X > x) ∝ x^{-α}
- α ≤ 1 で期待値が発散、α ≤ 2 で分散が発散

### 3.9 コーシー分布 Cauchy(x₀, γ)

- PDF: 1 / (πγ(1 + ((x-x₀)/γ)²))
- 期待値・分散ともに**存在しない**（非有限モーメント）
- 標準コーシー: t分布で自由度1と同一
- MGF は存在しないが特性関数 φ(t) = exp(ix₀t - γ|t|) は存在
- 外れ値に非常に鈍感な分布 → ロバスト統計の反例に頻出

### 3.10 標本分布族

#### カイ二乗分布 χ²(n)

- **導出**: Z₁,...,Zₙ が独立 N(0,1) のとき Σ Zᵢ² ~ χ²(n)
- E[X] = n, Var(X) = 2n
- Gamma(n/2, 1/2) と同一
- 再生性: χ²(n₁) + χ²(n₂) = χ²(n₁+n₂)
- 用途: 分散の検定、適合度検定、独立性検定

#### t 分布 t(n)

- **導出**: Z ~ N(0,1), V ~ χ²(n) が独立のとき T = Z/√(V/n) ~ t(n)
- E[T] = 0 (n > 1), Var(T) = n/(n-2) (n > 2)
- n → ∞ で N(0,1) に収束
- n = 1 でコーシー分布
- 用途: 母平均の検定・信頼区間（σ² 未知時）

#### F 分布 F(m, n)

- **導出**: U ~ χ²(m), V ~ χ²(n) が独立のとき (U/m)/(V/n) ~ F(m,n)
- E[X] = n/(n-2) (n > 2)
- T ~ t(n) のとき T² ~ F(1,n)
- 用途: 分散比の検定、分散分析（ANOVA）

### 3.11 分布間の関係図

```
Bernoulli(p)
  └─ 二項 Bin(n,p) ──→ ポアソン Poi(λ) [n→∞, p→0, np→λ]
       │                    │
       │ [CLT]              │ [CLT]
       ▼                    ▼
     正規 N(μ,σ²) ◄──── 正規 N(λ,λ)
       │
       ├─ X² → χ²(n) ─── Gamma(n/2, 1/2)
       │                        │
       │                        ├── α=1 → Exp(β)
       │                        │
       ├─ Z/√(V/n) → t(n) ── n→∞ → N(0,1)
       │
       └─ (U/m)/(V/n) → F(m,n) ── T² ~ F(1,n)

Beta(α,β)  ←→  F分布: X~Beta(α,β) ⟹ X/(1-X)·β/α ~ F(2α,2β)

対数正規 ←── exp(N(μ,σ²))
ワイブル ←── (-log U)^{1/k}·λ,  U~U(0,1)
```

### 3.12 実務活用

- **信頼性工学**: ワイブル分布で製品寿命モデリング、k推定で故障パターン識別
- **金融リスク**: 対数正規で株価、パレートで損失の裾リスク（VaR/CVaR）
- **レスポンスタイム**: 対数正規またはガンマでサーバー応答時間をモデリング
- **ベイズA/Bテスト**: Beta事前分布 + 二項データ → Beta事後分布で勝率直接計算

## §4. 多変量確率分布

### 4.1 同時分布・周辺分布・条件付き分布

- **同時分布**: f_{X,Y}(x,y) または P(X=xᵢ, Y=yⱼ)
- **周辺分布**: f_X(x) = ∫ f_{X,Y}(x,y) dy（連続）、P(X=xᵢ) = Σⱼ P(X=xᵢ, Y=yⱼ)（離散）
- **条件付き分布**: f_{Y|X}(y|x) = f_{X,Y}(x,y) / f_X(x)
- **独立性**: f_{X,Y}(x,y) = f_X(x)·f_Y(y) ⟺ X, Y 独立

### 4.2 多変量正規分布 N(μ, Σ)

- **μ**: 平均ベクトル (p×1)、**Σ**: 共分散行列 (p×p, 正定値)
- PDF: (2π)^{-p/2}·|Σ|^{-1/2}·exp(-½(x-μ)ᵀΣ⁻¹(x-μ))

**主要性質**:
- 周辺分布: Xₐ ~ N(μₐ, Σₐₐ)（部分ベクトルも正規）
- 条件付き分布: X₁|X₂=x₂ ~ N(μ_{1|2}, Σ_{1|2})
  - μ_{1|2} = μ₁ + Σ₁₂Σ₂₂⁻¹(x₂ - μ₂)
  - Σ_{1|2} = Σ₁₁ - Σ₁₂Σ₂₂⁻¹Σ₂₁
- 線形変換: AX + b ~ N(Aμ + b, AΣAᵀ)
- 無相関 ⟺ 独立（多変量正規の場合のみ成立する特別な性質）

### 4.3 マハラノビス距離

- D² = (x - μ)ᵀΣ⁻¹(x - μ)
- X ~ N(μ, Σ) のとき D² ~ χ²(p)
- 異常検知: D² > χ²_{p,α} なら外れ値と判定

### 4.4 共分散行列の性質

- **対称**: Σ = Σᵀ
- **半正定値**: 任意のベクトル a に対し aᵀΣa ≥ 0
- **分解**: Σ = UΛUᵀ（固有値分解）、Σ = LLᵀ（コレスキー分解）
- **相関行列**: R = D⁻¹ΣD⁻¹, D = diag(σ₁,...,σₚ)
- ρ_{ij} = Cov(Xᵢ,Xⱼ) / (σᵢσⱼ), -1 ≤ ρ ≤ 1

### 4.5 コプラの基礎

- **スクラーの定理**: 任意の同時分布関数 H(x,y) は
  H(x,y) = C(F_X(x), F_Y(y)) と書ける
  - F_X, F_Y: 周辺分布関数
  - C: コプラ関数 [0,1]² → [0,1]
- 周辺分布と依存構造を**分離して**モデリング可能
- 代表的コプラ:
  - **ガウスコプラ**: 相関行列 R で定義、裾依存性なし
  - **t コプラ**: ガウス + 裾依存性あり（金融リスク向き）
  - **クレイトンコプラ**: 下裾依存性が強い
  - **ガンベルコプラ**: 上裾依存性が強い

### 4.6 実務活用

- **ポートフォリオリスク**: 多変量正規で共分散推定 → VaR/CVaR算出
- **裾リスク**: ガウスコプラは裾依存性を過小評価 → t コプラ/クレイトン推奨
- **多変量異常検知**: マハラノビス距離で次元の呪いを考慮した外れ値検出
- **条件付き予測**: 多変量正規の条件付き分布 → 回帰分析と同等

## §5. 確率変数の変換

### 5.1 1変量の変換

- Y = g(X), g が単調（微分可能）のとき:
  - f_Y(y) = f_X(g⁻¹(y)) · |d/dy g⁻¹(y)|
- g が単調でない場合: 単調区間に分割して各逆関数を加算

**典型例**:
- X ~ N(0,1), Y = X² → f_Y(y) = y^{-1/2}·e^{-y/2} / √(2π) → χ²(1)
- X ~ U(0,1), Y = -log(X)/λ → Y ~ Exp(λ)
- X ~ Exp(λ), Y = X^{1/k}·λ^{1/k} → Y ~ Weibull(k, λ^{1/k})

### 5.2 多変量の変換（ヤコビアン）

- (Y₁,...,Yₙ) = g(X₁,...,Xₙ) のとき:
  - f_{Y₁,...,Yₙ}(y) = f_{X₁,...,Xₙ}(g⁻¹(y)) · |J|⁻¹
  - ヤコビアン J = det(∂(y₁,...,yₙ)/∂(x₁,...,xₙ))

**典型例（極座標変換）**:
- X = R·cosΘ, Y = R·sinΘ, ヤコビアン |J| = r
- X,Y ~ N(0,1) 独立 → R² ~ Exp(1/2), Θ ~ U(0,2π)
  → Box-Muller 法の理論的根拠

### 5.3 畳み込み（独立な確率変数の和）

- Z = X + Y（X, Y 独立）のとき:
  - f_Z(z) = ∫ f_X(x)·f_Y(z-x) dx（畳み込み積分）
  - M_Z(t) = M_X(t)·M_Y(t)（MGFの積 → 計算が楽）

**再生性のある分布**:
| 分布 | 再生性条件 |
|------|-----------|
| 正規 | N(μ₁,σ₁²) + N(μ₂,σ₂²) = N(μ₁+μ₂, σ₁²+σ₂²) |
| ポアソン | Poi(λ₁) + Poi(λ₂) = Poi(λ₁+λ₂) |
| ガンマ | Gamma(α₁,β) + Gamma(α₂,β) = Gamma(α₁+α₂,β)（β共通） |
| カイ二乗 | χ²(n₁) + χ²(n₂) = χ²(n₁+n₂) |
| 二項 | Bin(n₁,p) + Bin(n₂,p) = Bin(n₁+n₂,p)（p共通） |
| 負の二項 | NB(r₁,p) + NB(r₂,p) = NB(r₁+r₂,p)（p共通） |
| コーシー | Cauchy(x₁,γ₁) + Cauchy(x₂,γ₂) = Cauchy(x₁+x₂, γ₁+γ₂) |

### 5.4 順序統計量

- X₍₁₎ ≤ X₍₂₎ ≤ ... ≤ X₍ₙ₎: n個の独立同一分布の順序統計量
- **最小値**: F_{X₍₁₎}(x) = 1 - (1-F(x))ⁿ
- **最大値**: F_{X₍ₙ₎}(x) = (F(x))ⁿ
- **k番目**: f_{X₍ₖ₎}(x) = n! / ((k-1)!(n-k)!) · (F(x))^{k-1} · (1-F(x))^{n-k} · f(x)
- **範囲**: R = X₍ₙ₎ - X₍₁₎
- **中央値**: X₍(n+1)/2₎（nが奇数）

**U(0,1) の順序統計量**: X₍ₖ₎ ~ Beta(k, n-k+1)

### 5.5 デルタ法（分散の近似）

- √n(Xₙ - μ) →ᵈ N(0, σ²) のとき、g が微分可能で g'(μ) ≠ 0 なら:
  - √n(g(Xₙ) - g(μ)) →ᵈ N(0, σ²·(g'(μ))²)
  - すなわち Var(g(Xₙ)) ≈ (g'(μ))²·σ²/n

**典型例**:
- g(p) = log(p/(1-p))（ロジット）: Var ≈ 1/(np(1-p))
- g(p) = arcsin(√p): Var ≈ 1/(4n)（分散安定化変換）
- g(μ) = 1/μ: Var ≈ σ²/(nμ⁴)

**2次デルタ法**: g'(μ) = 0 の場合
- n(g(Xₙ) - g(μ)) →ᵈ (σ²/2)·g''(μ)·χ²(1)

### 5.6 実務活用

- **乱数生成**: 逆関数法 + Box-Muller 法でほぼ全分布の乱数を生成可能
- **信頼区間**: デルタ法でオッズ比・ハザード比の分散を近似
- **極値統計**: 順序統計量 → サーバーの最大レスポンスタイム分布推定
- **シミュレーション**: 畳み込みではなくMGFの積で和の分布を同定 → 高速

## §6. 特性関数・積率母関数

### 6.1 積率母関数 (MGF)

- **定義**: M_X(t) = E[e^{tX}]（存在すれば t=0 の近傍で）
- **k次モーメント**: E[X^k] = M^{(k)}(0)（k回微分して t=0）
- **一意性**: M_X(t) が t=0 の近傍で存在すれば分布を一意に決定
- **独立和**: M_{X+Y}(t) = M_X(t)·M_Y(t)

**注意**: MGF は常に存在するとは限らない（例: コーシー、対数正規）

### 6.2 特性関数 (CF)

- **定義**: φ_X(t) = E[e^{itX}]（i は虚数単位）
- **常に存在**: |e^{itX}| = 1 なので有界 → 全分布で定義可能
- **k次モーメント**（存在すれば）: E[X^k] = (-i)^k · φ^{(k)}(0)
- **一意性**: 特性関数は分布を一意に決定（逆は反転公式）

**主要分布の特性関数**:
| 分布 | φ(t) |
|------|-------|
| N(μ,σ²) | exp(iμt - σ²t²/2) |
| Poi(λ) | exp(λ(e^{it} - 1)) |
| Exp(λ) | λ/(λ - it) |
| Cauchy(0,1) | exp(-|t|) |
| Bin(n,p) | (1-p+pe^{it})ⁿ |

### 6.3 キュムラント母関数

- **定義**: K(t) = log M(t)
- **キュムラント**: κₖ = K^{(k)}(0)
  - κ₁ = E[X]（平均）
  - κ₂ = Var(X)（分散）
  - κ₃ = E[(X-μ)³]（3次中心モーメント、歪度の分子）
  - κ₄ = E[(X-μ)⁴] - 3σ⁴（尖度の分子）
- 正規分布: κ₁ = μ, κ₂ = σ², κₖ = 0 (k ≥ 3)
  → 正規分布はキュムラントが2次で打ち切り

### 6.4 反転公式

- 離散: P(X=k) = (1/2π) ∫_{-π}^{π} φ(t)·e^{-ikt} dt
- 連続: f(x) = (1/2π) ∫_{-∞}^{∞} φ(t)·e^{-itx} dt（フーリエ逆変換）

### 6.5 連続性定理（レヴィ）

- φₙ(t) → φ(t) が各 t で成立し、φ が t=0 で連続
  ⟹ 対応する分布 Fₙ →ᵈ F
- CLT の証明に本質的に使用される

### 6.6 実務活用

- **分布の同定**: MGF/CF を計算し既知の分布と照合 → 再生性の証明
- **和の分布**: 独立確率変数の和の分布をMGFの積で求める
- **正規性からのズレ**: κ₃（歪度）≠ 0 → 非対称、κ₄（超過尖度）≠ 0 → 裾の厚さ
- **安定分布**: 特性関数のみで定義可能（PDFが閉じた形を持たない場合）

## §7. 条件付き期待値

### 7.1 定義

- **離散**: E[X|Y=y] = Σ_x x · P(X=x|Y=y)
- **連続**: E[X|Y=y] = ∫ x · f_{X|Y}(x|y) dx
- **測度論**: E[X|G] は G-可測な確率変数で、任意の G ∈ G に対し
  ∫_G E[X|G] dP = ∫_G X dP を満たす（Radon-Nikodym 微分）

### 7.2 主要性質

1. **線形性**: E[aX + bY | Z] = aE[X|Z] + bE[Y|Z]
2. **取り出し規則**: E[g(Y)·X | Y] = g(Y)·E[X|Y]（Y で条件付けた世界で g(Y) は定数）
3. **全期待値の法則**: E[X] = E[E[X|Y]]（塔の性質、law of total expectation）
4. **独立性**: X, Y 独立 ⟹ E[X|Y] = E[X]
5. **繰り返し条件付け**: E[E[X|Y,Z] | Y] = E[X|Y]（σ-加法族の包含関係）
6. **イェンセン**: g が凸 ⟹ E[g(X)|Y] ≥ g(E[X|Y])

### 7.3 全分散の公式

- **Var(X) = E[Var(X|Y)] + Var(E[X|Y])**
  - 第1項: 群内分散の平均（Y を固定した時の X のばらつき）
  - 第2項: 群間分散（Y によって X の平均がどれだけ変わるか）

**応用例**:
- 混合正規分布: X|Z ~ N(μ_Z, σ²_Z)、Z が混合成分
  - E[X] = E[μ_Z], Var(X) = E[σ²_Z] + Var(μ_Z)

### 7.4 条件付き期待値の最適性

- E[X|Y] は Y の関数 g(Y) の中で E[(X - g(Y))²] を最小にする
  → 最小平均二乗誤差（MMSE）推定量
- 回帰関数: E[Y|X=x] は X による Y の最良予測

### 7.5 実務活用

- **階層ベイズモデル**: パラメータの事前分布を条件付き期待値で積分消去
- **EM アルゴリズム**: E ステップ = 潜在変数の条件付き期待値を計算
- **分散分解**: ANOVA は全分散公式の直接的応用
- **強化学習**: 状態価値関数 V(s) = E[Σ γᵗrₜ | s₀ = s] は条件付き期待値
- **保険数理**: E[総損害額] = E[事故件数] × E[1件あたり損害額]（独立時の全期待値法則）

## §8. 極限定理

### 8.1 収束の4種類と関係

```
Lp 収束  ─→  確率収束  ─→  分布収束（法則収束）
               ↑
概収束（a.s.）─┘
```

| 収束 | 定義 | 記法 |
|------|------|------|
| **概収束** | P(lim Xₙ = X) = 1 | Xₙ →ᵃ·ˢ· X |
| **確率収束** | ∀ε>0: lim P(\|Xₙ-X\|>ε) = 0 | Xₙ →ᴾ X |
| **分布収束** | lim Fₙ(x) = F(x) (Fの連続点で) | Xₙ →ᵈ X |
| **Lp 収束** | lim E[\|Xₙ-X\|ᵖ] = 0 | Xₙ →ᴸᵖ X |

- 概収束 ⟹ 確率収束 ⟹ 分布収束（逆は一般に不成立）
- Lp 収束 ⟹ 確率収束（逆は一般に不成立）
- 分布収束で X = c（定数）の場合 ⟹ 確率収束

### 8.2 大数の弱法則 (WLLN)

- X₁,...,Xₙ が独立同一分布、E[X] = μ, Var(X) = σ² < ∞ のとき:
  - X̄ₙ →ᴾ μ（n → ∞）
- **証明**: チェビシェフの不等式 P(|X̄ₙ - μ| > ε) ≤ σ²/(nε²) → 0

### 8.3 大数の強法則 (SLLN)

- X₁,...,Xₙ が独立同一分布、E[|X|] < ∞ のとき:
  - X̄ₙ →ᵃ·ˢ· μ
- 分散の存在は不要（期待値の存在のみで成立）
- 実用上は WLLN で十分だが、確率1の収束が必要な理論的場面では SLLN

### 8.4 中心極限定理 (CLT)

#### リンドバーグ・レヴィの CLT（i.i.d.版）

- X₁,...,Xₙ が i.i.d.、E[X] = μ, Var(X) = σ² (0 < σ² < ∞) のとき:
  - √n(X̄ₙ - μ)/σ →ᵈ N(0,1)
  - 同値: Σᵢ Xᵢ ≈ N(nμ, nσ²)

#### リンドバーグ・フェラーの CLT（独立・非同一分布版）

- Xₖ が独立（同一分布でなくてよい）、E[Xₖ]=μₖ, Var(Xₖ)=σₖ² のとき:
  - sₙ² = Σ σₖ², リンドバーグ条件を満たせば CLT が成立
  - **リンドバーグ条件**: 任意の ε > 0 に対し
    lim (1/sₙ²) Σₖ E[(Xₖ-μₖ)²·1_{|Xₖ-μₖ|>εsₙ}] = 0
  - 直感: どの1つの変数も和を支配しない

### 8.5 ベリー・エシーン不等式（CLT の収束速度）

- sup_x |Fₙ(x) - Φ(x)| ≤ C·ρ / (σ³·√n)
  - ρ = E[|X - μ|³]（3次絶対モーメント）
  - C ≤ 0.4748（最良の既知定数）
- n ≈ 30 で正規近似が「実用的に十分」とされる根拠

### 8.6 スラツキーの定理

- Xₙ →ᵈ X かつ Yₙ →ᴾ c（定数）のとき:
  - Xₙ + Yₙ →ᵈ X + c
  - Xₙ · Yₙ →ᵈ c·X
  - Xₙ / Yₙ →ᵈ X/c（c ≠ 0）
- **応用**: t 統計量 = Z/√(S²/σ²) で S² →ᴾ σ² を使い、t →ᵈ N(0,1) を導出

### 8.7 連続写像定理 (CMT)

- Xₙ →ᵈ X かつ g が連続 ⟹ g(Xₙ) →ᵈ g(X)
- 確率収束・概収束でも成立
- **デルタ法の根拠**: √n(Xₙ-μ) →ᵈ N(0,σ²) に g を適用

### 8.8 実務活用

- **サンプルサイズ設計**: CLT + 望む精度 ε から n ≥ (z_{α/2}·σ/ε)² を算出
- **ブートストラップ**: SLLN + CLT が理論的正当化（経験分布 →ᵃ·ˢ· 真の分布）
- **A/Bテスト**: 二項分布の正規近似 → z 検定の根拠
- **品質管理**: X̄ 管理図は CLT に基づく（±3σ ルール）
- **モンテカルロ法**: 推定精度 O(1/√n) は CLT から直接導出

## §9. 確率不等式

### 9.1 マルコフの不等式

- X ≥ 0, a > 0 のとき: **P(X ≥ a) ≤ E[X] / a**
- 非負確率変数にのみ適用可。非常に緩い上界

### 9.2 チェビシェフの不等式

- Var(X) = σ² < ∞ のとき: **P(|X - μ| ≥ kσ) ≤ 1/k²**
- マルコフを (X-μ)² に適用して導出
- k=2: P ≤ 0.25, k=3: P ≤ 0.111（分布フリー）
- 正規分布では k=2 で P ≈ 0.046 → チェビシェフは保守的

### 9.3 イェンセンの不等式

- g が凸関数 ⟹ **E[g(X)] ≥ g(E[X])**
- g が凹関数 ⟹ E[g(X)] ≤ g(E[X])

**典型例**:
- g(x) = x²: E[X²] ≥ (E[X])² → Var(X) ≥ 0 の別証
- g(x) = -log(x): E[-log X] ≥ -log E[X] → KL ダイバージェンスの非負性
- g(x) = e^x: E[e^X] ≥ e^{E[X]}

### 9.4 コーシー・シュワルツの不等式

- **(E[XY])² ≤ E[X²]·E[Y²]**
- 等号: Y = aX (a.s.) のとき
- 系: |Cor(X,Y)| ≤ 1, |Cov(X,Y)| ≤ σ_X·σ_Y

### 9.5 ヘルダーの不等式

- 1/p + 1/q = 1 (p,q > 1) のとき:
  - **E[|XY|] ≤ (E[|X|ᵖ])^{1/p} · (E[|Y|ᵍ])^{1/q}**
- p = q = 2 でコーシー・シュワルツに帰着

### 9.6 ミンコフスキーの不等式

- p ≥ 1 のとき:
  - **(E[|X+Y|ᵖ])^{1/p} ≤ (E[|X|ᵖ])^{1/p} + (E[|Y|ᵖ])^{1/p}**
- Lp ノルムの三角不等式

### 9.7 チェルノフ境界

- 任意の t > 0 に対し: **P(X ≥ a) ≤ e^{-ta} · M_X(t)**
- マルコフの不等式を e^{tX} に適用して導出
- t を最適化（saddlepoint）で最もタイトな指数的上界を得る

**二項分布への適用** (X ~ Bin(n,p), a > np):
- P(X ≥ (1+δ)np) ≤ exp(-npδ²/3)（δ > 0 の乗法型）
- P(X ≥ np + t) ≤ exp(-2t²/n)（ヘフディング型）

### 9.8 ヘフディングの不等式

- X₁,...,Xₙ が独立、aᵢ ≤ Xᵢ ≤ bᵢ のとき:
  - **P(X̄ₙ - μ ≥ t) ≤ exp(-2n²t² / Σ(bᵢ-aᵢ)²)**
- 分散を使わず範囲のみで上界を与える → 分布フリーな集中不等式

### 9.9 サブガウシアン性

- 確率変数 X が σ²-サブガウシアン:
  - E[e^{t(X-μ)}] ≤ exp(σ²t²/2) （∀t）
- 有界確率変数はサブガウシアン（ヘフディングの補題）
- サブガウシアンの和もサブガウシアン → 高次元統計で基本的

### 9.10 実務活用

- **外れ値検出**: チェビシェフ → 分布仮定なしの保守的閾値（k=3 で外れ値率 ≤ 11%）
- **機械学習**: チェルノフ/ヘフディング → PAC学習のサンプル複雑性の上界
- **アルゴリズム**: ランダムアルゴリズムの失敗確率をチェルノフ境界で評価
- **A/Bテスト**: ヘフディングで分布フリーな信頼区間を構築（保守的だが頑健）
- **KL ダイバージェンス**: イェンセンから D_KL(P||Q) ≥ 0 を導出（変分推論の根幹）
- **ポートフォリオ**: コーシー・シュワルツで相関係数の範囲を保証
