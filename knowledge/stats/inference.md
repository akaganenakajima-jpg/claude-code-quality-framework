> 統計検定1級 満点レベル対応。点推定・区間推定・仮説検定・漸近理論・サンプルサイズ設計を網羅。

## 1. 点推定（Point Estimation）

### 1.1 最尤推定（Maximum Likelihood Estimation: MLE）

#### 尤度関数と対数尤度

- **尤度関数**: 観測データ x₁,...,xₙ が与えられたとき、パラメータ θ の関数として
  ```
  L(θ) = L(θ; x₁,...,xₙ) = ∏ᵢ₌₁ⁿ f(xᵢ; θ)
  ```
- **対数尤度**: ℓ(θ) = log L(θ) = Σᵢ₌₁ⁿ log f(xᵢ; θ)
- **MLE**: θ̂_MLE = argmax_θ L(θ) = argmax_θ ℓ(θ)

#### スコア関数とフィッシャー情報量

- **スコア関数**: s(θ) = ∂ℓ(θ)/∂θ
  - 正則条件下で E[s(θ)] = 0
- **フィッシャー情報量**（1標本）:
  ```
  I(θ) = E[s(θ)²] = -E[∂²ℓ(θ)/∂θ²] = Var[s(θ)]
  ```
- **n標本のフィッシャー情報量**: Iₙ(θ) = n·I(θ)（iid の場合）
- **多次元**: フィッシャー情報行列 I(θ)ᵢⱼ = -E[∂²ℓ/∂θᵢ∂θⱼ]

#### 正則条件（Regularity Conditions）

| 条件 | 内容 |
|------|------|
| R1 | パラメータ空間 Θ が開集合 |
| R2 | 台（support）が θ に依存しない |
| R3 | ℓ(θ) が θ について3回微分可能 |
| R4 | 積分と微分の交換が可能 |
| R5 | I(θ) > 0（退化しない） |

#### MLEの漸近的性質（正則条件下）

- **一致性（Consistency）**: θ̂_MLE →ᵖ θ₀（n→∞）
- **漸近正規性（Asymptotic Normality）**:
  ```
  √n (θ̂_MLE - θ₀) →ᵈ N(0, I(θ₀)⁻¹)
  ```
- **漸近有効性（Asymptotic Efficiency）**: 漸近分散がクラメール・ラオの下界に到達
- **不変性（Invariance）**: g(θ) の MLE は g(θ̂_MLE)（g が1対1でなくても成立）

#### 主要分布のMLE

| 分布 | パラメータ | MLE |
|------|-----------|-----|
| 正規 N(μ,σ²) | μ | x̄ |
| 正規 N(μ,σ²) | σ² | (1/n)Σ(xᵢ-x̄)²（不偏ではない） |
| ベルヌーイ Ber(p) | p | x̄ |
| ポアソン Poi(λ) | λ | x̄ |
| 指数 Exp(λ) | λ | 1/x̄ |
| 一様 U(0,θ) | θ | max(x₁,...,xₙ)（正則条件を満たさない） |
| ガンマ Γ(α,β) | (α,β) | 解析解なし→数値解法 |

#### 数値最適化法

- **ニュートン・ラフソン法**:
  ```
  θ⁽ᵗ⁺¹⁾ = θ⁽ᵗ⁾ - [∂²ℓ/∂θ²]⁻¹ · ∂ℓ/∂θ  （θ⁽ᵗ⁾で評価）
  ```
  - フィッシャー・スコアリング: ヘシアンの代わりに期待フィッシャー情報量を使用
    ```
    θ⁽ᵗ⁺¹⁾ = θ⁽ᵗ⁾ + I(θ⁽ᵗ⁾)⁻¹ · s(θ⁽ᵗ⁾)
    ```
- **EMアルゴリズム**（潜在変数を含むモデル）:
  - E-step: Q(θ|θ⁽ᵗ⁾) = E_{Z|X,θ⁽ᵗ⁾}[log L(θ; X, Z)]
  - M-step: θ⁽ᵗ⁺¹⁾ = argmax_θ Q(θ|θ⁽ᵗ⁾)
  - 性質: ℓ(θ⁽ᵗ⁺¹⁾) ≥ ℓ(θ⁽ᵗ⁾)（単調増加）、局所最適に収束
  - 応用: 混合正規分布、欠測データ、隠れマルコフモデル
- **IRLS（反復再重み付け最小二乗法）**: GLMのMLE。各反復で重み行列を更新

#### 実務活用

- ロジスティック回帰: ℓ(β) = Σ[yᵢ·xᵢᵀβ - log(1+exp(xᵢᵀβ))] → IRLS or ニュートン法
- 生存分析: 打ち切りデータの尤度 L = ∏[f(tᵢ)]^δᵢ · [S(tᵢ)]^(1-δᵢ)
- 混合モデルのパラメータ推定（EMアルゴリズム）

### 1.2 モーメント法（Method of Moments: MoM）

#### 定義

- k個のパラメータ θ₁,...,θₖ に対し、母集団モーメント μₖ' = E[Xᵏ] と標本モーメント mₖ' = (1/n)ΣXᵢᵏ を等置
  ```
  μᵣ'(θ₁,...,θₖ) = mᵣ'   (r = 1,...,k)
  ```
- この連立方程式を解いて推定量を得る

#### MLEとの比較

| 項目 | MoM | MLE |
|------|-----|-----|
| 計算 | 一般に簡単 | 反復法が必要な場合あり |
| 漸近効率 | 一般に劣る | 漸近有効（クラメール・ラオの下界到達） |
| 一致性 | 一般に一致 | 一致 |
| 不偏性 | 必ずしも不偏でない | 必ずしも不偏でない |
| 台の制約 | 推定値が有効範囲外になることがある | 自動的にパラメータ空間内 |

#### 一般化モーメント法（GMM: Generalized Method of Moments）

- モーメント条件 E[g(Xᵢ, θ₀)] = 0 を満たす θ を推定
- 目的関数: Qₙ(θ) = [n⁻¹ Σg(Xᵢ,θ)]ᵀ W [n⁻¹ Σg(Xᵢ,θ)]
- 最適重み行列: W = [Var(g)]⁻¹
- 過剰識別制約の検定（ハンセンのJ検定）

### 1.3 十分統計量・完備性

#### 十分統計量（Sufficient Statistic）

- **定義**: T(X) が θ に対して十分統計量 ⟺ T(X) が与えられたときの X の条件付き分布が θ に依存しない
- **ネイマン・フィッシャーの分解定理**: T(X) が十分 ⟺
  ```
  f(x; θ) = g(T(x), θ) · h(x)
  ```
  ここで g は T(x) と θ のみに依存し、h は θ に依存しない

#### 最小十分統計量（Minimal Sufficient Statistic）

- すべての十分統計量の関数になっている十分統計量
- 構成法: f(x;θ)/f(y;θ) が θ に依存しない ⟺ T(x) = T(y)

#### 指数型分布族と十分統計量

- **指数型分布族**:
  ```
  f(x; θ) = h(x) · c(θ) · exp(Σⱼ₌₁ᵏ ηⱼ(θ) · Tⱼ(x))
  ```
- T(X) = (T₁(X),...,Tₖ(X)) が（自然パラメータ η に対する）十分統計量
- 自然パラメータ空間が開集合なら T は完備十分統計量

| 分布 | 十分統計量 T(X) |
|------|-----------------|
| N(μ,σ²) 両方未知 | (Σxᵢ, Σxᵢ²) |
| Poi(λ) | Σxᵢ |
| Ber(p) | Σxᵢ |
| Γ(α,β) α既知 | Σxᵢ |
| U(0,θ) | max(xᵢ)（指数型ではない） |

#### 完備性（Completeness）

- T(X) が完備 ⟺ すべての θ で E_θ[g(T)] = 0 ならば P_θ(g(T)=0) = 1
- 直感: T(X) に「θ に無関係な情報」が含まれない
- バスー（Basu）の定理: 完備十分統計量は任意の補助統計量と独立

---

### 1.4 不偏推定と有効性

#### 不偏推定量（Unbiased Estimator）

- E[T] = θ（すべての θ について）
- 不偏性は必ずしも望ましい性質ではない（MSEが大きい場合がある）

#### クラメール・ラオの下界（Cramér-Rao Lower Bound: CRLB）

- 不偏推定量 T に対して:
  ```
  Var(T) ≥ 1 / Iₙ(θ) = 1 / (n·I(θ))
  ```
- **有効推定量（Efficient Estimator）**: Var(T) = CRLB を達成する推定量
- 多次元版: Cov(T) - [Iₙ(θ)]⁻¹ が非負定値

#### 一様最小分散不偏推定量（UMVUE）

- すべての θ で、すべての不偏推定量の中で分散最小
- **ラオ・ブラックウェルの定理**: 不偏推定量 U と十分統計量 T に対し、E[U|T] は不偏かつ Var(E[U|T]) ≤ Var(U)
- **レーマン・シェッフェの定理**: 完備十分統計量の関数である不偏推定量は UMVUE

#### UMVUE の構成手順

1. 完備十分統計量 T を見つける
2. パラメータの任意の不偏推定量 U を見つける
3. E[U|T] を計算 → これが UMVUE

#### 主要分布のUMVUE

| 母集団 | 推定対象 | UMVUE |
|--------|---------|-------|
| N(μ,σ²) | μ | x̄ |
| N(μ,σ²) | σ² | s² = Σ(xᵢ-x̄)²/(n-1) |
| Poi(λ) | λ | x̄ |
| Poi(λ) | P(X=0)=e⁻λ | ((n-1)/n)^(Σxᵢ) |
| Ber(p) | p | x̄ |
| Ber(p) | p(1-p) | x̄(1-x̄)·n/(n-1) |
| U(0,θ) | θ | (n+1)/n · max(xᵢ) |

#### ジェームズ・スタインの推定量

- p次元正規平均 θ = (θ₁,...,θₚ) の推定（p ≥ 3）
- **縮小推定量**: θ̂_JS = (1 - (p-2)/(‖X‖²)) · X
- MSE(θ̂_JS) < MSE(X) （すべての θ で）— スタインのパラドックス
- X̄ は許容的でない（p ≥ 3 のとき）
- リッジ回帰・正則化の理論的根拠

### 1.5 ベイズ推定（Bayesian Estimation）

#### 基本構造

- **事前分布**: π(θ)（データ観測前のパラメータの信念）
- **尤度**: L(θ|x) = f(x|θ)
- **事後分布**: π(θ|x) ∝ f(x|θ) · π(θ)（ベイズの定理）
- **周辺尤度**: m(x) = ∫ f(x|θ)π(θ)dθ（正規化定数、モデル選択に使用）

#### 損失関数とベイズ推定量

| 損失関数 | ベイズ推定量 |
|---------|-------------|
| 二乗誤差 L=(θ-δ)² | 事後平均 E[θ|x] |
| 絶対誤差 L=|θ-δ| | 事後中央値 |
| 0-1損失 | 事後最頻値（MAP推定） |

#### 共役事前分布

| 尤度 | 共役事前 | 事後 | 事後パラメータ |
|------|---------|------|---------------|
| Bin(n,p) | Beta(α,β) | Beta(α+x, β+n-x) | 成功α+x, 失敗β+n-x |
| Poi(λ) | Gamma(α,β) | Gamma(α+Σxᵢ, β+n) | |
| N(μ,σ²既知) | N(μ₀,τ₀²) | N(μₙ, τₙ²) | μₙ=(τ₀⁻²μ₀+nσ⁻²x̄)/(τ₀⁻²+nσ⁻²) |
| N(μ既知,σ²) | InvGamma(α,β) | InvGamma(α+n/2, β+Σ(xᵢ-μ)²/2) | |
| Exp(λ) | Gamma(α,β) | Gamma(α+n, β+Σxᵢ) | |
| Multinomial | Dirichlet(α) | Dirichlet(α+x) | |

#### 事前分布の選択

- **無情報事前分布**: ジェフリーズ事前 π(θ) ∝ √I(θ)
  - 再パラメータ化に対して不変
  - 正規平均: π(μ) ∝ 1（一様）、正規分散: π(σ²) ∝ 1/σ²
- **弱情報事前分布**: 半コーシー、半正規（Stan推奨）
- **経験ベイズ**: 事前分布のハイパーパラメータをデータから推定

#### 事後分布の計算

- **解析的**: 共役事前を使用
- **MCMC**: メトロポリス・ヘイスティングス法、ギブスサンプリング、ハミルトニアンモンテカルロ（HMC）
- **変分推論**: 事後分布を簡単な分布族で近似（ELBO最大化）

## 2. 区間推定（Interval Estimation）

### 2.1 信頼区間の基礎

#### 定義（頻度論的解釈）

- **100(1-α)% 信頼区間 [L(X), U(X)]**:
  ```
  P_θ(L(X) ≤ θ ≤ U(X)) ≥ 1 - α   （すべての θ について）
  ```
- 「θ が区間に含まれる確率が 1-α」ではなく「この手続きを繰り返すと 100(1-α)% の割合で θ を含む」
- 被覆確率（coverage probability）が正確に 1-α のとき「正確な信頼区間」

#### ピボット量の構成法

- ピボット量 Q(X,θ) の分布が θ に依存しない量を構成
- P(a ≤ Q(X,θ) ≤ b) = 1-α を θ について解く

| 設定 | ピボット量 | 分布 |
|------|-----------|------|
| N(μ,σ²既知)のμ | (X̄-μ)/(σ/√n) | N(0,1) |
| N(μ,σ²未知)のμ | (X̄-μ)/(S/√n) | t(n-1) |
| N(μ,σ²)のσ² | (n-1)S²/σ² | χ²(n-1) |
| 2群の平均差（等分散） | (X̄₁-X̄₂-δ)/(Sp√(1/n₁+1/n₂)) | t(n₁+n₂-2) |
| 2群の平均差（不等分散） | ウェルチの近似 | t(ν)（サタスウェイトの自由度） |

### 2.2 正規母集団の信頼区間

#### 平均 μ の信頼区間

- σ 既知: x̄ ± z_{α/2} · σ/√n
- σ 未知: x̄ ± t_{α/2}(n-1) · s/√n
- ここで s² = Σ(xᵢ-x̄)²/(n-1)

#### 分散 σ² の信頼区間

```
((n-1)s²/χ²_{α/2}(n-1),  (n-1)s²/χ²_{1-α/2}(n-1))
```

#### 2群の分散比の信頼区間

```
(S₁²/S₂²) / F_{α/2}(n₁-1,n₂-1)  ≤  σ₁²/σ₂²  ≤  (S₁²/S₂²) / F_{1-α/2}(n₁-1,n₂-1)
```

### 2.3 二項比率の信頼区間

| 方法 | 区間 | 特徴 |
|------|------|------|
| ワルド（Wald） | p̂ ± z_{α/2}√(p̂(1-p̂)/n) | 簡便だが被覆確率が悪い（特にp≈0,1やn小） |
| ウィルソン（Wilson） | (p̂ + z²/2n ± z√(p̂q̂/n + z²/4n²)) / (1+z²/n) | 推奨。被覆確率が安定 |
| クロッパー・ピアソン | ベータ分布の分位点 | 正確（保守的）。B(x;n,p_L)=α/2, B(x;n,p_U)=1-α/2 |
| アグレスティ・コール | ñ=n+z², p̃=(x+z²/2)/ñ で ワルド式 | ウィルソンの良い近似 |

### 2.4 大標本での漸近的信頼区間

- MLEの漸近正規性を利用:
  ```
  θ̂ ± z_{α/2} / √(Iₙ(θ̂))
  ```
- **プロファイル尤度に基づく信頼区間**: {θ : 2(ℓ(θ̂)-ℓ(θ)) ≤ χ²_{1,α}} — 小標本で漸近ワルドより良い

### 2.5 同時信頼区間

| 方法 | 適用 | 区間幅 |
|------|------|--------|
| ボンフェローニ | 任意のk個の対比 | 各区間を α/k で構成 |
| シェッフェ | すべての線形対比 | √(k·F_{α}(k,n-k)) を使用 |
| テューキー（HSD） | すべてのペアワイズ比較 | スチューデント化された範囲分布 |
| ダネット | 対照群との比較 | 多変量t分布 |

### 2.6 ベイズ信用区間

- **信用区間（Credible Interval）**: P(θ ∈ C | x) = 1-α
- **最高事後密度区間（HPD）**: 区間内のすべての点の事後密度 ≥ 区間外のすべての点の事後密度
  - 最短の信用区間
  - 事後分布が対称なら等裾区間と一致
- **頻度論信頼区間との違い**: 信用区間は「θ が C に含まれる確率が 1-α」と直接解釈可能

## 3. 仮説検定（Hypothesis Testing）

### 3.1 基本理論

#### 検定の枠組み

- **帰無仮説 H₀**: 棄却の対象（「差がない」「効果がない」等）
- **対立仮説 H₁**: 採択したい仮説
- **検定統計量 T(X)**: データから計算される統計量
- **棄却域 R**: T(X) ∈ R のとき H₀ を棄却

#### 過誤と検出力

| | H₀ が真 | H₁ が真 |
|---|---------|---------|
| H₀ 棄却 | 第1種の過誤（α） | 正しい棄却（検出力 1-β） |
| H₀ 非棄却 | 正しい判断（1-α） | 第2種の過誤（β） |

- **有意水準**: α = P(H₀を棄却 | H₀が真) の上限
- **検出力（Power）**: 1-β = P(H₀を棄却 | H₁が真)
- **検出力関数**: β(θ) = P_θ(X ∈ R)

#### p値

- **定義**: 観測データと同等以上に H₀ に反する結果が得られる確率
  ```
  p = P_{H₀}(T(X) ≥ T(x_obs))  （片側の場合）
  ```
- p ≤ α のとき H₀ を棄却
- p値は H₀ が真である確率ではない
- p値は効果の大きさを示さない

### 3.2 ネイマン・ピアソンの基本定理

#### 単純仮説 vs 単純仮説

- H₀: θ = θ₀ vs H₁: θ = θ₁
- **最強力検定（Most Powerful Test）**: 有意水準 α で検出力最大の検定
- **定理**: 尤度比
  ```
  Λ(x) = L(θ₁; x) / L(θ₀; x) > k
  ```
  のとき H₀ を棄却する検定が最強力検定（k は α から決定）

#### 一様最強力検定（UMP）

- H₁ が複合仮説の場合、すべての θ ∈ Θ₁ で最強力な検定
- **単調尤度比（MLR）性質**: 指数型分布族で片側検定の場合に UMP が存在
  - H₀: θ ≤ θ₀ vs H₁: θ > θ₀ → T(x) > c が UMP
- 両側検定には一般に UMP が存在しない → UMPU（一様最強力不偏検定）を考える

#### 一様最強力不偏検定（UMPU）

- 不偏検定: β(θ) ≥ α（θ ∈ Θ₁ のとき） — 検出力が有意水準以上
- 指数型分布族の場合、UMPU が構成可能（条件付き検定）

### 3.3 尤度比検定（Likelihood Ratio Test: LRT）

#### 尤度比統計量

```
Λ = sup_{θ∈Θ₀} L(θ) / sup_{θ∈Θ} L(θ) = L(θ̂₀) / L(θ̂)
```
- 0 ≤ Λ ≤ 1、小さいほど H₀ に不利

#### ウィルクスの定理

- 正則条件下で n → ∞ のとき:
  ```
  -2 log Λ →ᵈ χ²(r)   （H₀ の下で）
  ```
  ここで r = dim(Θ) - dim(Θ₀)（制約の数）

#### 三位一体（Trinity of Tests）

| 検定 | 統計量 | 特徴 |
|------|--------|------|
| **尤度比検定** | -2(ℓ(θ̂₀)-ℓ(θ̂)) | 制約あり・なし両方の推定が必要 |
| **ワルド検定** | (θ̂-θ₀)ᵀ I(θ̂)(θ̂-θ₀) | 制約なしの推定のみ必要 |
| **スコア検定（ラオ）** | s(θ₀)ᵀ I(θ₀)⁻¹ s(θ₀) | 制約ありの推定のみ必要 |

- 漸近的にすべて χ²(r) に従い等価
- 有限標本: LRT ≈ ワルド ≈ スコア だが微妙に異なる
- 一般に ワルド ≥ LRT ≥ スコア（統計量の大きさの順）

#### 実務活用

- モデル比較: ネストされたモデル間の検定（GLMの変数選択）
- AIC = -2ℓ(θ̂) + 2k、BIC = -2ℓ(θ̂) + k·log(n) — LRTの情報量規準版
- デビアンス検定（GLM）: D = -2(ℓ_saturated - ℓ_model) → χ² で適合度判定

### 3.4 適合度検定・独立性検定

#### ピアソンのカイ二乗検定

- **適合度検定**: H₀: pᵢ = pᵢ₀ (i=1,...,k)
  ```
  χ² = Σᵢ (Oᵢ - Eᵢ)² / Eᵢ  →  χ²(k-1-m)
  ```
  ここで m はデータから推定したパラメータ数、Eᵢ = n·pᵢ₀
- **独立性検定**: r×c 分割表
  ```
  χ² = ΣΣ (Oᵢⱼ - Eᵢⱼ)² / Eᵢⱼ  →  χ²((r-1)(c-1))
  ```
  Eᵢⱼ = (行計×列計)/総計
- **適用条件**: すべての Eᵢ ≥ 5 が望ましい（小さい場合はフィッシャーの正確検定）

#### コルモゴロフ・スミルノフ検定

- H₀: F(x) = F₀(x)（連続分布への適合度）
- 検定統計量: D = sup_x |F̂ₙ(x) - F₀(x)|
- 分布自由（distribution-free）だが、パラメータをデータから推定した場合は修正が必要（リリフォース検定）
- 2標本版: D = sup_x |F̂ₘ(x) - Ĝₙ(x)|

#### 正規性検定

| 検定 | 統計量 | 感度 | 注意点 |
|------|--------|------|--------|
| シャピロ・ウィルク | W（順序統計量の線形結合とS²の比） | 裾・歪み両方に高感度 | n≤5000。最も検出力が高い |
| アンダーソン・ダーリング | A²（裾に重みを置いたEDF統計量） | 裾の逸脱に敏感 | |
| ジャック・ベラ | JB = n(S²/6 + (K-3)²/24) | 歪度・尖度を検定 | 大標本向け。JB→χ²(2) |
| リリフォース | KS統計量（パラメータ推定後） | 全体的な適合度 | KS臨界値は使えない |

### 3.5 主要なパラメトリック検定

| 検定 | H₀ | 統計量 | 分布（H₀下） |
|------|-----|--------|-------------|
| 1標本z検定 | μ=μ₀（σ既知） | z=(x̄-μ₀)/(σ/√n) | N(0,1) |
| 1標本t検定 | μ=μ₀（σ未知） | t=(x̄-μ₀)/(s/√n) | t(n-1) |
| 対応ありt検定 | μ_D=0 | t=d̄/(s_D/√n) | t(n-1) |
| 2標本t検定（等分散） | μ₁=μ₂ | t=(x̄₁-x̄₂)/(sp√(1/n₁+1/n₂)) | t(n₁+n₂-2) |
| ウェルチのt検定 | μ₁=μ₂ | t=(x̄₁-x̄₂)/√(s₁²/n₁+s₂²/n₂) | t(ν)（サタスウェイト） |
| F検定 | σ₁²=σ₂² | F=s₁²/s₂² | F(n₁-1,n₂-1) |
| 1元配置分散分析 | μ₁=...=μₖ | F=MS_B/MS_W | F(k-1,N-k) |

### 3.6 ノンパラメトリック検定

| 検定 | 対応するパラメトリック検定 | 特徴 |
|------|--------------------------|------|
| ウィルコクソンの符号順位検定 | 1標本/対応ありt検定 | 中央値の検定。対称性仮定 |
| マン・ホイットニーのU検定 | 2標本t検定 | 順位に基づく。分布自由 |
| クラスカル・ウォリス検定 | 1元配置分散分析 | k群の中央値比較 |
| フリードマン検定 | 反復測定分散分析 | ブロック付きk群比較 |
| 符号検定 | 1標本t検定 | 中央値の検定。対称性不要 |
| フィッシャーの正確検定 | 2×2 χ² 検定 | 小標本で正確。超幾何分布 |
| 順位相関（スピアマン/ケンドール） | ピアソン相関の検定 | 単調関係の検定 |

### 3.7 多重検定（Multiple Testing）

#### FWER制御（Family-Wise Error Rate）

- **FWER**: 1つ以上の偽陽性が出る確率
- **ボンフェローニ**: 各検定の有意水準を α/m に（m は検定数）。保守的
- **シダック**: 各検定を 1-(1-α)^(1/m) で — ボンフェローニより少し緩い
- **ホルムの方法（ステップダウン）**:
  1. p値を昇順にソート: p₍₁₎ ≤ p₍₂₎ ≤ ... ≤ p₍ₘ₎
  2. p₍ᵢ₎ ≤ α/(m-i+1) の最大の i まで棄却
  - ボンフェローニより検出力が高く、常に推奨
- **ホッホベルクの方法（ステップアップ）**: ステップダウンの逆。検定統計量が独立なら有効

#### FDR制御（False Discovery Rate）

- **FDR**: E[V/R]（棄却のうち偽陽性の割合の期待値）
- **ベンジャミニ・ホッホベルク（BH法）**:
  1. p値を昇順にソート
  2. p₍ᵢ₎ ≤ (i/m)·α の最大の i を見つける
  3. p₍₁₎,...,p₍ᵢ₎ を棄却
  - 独立または正の相関（PRDS）のとき FDR ≤ α
- **q値**: 各検定の「その検定を棄却したときのFDR」

| 方法 | 制御対象 | 保守性 | 適用場面 |
|------|---------|--------|---------|
| ボンフェローニ | FWER | 最も保守的 | 検定数が少ない場合 |
| ホルム | FWER | ボンフェローニより良い | 一般的なFWER制御 |
| BH | FDR | 緩い | 大量の検定（ゲノム、fMRI等） |

#### 実務活用

- A/B/C/Dテスト: 3ペア比較 → ホルム法でFWER制御
- ゲノムワイド関連解析: 数百万SNP → BH法でFDR制御
- fMRI: 数万ボクセル → FDR制御またはクラスター補正

## 4. 漸近理論（Asymptotic Theory）

### 4.1 収束の種類

| 収束 | 記法 | 定義 | 強さ |
|------|------|------|------|
| 概収束（a.s.） | Xₙ →ᵃ·ˢ· X | P(lim Xₙ = X) = 1 | 最強 |
| 確率収束 | Xₙ →ᵖ X | ∀ε>0, P(|Xₙ-X|>ε)→0 | |
| 分布収束 | Xₙ →ᵈ X | Fₙ(x)→F(x) （連続点で） | 最弱 |
| Lᵖ収束 | Xₙ →ᴸᵖ X | E[|Xₙ-X|ᵖ]→0 | |

- 関係: a.s. ⇒ 確率 ⇒ 分布、Lᵖ ⇒ 確率

### 4.2 大数の法則

- **弱法則（WLLN）**: E[X]<∞ のとき X̄ₙ →ᵖ μ
- **強法則（SLLN）**: E[X]<∞ のとき X̄ₙ →ᵃ·ˢ· μ

### 4.3 中心極限定理（CLT）

- **古典CLT（リンデベルグ・レヴィ）**: iid, E[X]=μ, Var(X)=σ² < ∞ のとき
  ```
  √n(X̄ₙ - μ)/σ →ᵈ N(0,1)
  ```
- **リンデベルグCLT**: 独立（同分布でない）の場合。リンデベルグ条件を満たせば成立
- **多変量CLT**: √n(X̄ₙ - μ) →ᵈ N(0, Σ)
- **ベリー・エセーン定理**: CLTの近似精度 — sup|Fₙ(x)-Φ(x)| ≤ C·E[|X|³]/(σ³√n)

### 4.4 デルタ法（Delta Method）

#### 1次デルタ法

- √n(Tₙ - θ) →ᵈ N(0, σ²) かつ g が θ で微分可能なら:
  ```
  √n(g(Tₙ) - g(θ)) →ᵈ N(0, σ² · [g'(θ)]²)
  ```

#### 2次デルタ法

- g'(θ) = 0 の場合:
  ```
  n(g(Tₙ) - g(θ)) →ᵈ (σ²/2) · g''(θ) · χ²(1)
  ```

#### 多変量デルタ法

- √n(Tₙ - θ) →ᵈ N(0, Σ) のとき:
  ```
  √n(g(Tₙ) - g(θ)) →ᵈ N(0, ∇g(θ)ᵀ Σ ∇g(θ))
  ```

#### 実務活用例

| 変換 g(θ) | g'(θ) | 用途 |
|-----------|-------|------|
| log(θ) | 1/θ | 比率の信頼区間（対数変換） |
| 1/θ | -1/θ² | 率（rate）の推定 |
| θ² | 2θ | 分散の関数 |
| arcsin(√θ) | 1/(2√(θ(1-θ))) | 二項比率の分散安定化変換 |
| log(θ/(1-θ)) | 1/(θ(1-θ)) | オッズ比のロジット変換 |
| tanh⁻¹(θ) | 1/(1-θ²) | 相関係数のフィッシャー変換 |

### 4.5 U統計量

- **定義**: 対称カーネル h(x₁,...,xₘ) に対し
  ```
  Uₙ = C(n,m)⁻¹ Σ h(Xᵢ₁,...,Xᵢₘ)
  ```
  和は n 個から m 個を選ぶすべての組合せ
- **性質**: θ = E[h(X₁,...,Xₘ)] の最小分散不偏推定量
- **漸近正規性**: √n(Uₙ-θ) →ᵈ N(0, m²σ₁²) ここで σ₁² = Var(E[h|X₁])
- **例**: 標本分散（m=2, h=(x₁-x₂)²/2）、ウィルコクソン統計量

### 4.6 M推定量

- **定義**: Σρ(Xᵢ, θ) を最小化する θ（または Σψ(Xᵢ, θ)=0 の解）
- MLE は ρ = -log f の特殊ケース
- **漸近正規性**:
  ```
  √n(θ̂-θ₀) →ᵈ N(0, A⁻¹ B (A⁻¹)ᵀ)
  ```
  A = E[∂ψ/∂θ], B = E[ψψᵀ]（サンドウィッチ分散）
- **ロバスト推定**: フーバーの M推定量 ψ(x) = max(-c, min(x, c))

### 4.7 スラツキーの定理・連続写像定理

- **スラツキー**: Xₙ →ᵈ X, Yₙ →ᵖ c（定数）⇒ Xₙ+Yₙ →ᵈ X+c, XₙYₙ →ᵈ cX
- **連続写像定理**: Xₙ →ᵈ X かつ g が連続 ⇒ g(Xₙ) →ᵈ g(X)

## 5. 検出力分析とサンプルサイズ設計

### 5.1 検出力関数

- β(θ) = P_θ(X ∈ R)
- θ ∈ Θ₀ では β(θ) ≤ α（第1種過誤の制御）
- θ ∈ Θ₁ では β(θ) が大きいほど良い

### 5.2 効果量（Effect Size）

| 効果量 | 定義 | 小/中/大の目安（コーエン） |
|--------|------|---------------------------|
| コーエンのd | (μ₁-μ₂)/σ | 0.2 / 0.5 / 0.8 |
| コーエンのh | 2arcsin(√p₁)-2arcsin(√p₂) | 0.2 / 0.5 / 0.8 |
| コーエンのf | σ_μ/σ（分散分析） | 0.10 / 0.25 / 0.40 |
| η²（イータ二乗） | SS_B/SS_T | 0.01 / 0.06 / 0.14 |
| ω²（オメガ二乗） | (SS_B-(k-1)MS_W)/SS_T+MS_W | η²よりバイアスが小さい |
| コーエンのw | √(Σ(p₁ᵢ-p₀ᵢ)²/p₀ᵢ)（χ²検定） | 0.1 / 0.3 / 0.5 |
| オッズ比 OR | (p₁/(1-p₁))/(p₂/(1-p₂)) | 1.5 / 2.5 / 4.3 |
| 相関係数 r | r | 0.1 / 0.3 / 0.5 |

### 5.3 サンプルサイズ計算

#### 平均の差の検定（2群、等分散）

```
n = 2 · ((z_{α/2} + z_β) / d)²
```
- d = |μ₁-μ₂|/σ（コーエンのd）
- 例: d=0.5, α=0.05, 1-β=0.80 → n = 2·(1.96+0.84)²/0.25 = 63/群

#### 比率の差の検定（2群）

```
n = ((z_{α/2}√(2p̄q̄) + z_β√(p₁q₁+p₂q₂)) / (p₁-p₂))²
```
p̄ = (p₁+p₂)/2

#### 1標本t検定

```
n = ((z_{α/2} + z_β) / d)²
```

#### 分散分析（1元配置、k群）

```
n = (z_{α/2} + z_β)² / (f²) + (k-1)/(2k)  （各群の標本サイズ）
```
- ただし正確には F 分布の非心パラメータ λ = nkf² を用いる

### 5.4 検出力分析の種類

| 分析タイプ | 既知 | 求める |
|-----------|------|--------|
| 事前分析（a priori） | α, 1-β, 効果量 | n |
| 事後分析（post hoc） | α, n, 効果量 | 1-β |
| 感度分析 | α, n, 1-β | 効果量 |
| 基準分析 | n, 1-β, 効果量 | α |

### 5.5 実務上の考慮事項

- **脱落率の考慮**: 必要 n を (1-脱落率) で割る
- **不等サイズ**: n₁:n₂ の比が 1:1 から離れるほど検出力低下
- **多重比較の考慮**: ボンフェローニ等で α を調整してから計算
- **クラスター RCT**: デザイン効果 DE = 1 + (m-1)·ICC で n を DE 倍
- **逐次検定**: 中間解析を行う場合はα消費関数（オブライエン・フレミング等）で調整

## 6. 補足トピック

### 6.1 順序統計量と推定

- X₍₁₎ ≤ X₍₂₎ ≤ ... ≤ X₍ₙ₎
- **標本中央値**: X₍(n+1)/2₎ または (X₍n/2₎+X₍n/2+1₎)/2
  - 漸近分散: 1/(4n·f(m)²)（m は母中央値、f は密度関数）
- **標本分位数**: Q̂(p) の漸近分散 = p(1-p)/(n·f(Q(p))²)
- **範囲**: R = X₍ₙ₎ - X₍₁₎ → σ の推定に使用（管理図の d₂ 定数）
- **順序統計量の密度**: f_{X₍ₖ₎}(x) = n!/(k-1)!(n-k)! · F(x)^(k-1) · (1-F(x))^(n-k) · f(x)

### 6.2 フィッシャー情報量の応用

- **実験計画**: 情報量を最大化するデザイン（D最適計画: det(I(θ)) 最大化）
- **逐次実験**: 現在の推定に基づき次の観測を最適化
- **欠測データ**: 観測情報量 < 完全情報量。欠測データの割合 ≈ 1 - I_obs/I_complete

### 6.3 経験尤度（Empirical Likelihood）

- パラメトリックな分布仮定なしで尤度比検定を構成
- EL比: R(θ) = max{∏ᵢpᵢ : Σpᵢg(Xᵢ,θ)=0, Σpᵢ=1, pᵢ≥0}
- ウィルクス型定理: -2 log R(θ₀) →ᵈ χ²(r)
- 信頼領域が自動的にデータ形状に適応（対称性不要）

### 6.4 ブートストラップ法

- **ノンパラメトリックブートストラップ**: 経験分布 F̂ₙ から復元抽出して統計量の分布を推定
- **パラメトリックブートストラップ**: 推定された分布 F(θ̂) から生成
- **ブートストラップ信頼区間**:
  - パーセンタイル法: [θ̂*_{α/2}, θ̂*_{1-α/2}]
  - BCa法（バイアス補正加速法）: バイアスと歪みを補正 — 最も推奨
  - ブートストラップt法: スチューデント化してからパーセンタイル
- **ブートストラップ検定**: p* = (帰無仮説下で |T*| ≥ |T_obs| の割合)
- **理論的正当化**: ブートストラップは CLT の改良（エッジワース展開で O(n⁻¹) の精度）

### 6.5 許容性・ミニマックス推定

- **許容性（Admissibility）**: 他のどの推定量にも MSE で支配されない
- **ミニマックス推定量**: sup_θ R(θ,δ) を最小化する δ
- **完備類定理**: ベイズ推定量の適切な極限は許容的
- **ミニマックスとベイズの関係**: 最悪ケース事前分布に対するベイズ推定量はミニマックス

### 6.6 決定理論の枠組み

| 概念 | 定義 |
|------|------|
| 行動空間 A | 推定量が取りうる値の集合 |
| 損失関数 L(θ,a) | パラメータ θ で行動 a を取ったときの損失 |
| リスク関数 R(θ,δ) | E_θ[L(θ,δ(X))] — 損失の期待値 |
| ベイズリスク | r(π,δ) = ∫R(θ,δ)π(θ)dθ |

## 公式・定理クイックリファレンス

### 主要不等式

| 不等式 | 内容 |
|--------|------|
| マルコフ | P(X≥a) ≤ E[X]/a （X≥0, a>0） |
| チェビシェフ | P(|X-μ|≥kσ) ≤ 1/k² |
| イェンセン | E[g(X)] ≥ g(E[X]) （g が凸） |
| コーシー・シュワルツ | |E[XY]|² ≤ E[X²]·E[Y²] |
| ホフディング | P(X̄-μ≥t) ≤ exp(-2nt²/(b-a)²) （a≤Xᵢ≤b） |

### 正規分布の重要な臨界値

| α (両側) | z_{α/2} | t₀.₀₂₅(∞) |
|-----------|---------|------------|
| 0.10 | 1.645 | 1.645 |
| 0.05 | 1.960 | 1.960 |
| 0.01 | 2.576 | 2.576 |
| 0.001 | 3.291 | 3.291 |

### χ² 分布の性質

- X ~ χ²(k) ⟹ E[X]=k, Var(X)=2k
- X ~ χ²(k₁), Y ~ χ²(k₂), 独立 ⟹ X+Y ~ χ²(k₁+k₂)
- nS²/σ² ~ χ²(n-1) （正規母集団）
- (X/k₁)/(Y/k₂) ~ F(k₁,k₂) （X~χ²(k₁), Y~χ²(k₂), 独立）

### t 分布の性質

- T = Z/√(V/k), Z~N(0,1), V~χ²(k), 独立 ⟹ T ~ t(k)
- E[T]=0 (k>1), Var(T)=k/(k-2) (k>2)
- k→∞ で N(0,1) に収束
- T² ~ F(1,k)

### 非心分布（検出力計算用）

- **非心t分布**: T ~ t(k, δ) — δ は非心パラメータ。検出力 = P(|T| > t_{α/2} | δ)
- **非心χ²分布**: χ²(k, λ) — λ = Σ(μᵢ/σ)²。分散分析の検出力計算
- **非心F分布**: F(k₁, k₂, λ) — 分散分析の検出力関数に使用
