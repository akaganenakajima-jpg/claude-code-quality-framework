> 統計検定1級水準の理論と実務活用を網羅する実践リファレンス

## 1. 多変量正規分布

### 1.1 定義

p次元確率ベクトル **x** が多変量正規分布 N_p(**μ**, **Σ**) に従うとき、密度関数は:

```
f(x) = (2π)^{-p/2} |Σ|^{-1/2} exp{ -½ (x - μ)ᵀ Σ⁻¹ (x - μ) }
```

- **μ**: p次元平均ベクトル（E[**x**] = **μ**）
- **Σ**: p×p 分散共分散行列（正定値対称）
- |**Σ**|: **Σ** の行列式

### 1.2 基本性質

**周辺分布**: **x** = (x₁, x₂)ᵀ の分割に対し、x₁ ~ N_{p₁}(μ₁, Σ₁₁) も正規分布。

**条件付き分布**:
```
x₁ | x₂ = a  ~  N(μ₁ + Σ₁₂ Σ₂₂⁻¹ (a - μ₂),  Σ₁₁ - Σ₁₂ Σ₂₂⁻¹ Σ₂₁)
```
条件付き平均は x₂ の線形関数、条件付き分散は x₂ に依存しない。
この性質が回帰分析の正規理論の基盤となる。

**線形変換**: A**x** + **b** ~ N(A**μ** + **b**, AΣAᵀ)

**二次形式**: (**x** - **μ**)ᵀ Σ⁻¹ (**x** - **μ**) ~ χ²(p)

**独立性**: 多変量正規分布では「無相関 ⟺ 独立」が成立（一般の分布では不成立）。

### 1.3 マハラノビス距離

```
D²(x, μ) = (x - μ)ᵀ Σ⁻¹ (x - μ)
```

分散共分散構造を考慮した距離。変数間の相関と尺度の違いを自動的に補正する。
標本版では Σ を標本分散共分散行列 S で置換。

用途: 外れ値検出、判別分析、クラスター分析の距離尺度。

### 1.4 ウィシャート分布

**x₁**, ..., **xₙ** が独立に N_p(**μ**, **Σ**) に従うとき:
```
W = Σᵢ (xᵢ - x̄)(xᵢ - x̄)ᵀ  ~  W_p(n-1, Σ)
```

ウィシャート分布 W_p(m, Σ) は、多変量正規分布における標本分散共分散行列の分布。
1変量のχ²分布の多変量版に相当する。

性質:
- E[W] = mΣ
- W₁ + W₂ ~ W_p(m₁ + m₂, Σ)（独立なら加法性あり）
- |W| の分布は Σ の検定に利用

### 1.5 ホテリングの T²

1標本問題: H₀: **μ** = **μ₀** の検定統計量:
```
T² = n (x̄ - μ₀)ᵀ S⁻¹ (x̄ - μ₀)
```

F変換:
```
(n - p) / {p(n - 1)} × T²  ~  F(p, n - p)
```

2標本問題: H₀: **μ₁** = **μ₂**（共通Σ仮定）:
```
T² = {n₁n₂ / (n₁ + n₂)} (x̄₁ - x̄₂)ᵀ S_pool⁻¹ (x̄₁ - x̄₂)
```

1変量のt検定の自然な多変量拡張。

### 1.6 多変量正規性の検定

**マーディアの歪度検定**:
```
b₁,p = (1/n²) Σᵢ Σⱼ {(xᵢ - x̄)ᵀ S⁻¹ (xⱼ - x̄)}³
```
帰無分布: nb₁,p / 6 ~ χ²{p(p+1)(p+2)/6}

**マーディアの尖度検定**:
```
b₂,p = (1/n) Σᵢ {(xᵢ - x̄)ᵀ S⁻¹ (xᵢ - x̄)}²
```
帰無分布: 正規近似 E[b₂,p] = p(p+2), Var は既知。

**ヘンツェ-ツィルクラー検定**: 特性関数に基づく一致検定。小標本でも検出力が高い。

### 1.7 実務活用

- **金融リスク管理**: ポートフォリオ収益率の同時分布モデリング。VaR計算の基盤
- **品質管理**: 多変量管制図（T²管制図）による工程監視
- **外れ値検出**: マハラノビス距離の χ² 分位点との比較で多次元外れ値を検出
- **MT法（マハラノビス・タグチ法）**: 品質工学での正常空間定義

## 2. 主成分分析 (PCA)

### 2.1 定義

分散共分散行列 Σ（または相関行列 R）の固有値分解:
```
Σ = PΛPᵀ     Λ = diag(λ₁, ..., λₚ),  λ₁ ≥ λ₂ ≥ ... ≥ λₚ ≥ 0
```

第k主成分: z_k = **p_k**ᵀ **x**（**p_k** は第k固有ベクトル）

性質:
- Var(z_k) = λ_k
- Cov(z_j, z_k) = 0（j ≠ k）
- z₁ は Var(aᵀx), ||a||=1 を最大化する線形結合

### 2.2 寄与率と次元選択

**寄与率**: λ_k / Σλᵢ（第k成分が全分散に占める割合）

**累積寄与率**: Σ_{i=1}^{k} λᵢ / Σλᵢ（上位k成分までの累積）

次元選択基準:
- **カイザー基準**: 相関行列使用時 λ_k > 1 の成分を採用
- **スクリープロット**: 固有値の減衰が緩やかになる「肘」で打ち切り
- **累積寄与率**: 70-90% を目安（分野による）
- **交差検証**: 予測目的では CV による選択が望ましい

### 2.3 相関行列 vs 分散共分散行列

| 基準 | 共分散行列 | 相関行列 |
|------|-----------|---------|
| 前提 | 変数の単位が同一 | 単位が異なる |
| 固有値 | 分散の大きい変数に支配される | 標準化で均等 |
| 解釈 | 元の単位で解釈可能 | 無次元 |
| 推奨場面 | 物理量（同一単位） | アンケート・混合単位 |

原則: 単位が異なる変数を混ぜるなら相関行列を使う。

### 2.4 カーネルPCA

非線形写像 φ: R^p → H（高次元特徴空間）を介した PCA。
カーネルトリックで内積 K(xᵢ, xⱼ) = φ(xᵢ)ᵀφ(xⱼ) のみ計算。

代表的カーネル:
- **RBF**: K(x, y) = exp(-γ||x - y||²)
- **多項式**: K(x, y) = (xᵀy + c)^d

グラム行列 K の固有値問題 Kα = nλα を解く。

### 2.5 スパースPCA

LASSO型のペナルティを加えて固有ベクトルのスパース性を誘導:
```
max aᵀΣa  subject to ||a||₂ = 1, ||a||₁ ≤ t
```

解釈性が向上（少数の変数で成分を構成）。高次元データに有効。

### 2.6 実務活用

- **次元削減**: 高次元データの前処理（数百変数→数十主成分）
- **多重共線性対策**: 回帰の説明変数に主成分を使用（主成分回帰）
- **可視化**: 第1-2主成分の散布図（バイプロット）
- **画像圧縮**: 固有顔（Eigenfaces）等の次元圧縮表現
- **異常検知**: 低寄与率成分のスコアが大きい点を異常とみなす（SPE統計量）

## 3. 因子分析

### 3.1 モデル

```
x = μ + Λf + ε
```

- **x**: p次元観測変数
- **f**: m次元共通因子（m < p）、f ~ N(0, I)
- **ε**: p次元独自因子、ε ~ N(0, Ψ)、Ψ = diag(ψ₁, ..., ψₚ)
- **Λ**: p×m 因子負荷行列

分散共分散構造:
```
Σ = ΛΛᵀ + Ψ
```

- **共通性** (communality): h²ᵢ = Σⱼ λ²ᵢⱼ（共通因子で説明される分散）
- **独自性** (uniqueness): ψᵢ = σ²ᵢ - h²ᵢ

### 3.2 推定法

**最尤法 (ML)**:
```
L(Λ, Ψ) ∝ |ΛΛᵀ + Ψ|^{-n/2} exp{-(n/2) tr[(ΛΛᵀ + Ψ)⁻¹ S]}
```
反復最適化（EM アルゴリズム等）で解く。尤度比検定で因子数を検定可能。

**主因子法**: S - Ψ̂ の固有値分解。Ψ̂ の初期値は SMC（重相関係数の二乗）等。
収束まで反復（iterated principal factor）。

**最小残差法 (MINRES)**: S - ΛΛᵀ - Ψ の非対角要素の二乗和を最小化。

### 3.3 因子回転

推定された Λ に直交行列 T を右からかけても Σ は不変（回転の不定性）。
解釈しやすい「単純構造」を求めるために回転する。

**直交回転**（因子間無相関を保持）:
- **バリマックス**: 各因子の負荷量の二乗の分散を最大化（最も普及）
- **クォーティマックス**: 各変数の負荷量パターンを単純化

**斜交回転**（因子間相関を許容）:
- **プロマックス**: バリマックス解をべき乗して歪めた後に斜交射影
- **ジオミン**: 負荷量の積の和を最小化
- **オブリミン**: 一般化パラメータ δ で回転制御

斜交回転では「構造行列」(因子と変数の相関) と「パターン行列」(偏回帰係数) を区別。

### 3.4 PCA との違い

| 観点 | PCA | 因子分析 |
|------|-----|---------|
| 目的 | 分散の最大化 | 潜在構造の推定 |
| モデル | 統計モデルなし | 潜在変数モデル |
| 誤差 | 考慮しない | 独自因子 ε |
| 解 | 一意 | 回転の不定性あり |
| 適用 | 次元削減 | 構成概念の測定 |

### 3.5 因子数の決定

- **MAP基準**: 偏相関行列の平均二乗偏相関を最小化する因子数
- **平行分析**: ランダムデータの固有値と比較（最も推奨される方法の一つ）
- **BIC/AIC**: ML推定時の情報量規準
- **適合度検定**: ML推定の χ² 検定（大標本で棄却されやすい欠点あり）

### 3.6 実務活用

- **心理測定**: 知能の g 因子、性格の Big Five 等の構成概念抽出
- **顧客セグメンテーション**: アンケート項目から潜在ニーズ因子を抽出
- **尺度開発**: 因子負荷量に基づく項目選定、信頼性評価（ω係数）
- **マーケティング**: ブランドイメージの因子構造把握

## 4. 判別分析

### 4.1 線形判別分析 (LDA)

2群の判別: 群1 ~ N(μ₁, Σ), 群2 ~ N(μ₂, Σ)（共通分散共分散を仮定）。

**フィッシャーの判別関数**: 群間分散/群内分散の比を最大化する線形結合:
```
a = S_W⁻¹ (x̄₁ - x̄₂)
```

**判別スコア**: y = aᵀx

**判別境界**: y = aᵀ(x̄₁ + x̄₂)/2（事前確率等しい場合）

多群への拡張: min(g-1, p) 個の判別関数。群間分散行列 S_B と群内分散行列 S_W の一般化固有値問題 S_B a = λ S_W a を解く。

### 4.2 二次判別分析 (QDA)

群ごとに異なる分散共分散行列を許容: Σ₁ ≠ Σ₂。

判別関数が二次式になる:
```
δ_k(x) = -½ log|Σ_k| - ½ (x - μ_k)ᵀ Σ_k⁻¹ (x - μ_k) + log π_k
```

LDA vs QDA: Σ が共通なら LDA（パラメータ少、安定）、異なるなら QDA（柔軟、要大標本）。

### 4.3 ベイズ判別

事後確率に基づく分類:
```
P(G = k | x) ∝ f_k(x) π_k
```

**最小誤判別率**: 事後確率最大のクラスに分類。
**最小コスト**: 誤分類コスト行列を考慮した期待損失最小化。

誤判別率の推定:
- **再代入誤り率**: 楽観的バイアスあり
- **交差検証 (LOO-CV)**: ほぼ不偏
- **ブートストラップ .632+**: LOO-CVとの折衷

### 4.4 正則化判別分析

高次元（p > n）では S_W が特異行列になる。

**正則化LDA (RDA)**: S̃ = αS_W + (1-α)diag(S_W) や リッジ型 S_W + λI
**収縮推定**: Ledoit-Wolf 収縮推定量で Σ を推定

### 4.5 実務活用

- **信用スコアリング**: 貸倒リスクの2群判別（デフォルト/正常）
- **医療診断**: 検査値のプロファイルから疾患の有無を判別
- **スパムフィルター**: テキスト特徴量による分類（ナイーブベイズと併用）
- **製造業**: 良品/不良品の自動判別（マハラノビス距離ベース）

## 5. 正準相関分析

### 5.1 定義

2組の変数群 **x** (p次元) と **y** (q次元) の間の線形関係を分析する。

相関が最大となる線形結合のペア（正準変数）を逐次的に求める:
```
u₁ = a₁ᵀx,  v₁ = b₁ᵀy     Corr(u₁, v₁) = ρ₁  (最大)
u₂ = a₂ᵀx,  v₂ = b₂ᵀy     Corr(u₂, v₂) = ρ₂  (u₁,v₁と無相関の下で最大)
...
```

正準相関の数: min(p, q) 個。

### 5.2 数学的定式化

分散共分散行列を分割:
```
Σ = ( Σ_xx  Σ_xy )
    ( Σ_yx  Σ_yy )
```

正準相関係数 ρ_k² は次の行列の固有値:
```
Σ_xx⁻¹ Σ_xy Σ_yy⁻¹ Σ_yx
```

### 5.3 検定

**ウィルクスの Λ**: H₀: ρ_{k+1} = ... = ρ_s = 0
```
Λ_k = Π_{i=k+1}^{s} (1 - ρ̂ᵢ²)
```
-{n - 1 - (p+q+1)/2} log Λ_k ~ χ²{(p-k)(q-k)} （近似）

逐次検定で有意な正準相関の数を決定する。

### 5.4 冗長性分析

正準変数 u が y の分散をどれだけ説明するかを定量化。
正準相関が高くても冗長性指標が低い場合がある（変数群内の説明力が低い）。

Stewart-Love の冗長性指標:
```
Rd(y|u_k) = ρ_k² × (bₖ が y の分散を説明する割合)
```

### 5.5 実務活用

- **マーケティング**: 顧客属性群と購買行動群の関連構造
- **教育**: 入試成績群と大学GPA群の関連分析
- **生態学**: 環境変数群と種分布群の対応関係
- **脳科学**: fMRI脳活動パターンと行動指標の結合分析

## 6. クラスター分析

### 6.1 階層的手法

**凝集型**: 各点を1クラスターとし、順次統合していく。

距離の更新法:
- **最短距離法** (single linkage): min d(i,j)。鎖効果（chaining）の問題
- **最長距離法** (complete linkage): max d(i,j)。球状クラスター向き
- **群平均法** (UPGMA): 平均距離。バランスの良い手法
- **ウォード法**: 統合による群内平方和の増分を最小化。最も広く使われる
- **重心法** (centroid): 重心間距離。逆転現象が起こりうる

Lance-Williams の更新式: 全手法を統一的に記述可能。

**デンドログラム**: 統合過程のツリー表現。切断位置でクラスター数を決定。

### 6.2 非階層的手法

**k-means**:
1. k個の初期中心を選択
2. 各点を最近接中心に割当
3. 中心を再計算
4. 収束まで 2-3 を繰り返す

目的関数: Σ_k Σ_{i∈C_k} ||xᵢ - μ_k||² （群内平方和の最小化）

局所最適解に収束。初期値依存性あり → k-means++（初期値選択の改善）。

**k-medoids (PAM)**: 中心を実データ点に限定。外れ値に頑健。

**混合ガウスモデル (GMM)**: EMアルゴリズムで推定。ソフト割当（確率的帰属）。
BICでモデル選択（成分数 k と共分散構造）。

### 6.3 距離尺度

| 距離 | 定義 | 特徴 |
|------|------|------|
| ユークリッド | √{Σ(xᵢ - yᵢ)²} | 標準、尺度に依存 |
| マハラノビス | √{(x-y)ᵀ S⁻¹ (x-y)} | 相関・尺度を補正 |
| マンハッタン | Σ|xᵢ - yᵢ| | 外れ値に頑健 |
| コサイン | 1 - (x·y)/(||x||||y||) | 方向のみ、大きさ無視 |
| 相関距離 | 1 - r(x, y) | プロファイルの類似性 |

### 6.4 クラスター数の決定

**シルエット係数**: s(i) = {b(i) - a(i)} / max{a(i), b(i)}
- a(i): 同一クラスター内の平均距離
- b(i): 最近接他クラスターへの平均距離
- 範囲: [-1, 1]。1に近いほど良い分離

**ギャップ統計量**: 観測データの群内分散とランダム一様分布の期待値との差:
```
Gap(k) = E*[log W_k] - log W_k
```
Gap が最大（または初めて Gap(k) ≥ Gap(k+1) - s_{k+1}）の k を選択。

**エルボー法**: W_k のプロットで減衰が緩やかになる点（主観的判断を伴う）。

**カリンスキー-ハラバシュ指標**: 群間分散/群内分散の比率。大きいほど良い。

### 6.5 実務活用

- **顧客セグメンテーション**: RFM分析での購買行動クラスタリング
- **異常検知**: 正常クラスターから遠い点を異常と判定（DBSCAN等の密度ベース手法も有効）
- **遺伝子発現解析**: 共発現パターンによる遺伝子グルーピング
- **画像セグメンテーション**: ピクセル値のクラスタリングによる領域分割

## 7. 多変量分散分析 (MANOVA)

### 7.1 モデル

g群の多変量データに対する1元配置 MANOVA:
```
xᵢⱼ = μ + τᵢ + εᵢⱼ     (i=1,...,g;  j=1,...,nᵢ)
```
H₀: τ₁ = τ₂ = ... = τ_g = **0**

群間変動行列: H = Σᵢ nᵢ (x̄ᵢ - x̄)(x̄ᵢ - x̄)ᵀ
群内変動行列: E = Σᵢ Σⱼ (xᵢⱼ - x̄ᵢ)(xᵢⱼ - x̄ᵢ)ᵀ

### 7.2 検定統計量

| 統計量 | 定義 | 特徴 |
|--------|------|------|
| ウィルクスの Λ | \|E\| / \|E + H\| | 最も標準的、尤度比ベース |
| ピライのトレース | tr(H(E+H)⁻¹) | 帰無分布からの乖離に頑健 |
| ホテリング-ローリーのトレース | tr(HE⁻¹) | LDA固有値の和 |
| ロイの最大根 | max eigenvalue of HE⁻¹ | 1方向の差に敏感、検出力最高（ただし正確分布が複雑） |

各統計量のF近似は Rao, Pillai, McKeon 等の近似式による。

### 7.3 仮定と頑健性

- 多変量正規性
- 分散共分散行列の等質性（Box の M 検定で確認）
- 独立性

ピライのトレースは等分散性違反・非正規性に最も頑健。
群間で標本サイズが大きく異なる場合は注意が必要。

### 7.4 事後分析

MANOVA が有意 → どの応答変数・群の差が寄与しているかを特定:
- 各応答変数の一元配置 ANOVA（Bonferroni 補正）
- 判別分析の判別関数係数を解釈
- 同時信頼区間（Roy-Bose 法、Bonferroni 法）

### 7.5 実務活用

- **臨床試験**: 複数エンドポイント（血圧・心拍・体重等）の同時検定
- **教育研究**: 教授法の違いが複数科目の成績に与える影響
- **製造業**: 工程条件の変更が複数品質特性に与える影響
- **Type I エラー制御**: 個別 ANOVA の多重比較問題を回避

## 8. 構造方程式モデリング (SEM)

### 8.1 基本構造

**測定モデル**（確認的因子分析 CFA）:
```
x = Λ_x ξ + δ     (外生指標)
y = Λ_y η + ε     (内生指標)
```

**構造モデル**（パス解析）:
```
η = Bη + Γξ + ζ
```

- ξ: 外生潜在変数、η: 内生潜在変数
- B: 内生変数間の係数行列（対角要素は0）
- Γ: 外生→内生の係数行列
- ζ: 構造方程式の誤差

### 8.2 推定法

**最尤法 (ML)**:
```
F_ML = log|Σ(θ)| + tr{SΣ(θ)⁻¹} - log|S| - p
```
Σ(θ) はモデルから導かれる共分散行列、S は標本共分散行列。

**重み付き最小二乗法 (WLS/DWLS)**: カテゴリカル変数や非正規データに対応。
**ベイズ推定**: 事前分布を指定してMCMCで事後分布を得る。小標本に有効。

### 8.3 適合度指標

| 指標 | 基準 | 性質 |
|------|------|------|
| χ² 検定 | p > 0.05 | 標本サイズ依存（大標本で棄却されやすい） |
| GFI | > 0.90 | 分散説明率（パラメータ数の影響を受ける） |
| CFI | > 0.95 | 独立モデルとの相対比較（推奨） |
| RMSEA | < 0.05 (良), < 0.08 (許容) | 近似の悪さ（90%CI付きで報告） |
| SRMR | < 0.08 | 標準化残差の二乗平均平方根 |

Hu & Bentler (1999) の基準: CFI ≥ 0.95 かつ SRMR ≤ 0.08 または RMSEA ≤ 0.06 を推奨。

### 8.4 識別条件

必要条件: 推定パラメータ数 ≤ p(p+1)/2（観測共分散行列の要素数）。
十分条件はモデル依存（2指標ルール、3指標ルール等）。

自由度 = p(p+1)/2 - 推定パラメータ数 ≥ 0 が必要。

### 8.5 媒介分析

```
X → M → Y （間接効果: ab）
X → Y     （直接効果: c'）
```

間接効果の検定:
- **ソーベル検定**: z = ab / √(a²s_b² + b²s_a²)（正規近似、低検出力）
- **ブートストラップ法**: 間接効果 ab の信頼区間を直接構成（推奨）
- **モンテカルロ法**: a, b の標本分布から ab の分布をシミュレーション

### 8.6 多母集団分析

複数グループで同一モデルを当てはめ、等値制約を段階的に課す:
1. **配置不変性**: 同じ因子構造
2. **弱不変性**: 因子負荷量が等しい（測定不変性）
3. **強不変性**: 切片も等しい（平均の比較が可能に）
4. **厳密不変性**: 誤差分散も等しい

各段階で ΔCFI ≤ 0.01 なら不変性を支持（Chen, 2007）。

### 8.7 実務活用

- **因果推論**: 理論に基づく変数間の因果経路の検証（観察データの制約に注意）
- **尺度開発**: CFA による構成概念妥当性の確認
- **媒介・調整分析**: マーケティング施策の効果メカニズム解明
- **縦断分析**: 潜在成長曲線モデルによる変化パターンの分析

## 9. 対応分析

### 9.1 単純対応分析

r×c の分割表 N = (nᵢⱼ) を対象とする。

**対応行列**: P = N / n（相対度数行列）
**行プロファイル**: (pᵢⱼ / pᵢ.)_j
**列プロファイル**: (pᵢⱼ / p.ⱼ)_i

標準化残差行列の特異値分解:
```
D_r^{-1/2} (P - rc') D_c^{-1/2} = UDᵧVᵀ
```

ここで D_r = diag(pᵢ.), D_c = diag(p.ⱼ)。

### 9.2 カイ二乗距離と慣性

**カイ二乗距離**: 行プロファイル間の加重ユークリッド距離:
```
d²χ(i, i') = Σⱼ (1/p.ⱼ)(pᵢⱼ/pᵢ. - pᵢ'ⱼ/pᵢ'.)²
```

**総慣性**: φ² = χ² / n（分割表全体の関連度）

各次元の慣性 = γ²_k（特異値の二乗）。慣性の分解が寄与率に対応。

### 9.3 バイプロット

行カテゴリと列カテゴリを同一平面にプロットする。

表示座標の種類:
- **主座標**: F = D_r^{-1/2} U D_γ（行の主座標は加重カイ二乗距離を保存）
- **標準座標**: Φ = D_r^{-1/2} U（寄与率の重みなし）

対称プロット（行列とも主座標）は視覚的にわかりやすいが、行間距離と列間距離の尺度が異なる点に注意。

### 9.4 多重対応分析 (MCA)

3変数以上のカテゴリ変数に拡張。指示行列（バート行列）に対応分析を適用。

固有値の過小評価問題 → Greenacre の修正慣性、JCA（Joint Correspondence Analysis）。

### 9.5 実務活用

- **アンケート分析**: 自由回答カテゴリの布置関係を2次元で可視化
- **テキストマイニング**: 単語×文書の分割表から潜在意味空間を抽出
- **生態学**: 種×サイト行列の対応関係分析（CCA: 正準対応分析へ拡張）
- **マーケティング**: ブランドイメージ×属性の知覚マップ

## 10. 多次元尺度構成法 (MDS)

### 10.1 計量MDS（古典的MDS）

非類似度行列 D = (dᵢⱼ) から、ユークリッド空間の座標 X を復元する。

**Torgerson の方法**:
1. 二重中心化: B = -½ J D² J（J = I - 11ᵀ/n）
2. B の固有値分解: B = VAVᵀ
3. X = V_k A_k^{1/2}（上位 k 個の正の固有値）

B が半正定値なら D はユークリッド距離と整合（ユークリッド性条件）。

### 10.2 非計量MDS

順序情報のみを利用: dᵢⱼ < dₖₗ ⟹ d̂ᵢⱼ ≤ d̂ₖₗ（単調性制約）。

**ストレス関数**（Kruskal の STRESS-1）:
```
STRESS = √{ Σ(dᵢⱼ - d̂ᵢⱼ)² / Σ dᵢⱼ² }
```

- < 0.05: 良い適合
- 0.05-0.10: 許容
- 0.10-0.20: やや不良
- > 0.20: 不良

SMACOF アルゴリズム（majorization）で反復最適化。

### 10.3 次元数の決定

- ストレスの肘: 次元数を増やした時のストレス減少が緩やかになる点
- Shepard 図: 元の非類似度 vs 復元距離の散布図（単調関係を確認）
- 解釈可能性: 低次元（2-3次元）で解釈できるか

### 10.4 個人差MDS (INDSCAL)

複数の非類似度行列（複数被験者/条件）を同時に分析:
```
dᵢⱼ(t) = √{ Σ_k w_k(t) (x_ik - x_jk)² }
```

共通空間 X と個人ウェイト w(t) を同時推定。
次元の重みの違いで個人差を表現。

### 10.5 実務活用

- **知覚マップ**: ブランド間の類似性評定から知覚空間を構成
- **分子生物学**: 遺伝的距離からの系統関係可視化
- **推薦システム**: アイテム間類似度の低次元埋め込み
- **自然言語処理**: 単語ベクトル空間の可視化（t-SNE, UMAPとの併用）

## 付録: 手法選択ガイド

| 目的 | 推奨手法 | 備考 |
|------|---------|------|
| 次元削減（量的） | PCA | 前処理として最も汎用 |
| 潜在構造の推定 | 因子分析 | 構成概念を測定したい場合 |
| 群分類（教師あり） | LDA/QDA | Σ共通なら LDA、異なるなら QDA |
| 群分類（教師なし） | クラスター分析 | 目的に応じて階層/非階層を選択 |
| 2変数群の関連 | 正準相関分析 | 変数群間の構造把握 |
| 群差の多変量検定 | MANOVA | Type I エラー制御 |
| 因果モデルの検証 | SEM | 理論に基づくモデル構築が前提 |
| カテゴリ変数の可視化 | 対応分析 | 分割表データに限定 |
| 類似度データの可視化 | MDS | 距離/非類似度行列が入力 |

## 付録: 多変量解析の前提条件チェックリスト

1. **標本サイズ**: 変数の5-20倍が目安（SEM: 200以上推奨、因子分析: 変数の5-10倍）
2. **多変量正規性**: Mardia, Henze-Zirkler 検定。非正規なら頑健推定法を検討
3. **多重共線性**: VIF > 10 または条件数 > 30 なら変数の除外・統合を検討
4. **外れ値**: マハラノビス距離の χ² 分位点で検出。影響分析を実施
5. **欠測値**: MCAR/MAR/MNAR の判定。多重代入法 (MI) または FIML を推奨
6. **線形性**: 残差プロット、偏回帰プロットで確認。非線形なら変数変換やカーネル法
7. **等分散性**: Box の M 検定（MANOVA/LDA）。違反時は正則化や WLS を検討
