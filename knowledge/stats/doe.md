## Part A: 実験計画法 (Design of Experiments)

### 1. フィッシャーの3原則

実験計画の基盤をなす3原則。いずれも省略すると因果推論の正当性が崩れる。

**反復 (Replication)**
- 同一処理を複数の実験単位に適用し、誤差分散の推定を可能にする
- 反復数 n が大きいほど検出力が上がる: 検出力 = P(H₀棄却 | H₁真)
- 必要反復数の決定: δ（効果量）、σ²（誤差分散）、α、1-β を指定して逆算
  - n ≥ 2(z_{α/2} + z_β)²σ² / δ²（2群等分散の場合）

**無作為化 (Randomization)**
- 処理の割り当てを確率的に行い、未測定交絡因子の影響を期待値レベルで均等化
- ルービンの因果モデル (Rubin Causal Model) との接続:
  - 潜在結果 Y_i(1), Y_i(0) が処理割当 W_i と独立: {Y_i(1), Y_i(0)} ⊥ W_i
  - 無作為化がこの独立性を保証 → ATE = E[Y(1)] - E[Y(0)] の不偏推定が可能
- フィッシャーの鋭帰無仮説: H₀: Y_i(1) = Y_i(0) ∀i（個体レベルで効果なし）

**局所管理 (Local Control)**
- 既知の変動要因をブロックとして管理し、誤差分散を縮小
- ブロック化により SS_E が減少 → F値が増大 → 検出力向上
- ブロック内の均質性が高いほど効果的

**実務活用**: A/Bテストでは無作為化（ランダム割当）が必須。バッチ処理やサーバー差異はブロック因子として扱い、反復数はMDE（最小検出効果量）から逆算する。

### 2. 完全無作為化計画 (Completely Randomized Design)

**一元配置分散分析 (One-way ANOVA)**

a 群、各群 n_j 個（合計 N = Σn_j）のモデル:

```
Y_ij = μ + τ_i + ε_ij,  ε_ij ~ N(0, σ²),  Σn_iτ_i = 0
```

**分散分析表**

| 要因 | SS | df | MS | F |
|------|----|----|----|----|
| 処理間 (B) | SS_B = Σn_j(Ȳ_j - Ȳ)² | a-1 | MS_B = SS_B/(a-1) | F = MS_B/MS_W |
| 処理内 (W) | SS_W = ΣΣ(Y_ij - Ȳ_j)² | N-a | MS_W = SS_W/(N-a) | |
| 全体 (T) | SS_T = ΣΣ(Y_ij - Ȳ)² | N-1 | | |

- SS_T = SS_B + SS_W（平方和の分解）
- H₀: τ₁ = τ₂ = … = τ_a = 0 のもとで F ~ F(a-1, N-a)

**効果量**
- η² = SS_B / SS_T（母集団では過大推定）
- ω² = (SS_B - (a-1)MS_W) / (SS_T + MS_W)（不偏推定量）
- Cohen の基準: 小=0.01, 中=0.06, 大=0.14

**等分散性の検定**
- バートレット検定: 正規性を仮定。χ² 近似。正規分布からの逸脱に敏感
- ルビーン検定: |Y_ij - Ȳ_j| に対するANOVA。ロバスト（非正規でも使える）
- ブラウン・フォーサイス検定: |Y_ij - Med_j| 版。ルビーンより保守的

**多重比較法**

| 方法 | 比較対象 | FWER制御 | 特徴 |
|------|----------|----------|------|
| テューキーHSD | 全対比較 | 正確 | スチューデント化範囲分布 q |
| ダネット | 対コントロール | 正確 | a-1 本の比較のみ |
| シェフェ | 全対比・対比 | 保守的 | 任意のコントラストに適用可 |
| ボンフェローニ | 任意 | 保守的 | α/m（比較数m） |
| ホルム | 任意 | ステップダウン | ボンフェローニより検出力高い |

**実務活用**: システムの応答時間を3つの設定で比較する場合、一元配置ANOVAで全体効果を確認後、テューキーHSDで対ごとの差を特定する。等分散が疑わしければウェルチのANOVAを使用。

### 3. 乱塊法 (Randomized Complete Block Design)

ブロック因子 b 水準、処理因子 a 水準のモデル:

```
Y_ij = μ + τ_i + β_j + ε_ij,  ε_ij ~ N(0, σ²)
```

**分散分析表**

| 要因 | SS | df |
|------|----|----|
| 処理 (B) | SS_B = bΣ(Ȳ_i. - Ȳ)² | a-1 |
| ブロック | SS_Block = aΣ(Ȳ_.j - Ȳ)² | b-1 |
| 誤差 (E) | SS_E = SS_T - SS_B - SS_Block | (a-1)(b-1) |

- 交互作用項なし（1セル1観測のため推定不能）
- ブロック効果が大きいほど SS_E が小さくなり検出力向上

**フリードマン検定**（ノンパラメトリック版）
- 各ブロック内で順位づけ: R_ij = rank of Y_ij within block j
- 検定統計量: Q = 12/(ba(a+1)) × Σ(R_i.)² - 3b(a+1)
- Q ~ χ²(a-1)（漸近分布）

**実務活用**: 複数サーバー（ブロック）上で3種のアルゴリズム（処理）のレイテンシを比較。サーバー間変動をブロックで除去し、アルゴリズム間の純粋な差を検出。

### 4. 二元配置分散分析 (Two-way ANOVA)

因子A（a水準）×因子B（b水準）、各セル n 個のモデル:

```
Y_ijk = μ + α_i + β_j + (αβ)_ij + ε_ijk
```

**繰り返しありの分散分析表**

| 要因 | SS | df |
|------|----|----|
| A主効果 | SS_A | a-1 |
| B主効果 | SS_B | b-1 |
| 交互作用 A×B | SS_AB | (a-1)(b-1) |
| 誤差 | SS_E | ab(n-1) |

- 繰り返しなし（n=1）: SS_AB と SS_E が分離不能 → 交互作用なしの仮定が必要
- 交互作用が有意なら主効果の解釈は単独では不十分 → 単純主効果の分析へ

**交互作用プロット**: 各セル平均を折れ線で描画。線が平行なら交互作用なし。交差があれば交互作用の存在を示唆。

**Type I / II / III 平方和**（不均衡データで差異が生じる）

| Type | 定義 | 用途 |
|------|------|------|
| I (逐次型) | モデルに投入した順序で分解 | 順序に意味がある場合 |
| II (階層型) | 同格以下の項を調整、交互作用は調整しない | 交互作用なしの仮定 |
| III (偏型) | 全ての他の効果を調整 | 不均衡データの標準 |

- 均衡データでは I = II = III
- R の `car::Anova()` で Type II/III を指定可能

**実務活用**: キャッシュ方式（A）×スレッド数（B）の2要因実験。交互作用があれば「特定のキャッシュ方式は特定のスレッド数でのみ有効」という知見が得られる。

### 5. 要因計画 (Factorial Design)

**2^k 要因計画**

k 因子、各2水準（-1, +1）の完全実施。2^k 回の実験で全ての主効果・交互作用を推定。

コントラスト（効果の推定量）:
```
Effect_A = (1/2^{k-1}) × Σ(x_Ai × Y_i)
```
x_Ai は実験 i における因子Aの水準（±1）。

2³ の例: 主効果 A, B, C（3本）、2因子交互作用 AB, AC, BC（3本）、3因子交互作用 ABC（1本）。合計 2³-1 = 7 効果。

**一部実施要因計画 (Fractional Factorial: 2^{k-p})**

全実験が非現実的な場合、2^{k-p} 回に削減。p 個の生成子を定義。

**分解能 (Resolution)**

| 分解能 | 表記 | エイリアス構造 |
|--------|------|--------------|
| III | 2^{k-p}_III | 主効果が2因子交互作用と交絡 |
| IV | 2^{k-p}_IV | 主効果は2因子交互作用と交絡しない。2因子同士が交絡 |
| V | 2^{k-p}_V | 主効果・2因子交互作用が互いに交絡しない |

- 分解能 = 定義関係の最短ワード長
- 生成子 I = ABCD なら Resolution IV（ABCDの長さ4）

**エイリアス構造**: 定義関係から導出。I = ABD のとき A = BD, B = AD, D = AB。主効果Aの推定値にBDの交互作用が混入。

**実務活用**: Webサービスの性能チューニングで7因子（キャッシュ/圧縮/CDN/DB設定等）を 2^{7-4} = 8 回の実験で主効果をスクリーニング。Resolution III で十分（交互作用は2次スクリーニングで確認）。

### 6. 応答曲面法 (Response Surface Methodology)

逐次的にモデルを構築し、最適条件を探索する。

**Phase 1: 1次モデル + 最急降下法**
```
Y = β₀ + β₁x₁ + β₂x₂ + … + βₖxₖ + ε
```
- 2^k または 2^{k-p} で係数を推定
- 勾配方向 ∇Y = (β̂₁, β̂₂, …, β̂ₖ) に沿ってステップ移動
- 曲率が有意になるまで繰り返し（適合度の欠如検定）

**Phase 2: 2次モデル**
```
Y = β₀ + Σβᵢxᵢ + ΣΣβᵢⱼxᵢxⱼ + Σβᵢᵢxᵢ² + ε
```

**中心複合計画 (CCD: Central Composite Design)**
- 2^k 頂点 + 2k 軸点（±α）+ 中心点
- 回転可能性: α = (2^k)^{1/4}
- 直交ブロック化可能

**ボックス・ベンケン計画 (Box-Behnken)**
- 各因子は3水準（-1, 0, +1）、頂点を含まない
- CCDより実験回数が少ない場合がある（k≥3）
- 因子の極端な組合せを避けたい場合に有効

**定常点の分析**
- ∂Y/∂xᵢ = 0 を連立して定常点 x* を求める
- ヘッセ行列 H = 2B（B は2次項の係数行列）の固有値で判定:
  - 全負 → 極大、全正 → 極小、混在 → 鞍点

**実務活用**: 機械学習のハイパーパラメータ（学習率×バッチサイズ）をCCDで探索し、2次モデルでバリデーション精度の最適点を推定。グリッドサーチより効率的。

---

### 7. 分割法・直交配列

**分割法 (Split-plot Design)**

一部の因子が変更困難（hard-to-change）な場合の設計。

- 一次単位（whole plot）: 変更困難な因子を割当
- 二次単位（sub-plot）: 変更容易な因子を割当
- 誤差が2層: whole-plot誤差 ε_w と sub-plot誤差 ε_s
- whole-plot因子のF検定は ε_w を使用（df が小さく検出力が低い）

```
Y_ijk = μ + α_i + δ_ij + β_k + (αβ)_ik + ε_ijk
```
δ_ij ~ N(0, σ²_w): whole-plot誤差、ε_ijk ~ N(0, σ²_s): sub-plot誤差

**タグチメソッド**

直交表（L8, L12, L18 等）を用いた実験計画。

- **SN比** (Signal-to-Noise Ratio):
  - 望目特性: SN = 10 log₁₀(Ȳ²/s²)
  - 望小特性: SN = -10 log₁₀((1/n)ΣY²)
  - 望大特性: SN = -10 log₁₀((1/n)Σ(1/Y²))

- **内側配列 × 外側配列**: 制御因子（内側）と誤差因子（外側）を直積実験
- **確認実験**: 最適条件のSN比を再現実験で確認（加法性の検証）

**実務活用**: 製造工程の品質工学。温度（変更困難）を一次単位、添加剤配合（変更容易）を二次単位として分割法を適用。環境ノイズを外側配列に配置してロバスト設計。

---

### 8. ラテン方格 (Latin Square)

3つの変動要因を同時に管理。a×a の正方配列で、各処理が各行・各列に1回ずつ出現。

```
Y_ijk = μ + ρ_i + γ_j + τ_k + ε_ijk
```

| 要因 | df |
|------|----|
| 行 | a-1 |
| 列 | a-1 |
| 処理 | a-1 |
| 誤差 | (a-1)(a-2) |

- 制約: 因子数 = 水準数 = a。交互作用は推定不能。
- 誤差の df が小さい → a ≥ 5 が望ましい

**グレコラテン方格**: 2つのラテン方格を直交に重ね合わせ、4因子を同時管理。

**実務活用**: 臨床試験のクロスオーバーデザイン（被験者×期間×処理）。農業試験（行×列×品種）。

---

### 9. 共分散分析 (ANCOVA)

処理効果を共変量で調整。

```
Y_ij = μ + τ_i + β(X_ij - X̄) + ε_ij
```

- 共変量 X が Y に線形に影響する場合、SS_E が減少して検出力向上
- 調整平均: Ȳ_i(adj) = Ȳ_i - β̂(X̄_i - X̄)

**前提条件**:
1. 共変量と処理が独立（無作為化で保証）
2. 共変量と Y の関係が線形
3. **回帰傾きの等質性**: 全群で傾き β が同一（交互作用なし）
   - 検定: モデルに τ_i × X_ij 項を追加して有意性を確認

**実務活用**: 新アルゴリズムの処理速度を比較する際、入力データサイズ（共変量）で調整。データサイズの偏りが処理効果の推定を歪めるのを防ぐ。

---

### 10. 最適計画 (Optimal Design)

実験回数が限られる場合、情報行列 M = X'X を最大化する配置を選択。

| 基準 | 最適化対象 | 意味 |
|------|-----------|------|
| D最適 | max det(X'X) | パラメータ推定の同時精度最大化 |
| A最適 | min tr((X'X)⁻¹) | 推定量の分散の和を最小化 |
| G最適 | min max_x Var(ŷ(x)) | 予測分散の最大値を最小化 |
| I最適 | min ∫Var(ŷ(x))dx | 予測分散の積分を最小化 |

- D最適 ↔ G最適: 一般等価定理（Kiefer-Wolfowitz）
- 計算: 座標交換法（coordinate exchange）、候補点法

**実務活用**: 高コスト実験（半導体プロセス等）で、限られた予算内で最大の情報を得る計画を自動生成。Rの `AlgDesign` パッケージ等で実行可能。

## Part B: ノンパラメトリック統計・ロバスト手法

### 11. 1標本・2標本検定

**ウィルコクソンの符号順位検定 (Wilcoxon Signed-Rank Test)**
- 対象: 対応のある2群の差 D_i = Y_i1 - Y_i2
- H₀: D_i の分布が 0 を中心に対称
- 手順:
  1. |D_i| の順位 R_i を付ける（D_i = 0 は除外）
  2. W+ = Σ R_i × I(D_i > 0)
  3. H₀ のもとで E[W+] = n(n+1)/4, Var[W+] = n(n+1)(2n+1)/24
  4. 大標本近似: Z = (W+ - E[W+]) / √Var[W+] ~ N(0,1)

**マン・ホイットニーのU検定 (Mann-Whitney U / Wilcoxon Rank-Sum)**
- 対象: 独立2群
- H₀: P(X > Y) = 0.5（確率的優越なし）
- 全観測値を合わせて順位づけ → 各群の順位和から U を計算
- U = n₁n₂ + n₁(n₁+1)/2 - R₁
- 大標本近似: E[U] = n₁n₂/2, Var[U] = n₁n₂(n₁+n₂+1)/12

**符号検定 (Sign Test)**
- D_i の正負のみを使用（順位を無視）
- B = #{D_i > 0} ~ Bin(n, 0.5) under H₀
- 情報量が少ないが前提が最も弱い（対称性すら不要）

**同順位 (Ties) の処理**:
- 平均順位を付与
- 分散の修正項: Var修正 = Var × (1 - Σ(t³-t)/(N³-N))

**実務活用**: レイテンシ分布は典型的に右裾が長く正規性を満たさない。2つのシステム設定のレイテンシ比較にはマン・ホイットニー検定が適切。

### 12. 独立k標本

**クラスカル・ウォリス検定 (Kruskal-Wallis Test)**
- 一元配置ANOVAのノンパラメトリック版
- 全観測値を順位づけ → 各群の順位平均を比較
- 検定統計量:
```
H = (12/(N(N+1))) × Σ(n_j × (R̄_j - (N+1)/2)²)
```
- H ~ χ²(k-1)（漸近分布）
- 有意な場合 → 事後比較へ

**Steel-Dwass多重比較**
- 全対比較のノンパラメトリック版（テューキーHSDに対応）
- 各対についてウィルコクソン順位和統計量を計算し、スチューデント化範囲分布で補正
- FWER を制御

**実務活用**: 3つのクラウドリージョンのレスポンスタイム（非正規分布）を比較。クラスカル・ウォリスで全体差を確認後、Steel-Dwassでどのリージョン間に差があるか特定。

---

### 13. 相関

**スピアマンの順位相関 (Spearman's ρ)**
```
ρ_s = 1 - 6Σd²_i / (n(n²-1))
```
d_i = rank(X_i) - rank(Y_i)。ピアソン相関の順位版。単調関係を捉える。

**ケンドールの τ (Kendall's tau)**
```
τ = (C - D) / (n(n-1)/2)
```
C = 一致対数、D = 不一致対数。(X_i-X_j)(Y_i-Y_j) > 0 なら一致。

- τ_b: 同順位を考慮した修正版
  τ_b = (C - D) / √((n₀ - n₁)(n₀ - n₂))
  n₀ = n(n-1)/2, n₁ = ΣtX(tX-1)/2, n₂ = ΣtY(tY-1)/2

**ρ_s vs τ**:
- |ρ_s| ≥ |τ|（一般に）
- τ は確率的解釈が明確: P(一致) - P(不一致)
- τ の漸近分散は解析的に求まり、信頼区間が正確

**実務活用**: 機械学習モデルのランキング性能評価。複数モデルの予測順位と真の順位の相関にケンドール τ を使用（推薦システムのnDCGと併用）。

### 14. ノンパラメトリック回帰

**カーネル回帰（ナダラヤ・ワトソン推定量）**
```
m̂(x) = Σ K_h(x - X_i) Y_i / Σ K_h(x - X_i)
```
K_h(u) = (1/h)K(u/h)。局所加重平均。

- バイアス-分散トレードオフ: h↑ → バイアス↑分散↓、h↓ → バイアス↓分散↑
- 境界バイアス問題: データ端でバイアスが増大

**局所多項式回帰 (Local Polynomial Regression / LOESS)**
- 各点 x₀ で重み付き最小二乗:
  min_β Σ K_h(x₀ - X_i)(Y_i - β₀ - β₁(X_i-x₀) - … - β_p(X_i-x₀)^p)²
- m̂(x₀) = β̂₀
- 局所線形（p=1）: 境界バイアスを自動的に軽減（ナダラヤ・ワトソンの欠点を解消）
- LOESS/LOWESS: スパンパラメータ α（近傍割合）で帯域幅を制御

**カーネル密度推定 (KDE)**
```
f̂(x) = (1/nh) Σ K((x - X_i)/h)
```

主要カーネル:
| カーネル | K(u) | 効率 (AMISE比) |
|----------|------|----------------|
| ガウス | (2π)^{-1/2} exp(-u²/2) | 95.1% |
| エパネチニコフ | (3/4)(1-u²)I(|u|≤1) | 100% (最適) |
| 三角 | (1-|u|)I(|u|≤1) | 98.6% |

**バンド幅選択**
- Silverman のルール・オブ・サム: h = 0.9 × min(s, IQR/1.34) × n^{-1/5}
- プラグイン法: f'' の推定量を代入して AMISE を最小化
- 交差検証法 (LSCV): L̂(h) = ∫f̂²dx - (2/n)Σf̂_{-i}(X_i) を最小化
- バイアス付き交差検証 (BCV): AMISE の推定量を直接最小化

**収束レート**: MISE 最適で O(n^{-4/5})。パラメトリック O(n^{-1}) より遅い（次元の呪い）。

**実務活用**: 時系列データのトレンド抽出に LOESS を使用（季節性を除去した滑らかな傾向成分）。KDE はシステムメトリクスの分布可視化に有用（ヒストグラムのバンド幅依存性を軽減）。

### 15. 順列検定・ブートストラップ

**順列検定 (Permutation Test)**
- H₀: 2群のラベルが交換可能（同一分布）
- 手順:
  1. 観測された検定統計量 T_obs を計算
  2. ラベルを全順列（または十分な回数ランダムに）並べ替え → T の帰無分布を生成
  3. p値 = #{|T_perm| ≥ |T_obs|} / (全順列数)
- 正確検定: 全 C(n₁+n₂, n₁) 通りを列挙
- 近似: モンテカルロ順列（B=10,000回程度）

**ブートストラップ (Bootstrap)**

原標本 X₁,…,Xₙ から復元抽出で B 個のブートストラップ標本を生成。

信頼区間の構成法:

| 方法 | 定義 | 特徴 |
|------|------|------|
| パーセンタイル法 | [θ̂*_{α/2}, θ̂*_{1-α/2}] | 単純。バイアスに弱い |
| Basic法 | [2θ̂ - θ̂*_{1-α/2}, 2θ̂ - θ̂*_{α/2}] | パーセンタイルの反転 |
| BCa法 | バイアス補正+加速度補正 | 2次の正確性。推奨 |
| ブートストラップt法 | (θ̂*-θ̂)/ŝe* の分位点 | 分散推定が必要 |

**BCa法の詳細**:
- バイアス補正定数: ẑ₀ = Φ⁻¹(#{θ̂*_b < θ̂} / B)
- 加速度定数: â = Σ(θ̂_{(·)} - θ̂_{(-i)})³ / (6(Σ(θ̂_{(·)} - θ̂_{(-i)})²)^{3/2})
- 修正分位点: α₁ = Φ(ẑ₀ + (ẑ₀ + z_{α/2})/(1 - â(ẑ₀ + z_{α/2})))

**ジャックナイフ (Jackknife)**
- n 個の leave-one-out 推定量 θ̂_{(-i)} から:
  - バイアス推定: bias = (n-1)(θ̂_{(·)} - θ̂)
  - 分散推定: Var = ((n-1)/n) Σ(θ̂_{(-i)} - θ̂_{(·)})²
- ブートストラップより計算が軽いが、不連続統計量（中央値等）では不安定

**実務活用**: BETシステムのROI推定で信頼区間を構成する場合、収益分布は非正規（ゼロ多数+右裾）なので BCa ブートストラップが適切。順列検定は2つの戦略のパフォーマンス差の検定に使用。

### 16. ロバスト統計 (Robust Statistics)

外れ値や分布の逸脱に対して安定した推定を行う手法群。

**崩壊点 (Breakdown Point)**
- 推定量が無限大に発散するまでに必要な汚染割合の上限
- 標本平均: 0%（1つの外れ値で破綻）
- 中央値: 50%（最高の崩壊点）
- トリム平均（α-trimmed）: α

**M推定量 (M-estimator)**

Σρ(Y_i - θ) を最小化（ρ: 損失関数）。微分 ψ = ρ' を用いた推定方程式: Σψ(Y_i - θ) = 0

| 関数 | ρ(r) | ψ(r) | 特徴 |
|------|-------|-------|------|
| 最小二乗 | r²/2 | r | ロバストでない |
| Huber (c=1.345) | r²/2 (|r|≤c), c|r|-c²/2 (|r|>c) | r (|r|≤c), c·sign(r) (|r|>c) | 効率95%、崩壊点0% |
| Tukey biweight (c=4.685) | (c²/6)(1-(1-(r/c)²)³) (|r|≤c), c²/6 (|r|>c) | r(1-(r/c)²)² (|r|≤c), 0 (|r|>c) | 崩壊点~50%(再降下型) |

- Huber: 中心付近は二乗損失、遠方は線形 → 外れ値の影響を線形に制限
- Tukey biweight: 遠方で ψ=0 → 極端な外れ値を完全に無視（再降下型）
- IRLS (Iteratively Reweighted Least Squares) で計算

**中央値絶対偏差 (MAD)**
```
MAD = median(|X_i - median(X)|)
```
- 正規分布での標準偏差推定: σ̂ = 1.4826 × MAD
- 崩壊点 50%。標準偏差の代替としてロバスト

**トリム平均 (Trimmed Mean)**
```
X̄_α = (1/(n-2[nα])) × Σ X_{(i)}  (i = [nα]+1 to n-[nα])
```
- 上下 α×100% を除外した平均
- α=0: 標本平均、α=0.5: 中央値
- 通常 α=0.1〜0.2（10-20% トリム）

**ウィンザライズ平均 (Winsorized Mean)**
- トリムではなく、極値を境界値に置換してから平均
- X_{([nα]+1)} で下側を、X_{(n-[nα])} で上側を置換
- ウィンザライズ分散と組み合わせてロバストなt検定（Yuen検定）に使用

**ロバスト回帰**
- LMS (Least Median of Squares): median(r²_i) を最小化。崩壊点50%
- LTS (Least Trimmed Squares): Σ r²_{(i)} (i=1 to h) を最小化。h ≈ n/2
- MM推定量: 高崩壊点 + 高効率（S推定量で初期値 → M推定で仕上げ）

**実務活用**: BET収益データの要約統計にはトリム平均またはMADを使用（大当たり/大外れが平均を歪める）。異常検知のベースライン推定にはMADベースのZスコア: Z = (X - median) / MAD が有効。ロバスト回帰はセンサーデータの外れ値に頑健なトレンド推定に使用。

## 付録: 手法選択ガイド

### A. 群間比較の選択フロー

```
2群比較
├── 対応あり
│   ├── 正規性あり → 対応のあるt検定
│   └── 正規性なし → ウィルコクソン符号順位検定
└── 対応なし
    ├── 正規性+等分散 → 2標本t検定
    ├── 正規性+不等分散 → ウェルチのt検定
    └── 正規性なし → マン・ホイットニーU検定

k群比較
├── 対応あり
│   ├── 正規性あり → 反復測定ANOVA
│   └── 正規性なし → フリードマン検定
└── 対応なし
    ├── 正規性あり → 一元配置ANOVA + 多重比較
    └── 正規性なし → クラスカル・ウォリス + Steel-Dwass
```

### B. 実験計画の選択フロー

```
因子数
├── 1因子 → 完全無作為化計画 or 乱塊法
├── 2因子 → 二元配置 or ラテン方格（ブロック2つ）
├── 3-5因子 → 完全要因 2^k（実行可能なら）
├── 6因子以上 → 一部実施 2^{k-p}（スクリーニング）
└── 連続因子で最適化 → 応答曲面法（CCD or Box-Behnken）

特殊制約
├── 変更困難因子あり → 分割法
├── 実験回数厳密制限 → 最適計画（D/A/G最適）
└── ロバスト設計 → タグチメソッド（内側×外側配列）
```

### C. 検定の前提と頑健性

| 検定 | 正規性 | 等分散性 | 独立性 | 対称性 |
|------|--------|----------|--------|--------|
| t検定 | 必要(n大で緩和) | 必要(Welchで不要) | 必要 | - |
| ANOVA | 必要(n大で緩和) | 必要 | 必要 | - |
| Wilcoxon符号順位 | 不要 | - | - | 必要 |
| Mann-Whitney U | 不要 | 同形状仮定 | 必要 | - |
| 順列検定 | 不要 | 不要 | 交換可能性 | - |
| ブートストラップ | 不要 | - | i.i.d. | - |
