> 統計検定1級 満点レベル。各セクション末尾に実務活用を付記。

## 1. 線形回帰モデル

### 1.1 モデル定式化

$$y = X\beta + \varepsilon$$

- $y$: $n \times 1$ 応答ベクトル
- $X$: $n \times p$ 計画行列（$\text{rank}(X) = p$ を仮定）
- $\beta$: $p \times 1$ 回帰係数ベクトル
- $\varepsilon$: $n \times 1$ 誤差ベクトル、$E[\varepsilon] = 0$, $\text{Var}(\varepsilon) = \sigma^2 I_n$

### 1.2 OLS推定量の導出

残差二乗和 $S(\beta) = (y - X\beta)^T(y - X\beta)$ を最小化する。

$\partial S / \partial \beta = -2X^T(y - X\beta) = 0$ より**正規方程式**:

$$X^T X \hat{\beta} = X^T y$$

$$\hat{\beta}_{\text{OLS}} = (X^T X)^{-1} X^T y$$

- $\hat{\beta}$ は $\beta$ の不偏推定量: $E[\hat{\beta}] = \beta$
- 分散共分散行列: $\text{Var}(\hat{\beta}) = \sigma^2 (X^T X)^{-1}$
- $\sigma^2$ の不偏推定量: $\hat{\sigma}^2 = \text{RSS} / (n - p)$, ここで $\text{RSS} = \|y - X\hat{\beta}\|^2$

### 1.3 ガウス・マルコフの定理（BLUE）

**定理**: 誤差が $E[\varepsilon] = 0$, $\text{Var}(\varepsilon) = \sigma^2 I$ を満たすとき、$\hat{\beta}_{\text{OLS}}$ は $\beta$ の全ての線形不偏推定量の中で最小分散を持つ（Best Linear Unbiased Estimator）。

- 正規性は不要（1・2次モーメント条件のみ）
- 等分散・無相関が崩れると BLUE でなくなる → GLS（一般化最小二乗法）が必要

### 1.4 決定係数

**決定係数** $R^2$:

$$R^2 = 1 - \frac{\text{RSS}}{\text{TSS}} = 1 - \frac{\sum(y_i - \hat{y}_i)^2}{\sum(y_i - \bar{y})^2}$$

- $0 \leq R^2 \leq 1$（切片項ありモデル）
- 変数追加で必ず増加 → 過適合の指標にならない

**自由度調整決定係数** $R^2_{\text{adj}}$:

$$R^2_{\text{adj}} = 1 - \frac{\text{RSS}/(n-p)}{\text{TSS}/(n-1)} = 1 - \frac{n-1}{n-p}(1 - R^2)$$

- 不要な変数追加で減少しうる
- モデル比較に使用（同一データ内の変数数が異なるモデル間）

### 1.5 残差分析

回帰診断の4条件（LINE）:

| 仮定 | 検証方法 | 違反時の対処 |
|---|---|---|
| **L**inearity（線形性） | 残差 vs 予測値プロット | 多項式項・変数変換・GAM |
| **I**ndependence（独立性） | ダービン・ワトソン検定（DW≈2が理想） | 時系列モデル・GLS |
| **N**ormality（正規性） | QQプロット・シャピロ・ウィルク検定 | Box-Cox変換・ロバスト回帰 |
| **E**qual variance（等分散性） | 残差 vs 予測値のファンネル・ブロイシュ・ペーガン検定 | WLS・分散安定化変換 |

**標準化残差**: $r_i = e_i / (\hat{\sigma}\sqrt{1 - h_{ii}})$
**スチューデント化残差**: $t_i = e_i / (\hat{\sigma}_{(i)}\sqrt{1 - h_{ii}})$ — 外れ値検出に有効（$|t_i| > 3$ で要注意）

### 1.6 影響分析

**ハット行列**: $H = X(X^TX)^{-1}X^T$, $\hat{y} = Hy$

| 指標 | 定義 | 閾値目安 | 意味 |
|---|---|---|---|
| てこ比 $h_{ii}$ | $H$ の対角要素 | $> 2p/n$ | 説明変数空間での外れ値 |
| クックの距離 $D_i$ | $\sum(\hat{y}_j - \hat{y}_{j(i)})^2 / (p\hat{\sigma}^2)$ | $> 4/n$ または $> 1$ | 1点削除の全体影響 |
| DFFITS$_i$ | $(\hat{y}_i - \hat{y}_{i(i)}) / (\hat{\sigma}_{(i)}\sqrt{h_{ii}})$ | $> 2\sqrt{p/n}$ | 予測値への個別影響 |
| DFBETAS$_{j,i}$ | $(\hat{\beta}_j - \hat{\beta}_{j(i)}) / (\hat{\sigma}_{(i)}\sqrt{(X^TX)^{-1}_{jj}})$ | $> 2/\sqrt{n}$ | 各係数への個別影響 |

### 1.7 多重共線性

説明変数間に強い線形関係がある状態。推定量の分散が爆発する。

**VIF（分散拡大係数）**: $\text{VIF}_j = 1/(1 - R^2_j)$ — $R^2_j$ は $x_j$ を他の変数で回帰した決定係数
- VIF > 10 で多重共線性の疑い（保守的には > 5）

**条件数**: $\kappa(X^TX) = \lambda_{\max}/\lambda_{\min}$ — 固有値比
- $\kappa > 30$ で深刻な多重共線性

**対処**: 変数削除、主成分回帰（PCR）、リッジ回帰、PLS回帰

### 1.8 変数選択

| 方法 | 基準 | 特徴 |
|---|---|---|
| 前進選択法 | 有意な変数を逐次追加 | 局所最適に陥りやすい |
| 後退消去法 | 非有意な変数を逐次削除 | 全変数からスタート |
| ステップワイズ | 前進+後退を交互 | 最も一般的だが理論保証弱い |
| AIC | $-2\ln L + 2p$ | 予測精度重視、やや複雑なモデルを選ぶ |
| BIC | $-2\ln L + p\ln n$ | 一致性あり（$n \to \infty$ で真モデル選択）、AICよりスパース |
| Mallows' $C_p$ | $\text{RSS}_p/\hat{\sigma}^2 - n + 2p$ | $C_p \approx p$ で良いモデル |

### 実務活用
- **予測モデル構築**: 売上予測、需要予測で標準的に使用
- **因果推論の第一歩**: 交絡変数の調整に回帰分析。ただし因果解釈には注意（DAG/傾向スコア併用）
- **異常検知**: 残差が大きい観測点＝異常候補（影響分析と組み合わせ）
- **VIF監視**: 機械学習の特徴量エンジニアリングでも多重共線性チェックは有効

## 2. 一般化線形モデル (GLM)

### 2.1 3要素

1. **確率分布**: 応答変数 $Y$ が指数型分布族に従う
2. **線形予測子**: $\eta = X\beta$（説明変数の線形結合）
3. **リンク関数** $g$: $g(\mu) = \eta$, $\mu = E[Y]$

### 2.2 指数型分布族の標準形

$$f(y|\theta, \phi) = \exp\left\{\frac{y\theta - b(\theta)}{a(\phi)} + c(y, \phi)\right\}$$

- $\theta$: 自然パラメータ（正準パラメータ）
- $\phi$: 分散パラメータ（尺度パラメータ）
- $b(\theta)$: キュムラント関数 — $E[Y] = b'(\theta)$, $\text{Var}(Y) = a(\phi) \cdot b''(\theta)$

**分散関数** $V(\mu)$: $\text{Var}(Y) = a(\phi) \cdot V(\mu)$

| 分布 | $\theta$ | $b(\theta)$ | $V(\mu)$ | $a(\phi)$ |
|---|---|---|---|---|
| 正規 | $\mu$ | $\theta^2/2$ | $1$ | $\sigma^2$ |
| 二項 | $\ln(\mu/(1-\mu))$ | $n\ln(1+e^\theta)$ | $\mu(1-\mu)$ | $1/n$ |
| ポアソン | $\ln\mu$ | $e^\theta$ | $\mu$ | $1$ |
| ガンマ | $-1/\mu$ | $-\ln(-\theta)$ | $\mu^2$ | $\phi$ |
| 逆ガウス | $-1/(2\mu^2)$ | $-\sqrt{-2\theta}$ | $\mu^3$ | $\phi$ |

### 2.3 正準リンク関数

$g(\mu) = \theta$（自然パラメータと線形予測子を直結）

| 分布 | 正準リンク | $g(\mu)$ |
|---|---|---|
| 正規 | 恒等 | $\mu$ |
| 二項 | ロジット | $\ln(\mu/(1-\mu))$ |
| ポアソン | 対数 | $\ln\mu$ |
| ガンマ | 逆数 | $1/\mu$ |
| 逆ガウス | $1/\mu^2$ | $1/\mu^2$ |

正準リンクの利点: 十分統計量が $X^Ty$ となり、推定が安定

### 2.4 パラメータ推定

**スコア方程式**（対数尤度の1階微分 = 0）:

$$\frac{\partial \ell}{\partial \beta} = X^T W \Delta (y - \mu) = 0$$

ここで $W = \text{diag}(w_i)$, $w_i = (g'(\mu_i))^{-2} / V(\mu_i)$

**IRLS（反復重み付き最小二乗法）**:

1. 初期値 $\hat{\mu}^{(0)}$ を設定
2. 作業応答変数: $z_i = \eta_i + (y_i - \mu_i) g'(\mu_i)$
3. 重み: $w_i = 1/(V(\mu_i)(g'(\mu_i))^2)$
4. WLS で $\hat{\beta}^{(k+1)} = (X^TWX)^{-1}X^TWz$ を更新
5. 収束まで繰り返し

フィッシャー情報行列: $\mathcal{I}(\beta) = X^T W X$

### 2.5 デビアンスと残差

**デビアンス**（逸脱度）:

$$D = 2\phi \sum_{i=1}^n \{y_i(\tilde{\theta}_i - \hat{\theta}_i) - b(\tilde{\theta}_i) + b(\hat{\theta}_i)\}$$

$\tilde{\theta}$: 飽和モデルのパラメータ、$\hat{\theta}$: 当該モデルのパラメータ

**残差の種類**:
- **ピアソン残差**: $r_i^P = (y_i - \hat{\mu}_i) / \sqrt{V(\hat{\mu}_i)}$
- **逸脱度残差**: $r_i^D = \text{sign}(y_i - \hat{\mu}_i)\sqrt{d_i}$ — $d_i$ は各観測のデビアンス寄与分
- **アンスコム残差**: 分布固有の分散安定化変換を適用した残差

### 2.6 過分散の検出と対処

ポアソン・二項モデルで $\text{Var}(Y) > E[V(\mu)]$ となる現象。

**検出**: ピアソン $\chi^2$ 統計量 / 自由度 $\gg 1$

**対処法**:
- **準尤度法**: $\phi$ を推定して標準誤差を膨らませる（$\hat{\phi} = X^2/(n-p)$）
- **負の二項分布**: ポアソンの代わりに使用（§4参照）
- **ベータ二項分布**: 二項の過分散に対応

### 2.7 GLM一覧表

| 応答 | 分布 | リンク | モデル名 | 典型的用途 |
|---|---|---|---|---|
| 連続 | 正規 | 恒等 | 線形回帰 | 売上予測 |
| 二値 | 二項 | ロジット | ロジスティック回帰 | 信用スコアリング |
| 二値 | 二項 | プロビット | プロビット回帰 | 生物検定（用量反応） |
| 二値 | 二項 | 補対数対数 | 補対数対数回帰 | 稀な事象の二値応答 |
| カウント | ポアソン | 対数 | ポアソン回帰 | 事故件数 |
| カウント | 負の二項 | 対数 | 負の二項回帰 | 過分散カウント |
| 正値連続 | ガンマ | 逆数/対数 | ガンマ回帰 | 保険金額 |
| 正値連続 | 逆ガウス | $1/\mu^2$ | 逆ガウス回帰 | 待ち時間 |
| 半連続 | Tweedie | 対数 | Tweedie回帰 | 0膨張正値（保険金請求） |

### 実務活用
- **GLM選択の基本方針**: 応答変数の型（連続/二値/カウント/正値）でまず分布を決定 → リンク関数はデフォルト（正準リンク）から始める
- **IRLS不収束対策**: 初期値を変える、ステップハーフィングを導入、正準リンクを使う
- **デビアンス分析**: ANOVA的にモデル比較（ネストモデル間でデビアンス差が $\chi^2$ に従う）

## 3. ロジスティック回帰（詳細）

### 3.1 モデル

$$\ln\frac{P(Y=1|x)}{P(Y=0|x)} = \beta_0 + \beta_1 x_1 + \cdots + \beta_p x_p$$

$$P(Y=1|x) = \frac{1}{1 + e^{-x^T\beta}} = \frac{e^{x^T\beta}}{1 + e^{x^T\beta}}$$

### 3.2 オッズ比の解釈

$x_j$ が1単位増加したとき、オッズは $e^{\beta_j}$ 倍に変化。

- $\beta_j > 0$: $e^{\beta_j} > 1$ → オッズ増加（発生確率上昇）
- $\beta_j = 0$: $e^{\beta_j} = 1$ → オッズ不変（影響なし）
- $\beta_j < 0$: $e^{\beta_j} < 1$ → オッズ減少

**注意**: オッズ比は確率比ではない。稀な事象（$P \ll 1$）ではオッズ比 ≈ リスク比（相対危険度）。

### 3.3 最尤推定と検定

対数尤度: $\ell(\beta) = \sum \{y_i x_i^T\beta - \ln(1 + e^{x_i^T\beta})\}$

閉じた形の解はなく、ニュートン・ラフソン法（= IRLS）で反復求解。

| 検定 | 統計量 | 帰無仮説 | 分布 |
|---|---|---|---|
| ワルド検定 | $z = \hat{\beta}_j / \text{SE}(\hat{\beta}_j)$ | $\beta_j = 0$ | 漸近 $N(0,1)$ |
| 尤度比検定 | $-2(\ell_0 - \ell_1)$ | $\beta_j = 0$（ネスト） | 漸近 $\chi^2_q$ |
| スコア検定 | $U^T \mathcal{I}^{-1} U$ | $\beta_j = 0$ | 漸近 $\chi^2_q$ |

尤度比検定が最も信頼性が高い（ワルド検定は大標本でバイアスあり）。

### 3.4 分類性能指標

| 指標 | 定義 | 用途 |
|---|---|---|
| 感度（再現率） | $TP/(TP+FN)$ | 見落とし回避が重要な場面 |
| 特異度 | $TN/(TN+FP)$ | 偽陽性回避が重要な場面 |
| 適合率 | $TP/(TP+FP)$ | 陽性予測の信頼性 |
| F1スコア | $2 \cdot \text{適合率} \cdot \text{再現率} / (\text{適合率} + \text{再現率})$ | 適合率・再現率のバランス |
| ROC曲線 | 感度 vs (1-特異度) | 閾値非依存の識別能評価 |
| AUC | ROC曲線下面積 | 0.5=ランダム、1.0=完全識別 |
| ブライアスコア | $\frac{1}{n}\sum(y_i - \hat{p}_i)^2$ | 確率予測の較正精度 |

### 3.5 多値ロジスティック回帰

**多項ロジットモデル**（名義応答、$K$ カテゴリ）:

$$\ln\frac{P(Y=k|x)}{P(Y=K|x)} = x^T\beta_k, \quad k=1,\ldots,K-1$$

**順序ロジットモデル**（比例オッズモデル）:

$$\text{logit}(P(Y \leq k|x)) = \alpha_k - x^T\beta$$

比例オッズ仮定: 全てのカットポイント $k$ で $\beta$ が共通（ブラント検定で検証）。

### 3.6 完全分離問題

ある変数の値で応答が完全に分かれる場合、MLE が $\pm\infty$ に発散。

**対処**:
- **フィルスの修正**（Firth's penalized likelihood）: ジェフリーズ事前分布に相当するペナルティ $\ell^* = \ell + \frac{1}{2}\ln|\mathcal{I}|$
- **ベイズロジスティック回帰**: 弱情報事前分布（Cauchy(0,2.5) 等）

### 実務活用
- **信用スコアリング**: 金融機関のデフォルト予測（オッズ比でリスク因子の寄与を説明）
- **医学**: 疾患リスク因子の同定（調整オッズ比として報告）
- **マーケティング**: 購買予測、チャーン予測（AUCでモデル性能評価）
- **較正曲線**: 予測確率と実測頻度の一致を検証（ホスマー・レメショウ検定、較正プロット）

## 4. ポアソン回帰（詳細）

### 4.1 モデル

$$Y_i \sim \text{Pois}(\mu_i), \quad \ln\mu_i = x_i^T\beta$$

対数リンクにより $\mu_i > 0$ を自動的に保証。

### 4.2 オフセット項

観測期間 $t_i$ が異なる場合、**発生率** $\lambda_i = \mu_i / t_i$ をモデル化:

$$\ln\mu_i = \ln t_i + x_i^T\beta$$

$\ln t_i$ が**オフセット項**（係数1で固定される説明変数）。

**発生率比 (IRR)**: $e^{\beta_j}$ — $x_j$ が1単位増加したときの発生率の乗数。

### 4.3 ゼロ過剰モデル

データに0が過剰に含まれる場合（構造的ゼロ + サンプリングゼロ）。

**ゼロ過剰ポアソン (ZIP)**:

$$P(Y=0) = \pi + (1-\pi)e^{-\mu}$$
$$P(Y=k) = (1-\pi)\frac{\mu^k e^{-\mu}}{k!}, \quad k \geq 1$$

- $\pi$: 構造的ゼロの確率（ロジスティックモデルで予測可能）
- 2つのサブモデル（ゼロ生成 + カウント生成）を同時推定

**ハードルモデル**: 0 vs 正のカウントを分離（0を超える部分は切断ポアソン）

### 4.4 過分散対策

| 方法 | 分散構造 | 特徴 |
|---|---|---|
| 準ポアソン | $\text{Var}(Y) = \phi\mu$ | 分散パラメータ $\phi$ を推定。AIC使用不可 |
| 負の二項回帰 | $\text{Var}(Y) = \mu + \mu^2/k$ | ガンマ混合ポアソン。$k \to \infty$ でポアソン |

負の二項回帰の方がモデルベースで尤度が定義されるため、AIC/BICによるモデル比較が可能。

### 実務活用
- **疫学**: 疾患発生件数（人年オフセット付き）
- **保険**: 事故件数モデリング（契約期間オフセット）
- **Webアクセス**: ページ閲覧回数（ゼロ過剰ポアソンが有効）
- **製造業**: 不良品数（過分散の検出 → 負の二項に切り替え）

## 5. 正則化回帰

### 5.1 リッジ回帰（L2正則化）

$$\hat{\beta}_{\text{ridge}} = \arg\min_\beta \left\{\|y - X\beta\|^2 + \lambda\|\beta\|^2_2\right\}$$

閉じた形の解: $\hat{\beta}_{\text{ridge}} = (X^TX + \lambda I)^{-1}X^Ty$

- $\lambda = 0$: OLS に一致
- $\lambda \to \infty$: $\hat{\beta} \to 0$
- 係数は 0 にならない（スパース解を与えない）
- 多重共線性への対処として有効（$(X^TX + \lambda I)$ は常に正則）

### 5.2 LASSO（L1正則化）

$$\hat{\beta}_{\text{lasso}} = \arg\min_\beta \left\{\|y - X\beta\|^2 + \lambda\|\beta\|_1\right\}$$

- 閉じた形の解なし（座標降下法、LARS等で求解）
- **スパース解**: 一部の $\hat{\beta}_j = 0$ となり自動的に変数選択
- $n < p$ のとき最大 $n$ 個の非ゼロ変数しか選べない
- 相関の高い変数群から1つだけ選ぶ傾向（安定性に欠ける場合あり）

### 5.3 Elastic Net（L1 + L2）

$$\hat{\beta}_{\text{EN}} = \arg\min_\beta \left\{\|y - X\beta\|^2 + \lambda_1\|\beta\|_1 + \lambda_2\|\beta\|^2_2\right\}$$

混合パラメータ $\alpha = \lambda_1/(\lambda_1 + \lambda_2)$:
- $\alpha = 1$: LASSO
- $\alpha = 0$: リッジ
- $0 < \alpha < 1$: 両方の性質。相関変数群をグループで選択できる

### 5.4 正則化パラメータの選択

**k分割交差検証**:
1. データを $k$ 個に分割
2. 各 $\lambda$ に対して $k$ 回の学習・検証を繰り返し
3. 平均CVエラーが最小の $\lambda$ を選択
4. 1SE ルール: 最小CVエラー + 1SE 以内で最大の $\lambda$（よりスパースなモデル）

### 5.5 バイアス-バリアンストレードオフ

$$\text{MSE} = \text{Bias}^2 + \text{Variance} + \text{Irreducible Error}$$

- OLS: バイアス = 0、分散大（特に $p/n$ 大のとき）
- 正則化: バイアスを少し許容して分散を大幅に減少 → MSE減少

**縮小推定の幾何的解釈**: リッジ = 等高線が楕円（球制約）、LASSO = 等高線がひし形（角に解が来やすい = スパース）

### 実務活用
- **高次元データ**: 遺伝子発現データ($p \gg n$)でLASSO/Elastic Net が標準
- **特徴選択**: LASSO の非ゼロ係数 = 選択された特徴量
- **予測精度向上**: リッジ回帰は多重共線性存在下で OLS より低MSE
- **boatrace応用**: 多数のレース特徴量（モーター・枠番・勝率・風向等）からBET判断に有効な変数を絞り込み

## 6. 混合効果モデル（マルチレベルモデル）

### 6.1 基本構造

$$y_{ij} = x_{ij}^T\beta + z_{ij}^T u_i + \varepsilon_{ij}$$

- $\beta$: **固定効果**（母集団全体の平均的効果）
- $u_i$: **変動効果**（クラスター $i$ に固有のランダムな偏差）、$u_i \sim N(0, G)$
- $\varepsilon_{ij}$: 個体内誤差、$\varepsilon_{ij} \sim N(0, \sigma^2)$

### 6.2 ランダム切片・ランダム傾きモデル

**ランダム切片モデル**:

$$y_{ij} = (\beta_0 + u_{0i}) + \beta_1 x_{ij} + \varepsilon_{ij}$$

クラスターごとに切片がランダムに変動。

**ランダム傾きモデル**:

$$y_{ij} = (\beta_0 + u_{0i}) + (\beta_1 + u_{1i}) x_{ij} + \varepsilon_{ij}$$

切片と傾きの両方がクラスターごとに変動。$\text{Cov}(u_{0i}, u_{1i}) = \sigma_{01}$ も推定。

### 6.3 推定法

| 方法 | 特徴 | 用途 |
|---|---|---|
| ML（最尤法） | 分散成分を過小推定する傾向 | モデル比較（固定効果が異なるモデル間） |
| REML（制限付き最尤法） | 分散成分の不偏推定 | デフォルトの推定法。固定効果同一のモデル間比較 |

ML vs REML の違い: REML は固定効果をプロファイルアウトしてから分散を推定（自由度の損失を考慮）。

### 6.4 級内相関係数 (ICC)

$$\text{ICC} = \frac{\sigma^2_{u0}}{\sigma^2_{u0} + \sigma^2_\varepsilon}$$

- クラスター内の相関（応答の総変動に占めるクラスター間変動の割合）
- ICC ≈ 0: クラスター構造なし（通常の回帰で十分）
- ICC が高い: 混合効果モデルが必要

### 6.5 検定

- **固定効果の検定**: ワルド検定、t検定（自由度の近似: サタスウェイト法、ケンワード・ロジャー法）
- **変動効果の検定**: 尤度比検定（帰無仮説が分散の境界 $\sigma^2 = 0$ のため、$\chi^2$ 分布の混合 $0.5\chi^2_0 + 0.5\chi^2_1$ を使用）

### 実務活用
- **反復測定データ**: 同一被験者の経時的測定（個体差をランダム効果で吸収）
- **クラスター構造**: 学校内の生徒、病院内の患者、地域内の店舗
- **パネルデータ**: 経済学の個体追跡データ（固定効果モデルとの選択 → ハウスマン検定）
- **boatrace応用**: 会場効果をランダム効果として扱い、会場間のROI差異を構造的にモデリング

## 7. 一般化加法モデル (GAM)

### 7.1 モデル

$$g(\mu) = \beta_0 + f_1(x_1) + f_2(x_2) + \cdots + f_p(x_p)$$

$f_j$ は未知の平滑関数。GLMのリンク関数を維持しつつ非線形関係を柔軟にモデル化。

### 7.2 スプライン回帰

**基底関数展開**: $f(x) = \sum_{k=1}^K \beta_k B_k(x)$

| スプライン種別 | 特徴 |
|---|---|
| Bスプライン | 局所的サポート、計算効率が良い |
| 自然スプライン | 境界で線形に制約（外挿安定） |
| 平滑化スプライン | 全データ点にノットを配置+粗さペナルティ |
| 薄板スプライン | 多変量に自然に拡張可能 |
| P-スプライン | Bスプライン基底+差分ペナルティ |

**ペナルティ付き回帰スプライン**:

$$\min_\beta \left\{\|y - X\beta\|^2 + \lambda \int (f''(x))^2 dx\right\}$$

$\lambda$: 平滑化パラメータ（$\lambda = 0$ で補間、$\lambda \to \infty$ で線形回帰）

### 7.3 平滑化パラメータの選択

| 方法 | 基準 |
|---|---|
| GCV（一般化交差検証） | $\text{GCV} = \frac{n \cdot \text{RSS}}{(n - \text{edf})^2}$ |
| REML | 対数周辺尤度の最大化（推奨、過適合に強い） |
| AIC | $-2\ell + 2 \cdot \text{edf}$ |

**有効自由度 (edf)**: 平滑関数の複雑さの指標。edf = 1 で線形、大きいほど非線形。

### 7.4 テンソル積スプライン

2変数以上の交互作用: $f(x_1, x_2)$ をテンソル積基底で表現。

$$f(x_1, x_2) = \sum_j \sum_k \beta_{jk} B_j^{(1)}(x_1) B_k^{(2)}(x_2)$$

等方的な薄板スプライン `s(x1, x2)` vs 異方的なテンソル積 `te(x1, x2)` の選択は変数のスケールに依存。

### 実務活用
- **非線形パターンの探索**: 線形回帰の残差に曲線パターン → GAMで対処
- **生態学・環境科学**: 種分布モデリング（気温・降水量の非線形応答）
- **疫学**: 大気汚染と死亡率の用量反応関係
- **EDA**: スプライン項の edf で非線形性の程度を数値評価（edf ≈ 1 なら線形で十分）

## 8. 生存時間解析

### 8.1 打ち切りデータ

| 打ち切り種別 | 定義 | 例 |
|---|---|---|
| 右打ち切り | イベント未発生で観察終了 | 試験終了時に生存 |
| 左打ち切り | イベントが観察開始前に発生 | 検査時に既に陽性 |
| 区間打ち切り | イベント発生時刻が区間でしか分からない | 定期検診の間に発生 |

### 8.2 基本関数

**生存関数**: $S(t) = P(T > t) = 1 - F(t)$

**ハザード関数**: $h(t) = \lim_{\Delta t \to 0} \frac{P(t \leq T < t+\Delta t | T \geq t)}{\Delta t} = \frac{f(t)}{S(t)}$

**累積ハザード**: $H(t) = \int_0^t h(u)du = -\ln S(t)$

相互関係: $S(t) = \exp(-H(t))$, $f(t) = h(t)S(t)$

### 8.3 カプラン・マイヤー推定量

$$\hat{S}(t) = \prod_{t_i \leq t} \left(1 - \frac{d_i}{n_i}\right)$$

- $t_i$: イベント発生時刻（順序統計量）
- $d_i$: 時刻 $t_i$ でのイベント数
- $n_i$: 時刻 $t_i$ 直前のリスク集合サイズ

**グリーンウッドの公式**（分散推定）:

$$\widehat{\text{Var}}(\hat{S}(t)) = \hat{S}(t)^2 \sum_{t_i \leq t} \frac{d_i}{n_i(n_i - d_i)}$$

### 8.4 ログランク検定

2群以上の生存曲線の比較。

$$\chi^2 = \frac{(\sum_i O_{1i} - E_{1i})^2}{\sum_i V_i}$$

帰無仮説: $S_1(t) = S_2(t)$ for all $t$。漸近的に $\chi^2_1$ 分布。

バリエーション: ウィルコクソン（ゲハン）検定（早期の差異に重み）、フレミング・ハリントン検定（重み関数 $S(t)^p(1-S(t))^q$）。

### 8.5 Cox比例ハザードモデル

$$h(t|x) = h_0(t) \exp(x^T\beta)$$

- $h_0(t)$: 基準ハザード関数（非パラメトリック、推定不要）
- $\exp(\beta_j)$: **ハザード比** — $x_j$ が1単位増加でハザードが $e^{\beta_j}$ 倍

**部分尤度** (partial likelihood):

$$L(\beta) = \prod_{i: \delta_i=1} \frac{\exp(x_i^T\beta)}{\sum_{j \in R(t_i)} \exp(x_j^T\beta)}$$

$R(t_i)$: 時刻 $t_i$ のリスク集合、$\delta_i$: イベント指示変数。

**比例ハザード仮定の検証**:
- ショーエンフェルド残差の時間に対するプロット（傾向があれば仮定違反）
- ログ-ログ生存曲線プロット（$\ln(-\ln\hat{S}(t))$ vs $\ln t$ が平行なら仮定成立）
- 時間依存共変量の追加（$x \cdot g(t)$ 項が有意なら仮定違反）

### 8.6 パラメトリックモデル

| 分布 | ハザード $h(t)$ | 特徴 |
|---|---|---|
| 指数 | $\lambda$（一定） | 最も単純。無記憶性 |
| ワイブル | $\alpha\lambda(\lambda t)^{\alpha-1}$ | $\alpha>1$で増加、$\alpha<1$で減少ハザード |
| 対数正規 | 非単調 | 山型ハザード（初期増加→後期減少） |
| 対数ロジスティック | 非単調 | 同上。AFTモデルとして推定容易 |
| ゴンペルツ | $\lambda e^{\alpha t}$ | 指数的に増加するハザード（老化） |

### 8.7 加速故障時間モデル (AFT)

$$\ln T = x^T\beta + \sigma W$$

$W$ の分布で決まるモデル: 標準正規 → 対数正規AFT、標準ロジスティック → 対数ロジスティックAFT、最小値極値 → ワイブルAFT。

**解釈**: $\exp(\beta_j)$ は「時間加速因子」— 生存時間を何倍に伸縮するか。

### 8.8 競合リスクモデル

複数のイベントタイプ（死因1, 死因2, ...）が競合する状況。

**原因別ハザード**:

$$h_k(t) = \lim_{\Delta t \to 0} \frac{P(t \leq T < t+\Delta t, \text{cause}=k | T \geq t)}{\Delta t}$$

**累積発生関数 (CIF)**: $F_k(t) = \int_0^t h_k(u) S(u) du$

Fine-Gray モデル: CIF を直接モデル化（副分布ハザード）。

### 実務活用
- **医学**: 臨床試験の主要エンドポイント（OS, PFS）
- **信頼性工学**: 製品寿命分析（ワイブルモデルが標準）
- **チャーン分析**: 顧客離脱までの時間（Cox回帰でリスク因子特定）
- **boatrace応用**: モデル別の「連敗からの回復時間」を生存分析的に評価

## 9. ロバスト回帰

### 9.1 M推定

$$\hat{\beta} = \arg\min_\beta \sum_{i=1}^n \rho\left(\frac{y_i - x_i^T\beta}{\hat{\sigma}}\right)$$

| $\rho$ 関数 | 特徴 | 崩壊点 |
|---|---|---|
| 二乗（OLS） | $\rho(r) = r^2/2$ | 0%（1つの外れ値で崩壊） |
| Huber | $\rho(r) = r^2/2$ ($|r| \leq c$), $c|r|-c^2/2$ ($|r|>c$) | 0%（だが耐性は向上） |
| Tukey's biweight | $\rho(r) = (c^2/6)[1-(1-(r/c)^2)^3]$ ($|r| \leq c$) | ~50%（高崩壊点） |

$c$: チューニング定数（Huber: $c=1.345$ で効率95%、Tukey: $c=4.685$）

### 9.2 高崩壊点推定

**LTS（最小トリム二乗法）**: 残差の二乗の小さい方から $h$ 個の和を最小化

$$\hat{\beta}_{\text{LTS}} = \arg\min_\beta \sum_{i=1}^h r^2_{(i)}$$

$h \approx \lfloor n/2 \rfloor + \lfloor (p+1)/2 \rfloor$ で崩壊点約50%。

**LMS（最小メディアン二乗法）**: $\text{median}(r_i^2)$ を最小化。収束速度が $n^{-1/3}$（遅い）。

### 9.3 分位点回帰

$$\hat{\beta}_\tau = \arg\min_\beta \sum_{i=1}^n \rho_\tau(y_i - x_i^T\beta)$$

$\rho_\tau(u) = u(\tau - I(u < 0))$: チェック関数

- $\tau = 0.5$: 中央値回帰（LAD: Least Absolute Deviations）
- $\tau = 0.1, 0.9$: 分布の裾をモデル化
- 外れ値に頑健（L1損失の一般化）
- 線形計画法で効率的に求解

### 実務活用
- **金融**: 外れ値の多い収益データ（M推定で頑健な回帰）
- **不動産**: 価格の条件付き分布全体を分位点回帰で把握
- **品質管理**: 工程データに外れ値が混入する場合の傾向分析
- **賃金分析**: 所得分布の上位・下位で異なる影響を分位点回帰で検出

## 10. モデル診断・選択

### 10.1 残差プロット

| プロット | 目的 | 判読 |
|---|---|---|
| 残差 vs 予測値 | 等分散性・線形性 | ファンネル形 → 不等分散、曲線 → 非線形 |
| QQプロット | 正規性 | 直線から外れる → 非正規 |
| Scale-Location | 等分散性 | $\sqrt{|r_i|}$ vs $\hat{y}$ が水平なら等分散 |
| 残差 vs 説明変数 | 各変数との関係 | パターンがあれば交互作用・非線形を検討 |
| 偏残差プロット | 各変数の偏回帰関係 | 成分+残差プロット（component + residual） |

### 10.2 情報量規準

| 規準 | 定義 | 特性 |
|---|---|---|
| AIC | $-2\ln L + 2p$ | 予測損失の不偏推定（漸近）。KL情報量最小化 |
| AICc | $\text{AIC} + 2p(p+1)/(n-p-1)$ | 小標本補正。$n/p < 40$ で推奨 |
| BIC | $-2\ln L + p\ln n$ | 一致性あり。$n > e^2 \approx 7.4$ で AIC より厳しい |
| DIC | $-2\ln L(\bar{\theta}) + 2p_D$ | ベイズモデル用（$p_D$ は有効パラメータ数） |
| WAIC | LOO-CVの漸近近似 | ベイズモデル用。DICより安定 |

**AIC vs BIC の使い分け**:
- 予測精度重視 → AIC（過適合をやや許容）
- 真のモデル同定重視 → BIC（変数数にペナルティ大）
- 混合効果モデル → 条件付きAIC (cAIC)

### 10.3 交差検証

| 方法 | 計算量 | 特徴 |
|---|---|---|
| k-fold CV | $O(k)$ 回の学習 | $k=5$ or $10$ が一般的。バイアスとバリアンスのバランス |
| LOO-CV | $O(n)$ 回の学習 | 不偏だが分散大。線形モデルでは解析解あり: $\text{CV} = \frac{1}{n}\sum(e_i/(1-h_{ii}))^2$ |
| Repeated CV | $O(rk)$ 回の学習 | $r$ 回繰り返して分散を低減 |
| LOOCV-AIC等価性 | — | 線形モデルで AIC ∝ LOO-CV（漸近的に等価） |

### 10.4 尤度比検定

ネストモデル $M_0 \subset M_1$ の比較:

$$\Lambda = -2(\ell_0 - \ell_1) \sim \chi^2_{p_1 - p_0}$$

- $p_1 - p_0$: 制約の数
- 非ネストモデルには使えない → AIC/BIC やヴォン検定を使用
- GLMでは**デビアンス分析**: $D_0 - D_1 \sim \chi^2_{p_1 - p_0}$

### 10.5 ベイズモデル比較

**ベイズ因子**: $\text{BF}_{10} = P(\text{data}|M_1) / P(\text{data}|M_0)$

| $\ln(\text{BF}_{10})$ | 証拠の強さ |
|---|---|
| 0 〜 1 | わずか |
| 1 〜 3 | 肯定的 |
| 3 〜 5 | 強い |
| > 5 | 非常に強い |

**周辺尤度の計算**: ラプラス近似、ブリッジサンプリング、調和平均推定量（不安定で非推奨）。

### 実務活用
- **モデル構築ワークフロー**: (1) 残差プロットで仮定違反を確認 → (2) AIC/BICで候補モデル比較 → (3) CVで汎化性能を評価
- **過適合検出**: 訓練誤差とCV誤差の乖離が大きい → 正則化やモデル簡素化を検討
- **報告**: AIC/BICの絶対値は無意味（差分 $\Delta$AIC のみ意味がある。$\Delta > 10$ で明確な差）
- **GLMでの注意**: 準尤度モデル（準ポアソン等）ではAIC定義不可 → QIC（準情報量規準）を使用

## 付録A: 主要な分布と回帰モデルの対応

```
応答変数の型を確認
├─ 連続（実数全体）→ 線形回帰 / ロバスト回帰
├─ 正値連続 → ガンマ回帰 / 対数正規回帰 / Tweedie

│   ├─ 過分散 → 負の二項 / 準ポアソン
│   └─ ゼロ過剰 → ZIP / ハードル
├─ 多値（名義）→ 多項ロジット
├─ 多値（順序）→ 順序ロジット / 順序プロビット
├─ 時間（打ち切りあり）→ Cox / パラメトリック生存 / AFT
└─ 比率（0-1）→ ベータ回帰
```
