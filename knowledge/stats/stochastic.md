## Part A: 確率過程

### 1. 確率過程の基礎

#### 1.1 定義

確率空間 (Ω, F, P) 上で定義された確率変数の族 {X(t), t ∈ T} を**確率過程**と呼ぶ。
- T: 添字集合（時間集合）
- S: 状態空間（X(t) の値域）
- 各 ω ∈ Ω に対し t → X(t, ω) を**標本路**（sample path）と呼ぶ

#### 1.2 分類

| | 離散状態 | 連続状態 |
|---|---|---|
| **離散時間** (T = Z⁺) | マルコフ連鎖、ランダムウォーク | AR過程、GARCH |
| **連続時間** (T = R⁺) | ポアソン過程、出生死滅過程 | ブラウン運動、拡散過程 |

#### 1.3 有限次元分布族

確率過程の分布は、任意の n ≥ 1 と t₁ < t₂ < ... < tₙ に対する同時分布
F_{t₁,...,tₙ}(x₁,...,xₙ) = P(X(t₁) ≤ x₁, ..., X(tₙ) ≤ xₙ) の族で特徴づけられる。

**コルモゴロフの拡張定理**: 整合性条件（対称性・射影整合性）を満たす有限次元分布族に対し、
それを有限次元分布として持つ確率過程が存在する。

#### 1.4 定常性

**強定常（狭義定常）**: 任意の h に対し (X(t₁+h),...,X(tₙ+h)) と (X(t₁),...,X(tₙ)) が同分布。

**弱定常（広義定常・共分散定常）**:
- E[X(t)] = μ （一定）
- Cov(X(t), X(t+h)) = C(h) （時差 h のみに依存）

弱定常 ⊅ 強定常（反例: 正規でない場合）。正規過程では弱定常 ⟺ 強定常。

#### 1.5 実務活用
- 時系列データの定常性検定（ADF検定、KPSS検定）を適用する前に、プロセスの分類を明確にする
- 非定常過程は差分・対数変換で定常化してからモデリングする

### 2. マルコフ連鎖（離散時間）

#### 2.1 マルコフ性

P(Xₙ₊₁ = j | X₀ = i₀, ..., Xₙ = i) = P(Xₙ₊₁ = j | Xₙ = i)

未来の状態は現在の状態のみに依存し、過去の履歴に依存しない。

**時間斉次（homogeneous）**: P(Xₙ₊₁ = j | Xₙ = i) = pᵢⱼ が n に依存しない。

#### 2.2 推移確率行列

P = (pᵢⱼ) は確率行列（各行の和が1、全要素非負）。

n ステップ推移確率: pᵢⱼ⁽ⁿ⁾ = P(Xₙ = j | X₀ = i) → 行列表現で P⁽ⁿ⁾ = Pⁿ

#### 2.3 チャップマン・コルモゴロフ方程式

pᵢⱼ⁽ⁿ⁺ᵐ⁾ = Σₖ pᵢₖ⁽ⁿ⁾ pₖⱼ⁽ᵐ⁾

行列形式: P⁽ⁿ⁺ᵐ⁾ = P⁽ⁿ⁾ P⁽ᵐ⁾

#### 2.4 状態の分類

**到達可能**: i → j ⟺ ∃n ≥ 0 s.t. pᵢⱼ⁽ⁿ⁾ > 0
**連絡**: i ↔ j ⟺ i → j かつ j → i（同値関係 → 連絡クラスに分割）
**既約（irreducible）**: 全状態が1つの連絡クラス

**再帰時間**: Tᵢ = min{n ≥ 1 : Xₙ = i | X₀ = i}
- **再帰的（recurrent）**: P(Tᵢ < ∞) = 1 ⟺ Σₙ pᵢᵢ⁽ⁿ⁾ = ∞
- **過渡的（transient）**: P(Tᵢ < ∞) < 1 ⟺ Σₙ pᵢᵢ⁽ⁿ⁾ < ∞
- **正再帰（positive recurrent）**: E[Tᵢ] < ∞
- **零再帰（null recurrent）**: P(Tᵢ < ∞) = 1 だが E[Tᵢ] = ∞

**周期**: d(i) = gcd{n ≥ 1 : pᵢᵢ⁽ⁿ⁾ > 0}
- d(i) = 1 のとき**非周期的（aperiodic）**

**吸収状態**: pᵢᵢ = 1 の状態

#### 2.5 エルゴード定理

既約・非周期・正再帰なマルコフ連鎖では:
- 一意の定常分布 π が存在
- 任意の初期分布に対し pᵢⱼ⁽ⁿ⁾ → πⱼ (n → ∞)
- 時間平均 = 空間平均: (1/n) Σₖ₌₁ⁿ f(Xₖ) → Σⱼ f(j)πⱼ (a.s.)

有限状態の既約マルコフ連鎖は必ず正再帰。

#### 2.6 定常分布の計算

πP = π かつ Σᵢ πᵢ = 1 を連立方程式として解く。

**計算手法**: 1つの方程式を Σπᵢ = 1 に置き換えて解く（線形従属のため）。

#### 2.7 詳細つり合い条件

πᵢ pᵢⱼ = πⱼ pⱼᵢ （全 i, j に対して）

この条件を満たすマルコフ連鎖を**可逆（reversible）**と呼ぶ。
詳細つり合いを満たせば π は定常分布。逆は必ずしも成立しない。

#### 2.8 吸収鎖の基本行列

状態を過渡状態 T と吸収状態 A に分割:
P = | Q  R |
    | 0  I |

**基本行列**: N = (I - Q)⁻¹
- Nᵢⱼ = 過渡状態 i から出発して吸収前に状態 j を訪れる期待回数
- 吸収までの期待ステップ数: t = N1（各行の和）
- 吸収確率行列: B = NR

#### 2.9 実務活用
- **PageRank**: Webグラフ上のランダムウォーク。ダンピング因子 d ≈ 0.85 で非周期・既約化
- **MCMC**: Metropolis-Hastings、Gibbs サンプリング。定常分布が目標分布となるよう推移確率を設計
- **顧客行動モデル**: 購買状態遷移、解約予測（吸収マルコフ連鎖）
- **在庫管理**: (s, S) 政策のもとでの在庫水準推移

### 3. マルコフ連鎖（連続時間）

#### 3.1 生成行列（推移速度行列）

Q = (qᵢⱼ) ただし:
- qᵢⱼ ≥ 0 (i ≠ j): 状態 i から j への推移速度
- qᵢᵢ = -Σⱼ≠ᵢ qᵢⱼ（各行の和が0）
- qᵢ = -qᵢᵢ: 状態 i からの脱出速度

状態 i の滞在時間は指数分布 Exp(qᵢ) に従う。

#### 3.2 コルモゴロフ方程式

**前進方程式**: P'(t) = P(t)Q
**後退方程式**: P'(t) = QP(t)

解: P(t) = e^{Qt} = Σₙ₌₀^∞ (Qt)ⁿ/n!

#### 3.3 定常分布

πQ = 0 かつ Σᵢ πᵢ = 1

#### 3.4 出生死滅過程

状態空間 {0, 1, 2, ...} 上で隣接状態のみに遷移:
- 出生率: λₙ（状態 n → n+1）
- 死滅率: μₙ（状態 n → n-1）

定常分布: πₙ = π₀ × Πₖ₌₁ⁿ (λₖ₋₁/μₖ)

#### 3.5 実務活用
- **M/M/1 待ち行列**: λₙ = λ, μₙ = μ の出生死滅過程
- **電話交換**: アーランの損失/遅延モデル
- **人口動態モデル**: 確率的増殖過程

### 4. ポアソン過程

#### 4.1 定義（3つの等価な定義）

**定義A（カウント過程）**: {N(t), t ≥ 0} が強度 λ のポアソン過程 ⟺
1. N(0) = 0
2. 独立増分
3. N(t+s) - N(s) ~ Poi(λt)

**定義B（到着間隔）**: 到着間隔 T₁, T₂, ... が i.i.d. Exp(λ)

**定義C（微小条件）**:
1. N(0) = 0、独立増分、定常増分
2. P(N(h) = 1) = λh + o(h)
3. P(N(h) ≥ 2) = o(h)

#### 4.2 主要性質

- E[N(t)] = λt, Var(N(t)) = λt
- 到着時刻の条件付き分布: N(t) = n が与えられたとき、n 個の到着時刻は [0,t] 上の一様分布の順序統計量と同分布

#### 4.3 合流・分割・間引き

**合流**: 独立なポアソン過程 N₁(t) ~ Poi(λ₁t), N₂(t) ~ Poi(λ₂t) の和は Poi((λ₁+λ₂)t)

**分割（間引き）**: 各到着を確率 p で Type I、1-p で Type II に独立分類 →
Type I は Poi(λp)、Type II は Poi(λ(1-p))、かつ互いに独立。

#### 4.4 非斉次ポアソン過程

強度関数 λ(t) が時間により変化:
- N(t) ~ Poi(Λ(t)), ただし Λ(t) = ∫₀ᵗ λ(s) ds
- 到着間隔は非指数分布（一般には非定常）

**シミュレーション**: 間引き法（thinning method）で効率的に生成可能。

#### 4.5 複合ポアソン過程

S(t) = Σᵢ₌₁^{N(t)} Yᵢ （N(t) はポアソン過程、Yᵢ は i.i.d.）

- E[S(t)] = λt E[Y]
- Var(S(t)) = λt E[Y²]
- MGF: M_{S(t)}(u) = exp(λt(M_Y(u) - 1))

#### 4.6 実務活用
- **コールセンター**: 着信数モデリング（非斉次ポアソンでピーク時間帯を表現）
- **保険数理**: 保険金請求（複合ポアソン → クレーマー・ルンドベルクモデル）
- **Webトラフィック**: ページビュー、クリック数のモデル化
- **信頼性工学**: 故障発生モデル

### 5. ブラウン運動（ウィーナー過程）

#### 5.1 定義

{B(t), t ≥ 0} が標準ブラウン運動 ⟺
1. B(0) = 0
2. 独立増分
3. B(t) - B(s) ~ N(0, t-s) （0 ≤ s < t）
4. 標本路は連続（a.s.）

#### 5.2 主要性質

- E[B(t)] = 0, Var(B(t)) = t, Cov(B(s), B(t)) = min(s, t)
- **自己相似性**: {B(ct)} と {√c B(t)} は同分布
- **どこでも微分不可能**（a.s.）
- **二次変分**: [B, B]_t = t（dt 刻みで Σ(ΔBᵢ)² → t）
- **マルコフ性・マルチンゲール性を同時に持つ**

#### 5.3 幾何ブラウン運動

dS = μS dt + σS dB （伊藤の確率微分方程式）

解: S(t) = S(0) exp((μ - σ²/2)t + σB(t))

対数正規分布に従う: ln S(t) ~ N(ln S(0) + (μ - σ²/2)t, σ²t)

#### 5.4 反射原理

P(max_{0≤s≤t} B(s) ≥ a) = 2P(B(t) ≥ a) = 2(1 - Φ(a/√t)) （a > 0）

ブラウン運動の最大値の分布を求めるのに使用。

#### 5.5 初到達時刻

Tₐ = inf{t ≥ 0 : B(t) = a} の密度関数:
f_{Tₐ}(t) = |a| / √(2πt³) × exp(-a²/(2t)) （逆ガウス分布）

#### 5.6 実務活用
- **ブラック・ショールズモデル**: 株価を幾何ブラウン運動でモデル化 → オプション価格の解析解
- **リスク管理**: VaR の計算基盤
- **物理**: 拡散現象、熱伝導方程式の確率的解釈
- **A/Bテスト**: 逐次検定におけるブラウン運動近似

### 6. マルチンゲール

#### 6.1 定義

フィルトレーション {Fₙ} に適合する確率過程 {Xₙ} がマルチンゲール ⟺
1. Xₙ は Fₙ-可測
2. E[|Xₙ|] < ∞
3. E[Xₙ₊₁ | Fₙ] = Xₙ （a.s.）

**上マルチンゲール（supermartingale）**: E[Xₙ₊₁ | Fₙ] ≤ Xₙ
**下マルチンゲール（submartingale）**: E[Xₙ₊₁ | Fₙ] ≥ Xₙ

#### 6.2 典型例

- 公正な賭けの累積利得
- B(t)（標準ブラウン運動）
- B(t)² - t
- exp(θB(t) - θ²t/2)（指数マルチンゲール）
- 尤度比 Lₙ = dPₙ/dQₙ

#### 6.3 停止時刻と任意停止定理

**停止時刻**: τ が停止時刻 ⟺ {τ ≤ n} ∈ Fₙ （各 n に対して）

**任意停止定理（Optional Sampling Theorem）**:
{Xₙ} がマルチンゲール、τ が有界な停止時刻 ⟹ E[X_τ] = E[X₀]

一般の停止時刻では追加条件が必要:
- τ < ∞ a.s. かつ E[|X_τ|] < ∞ かつ E[Xₙ 1_{τ>n}] → 0

**応用例**: ランダムウォークの破産問題。初期資金 a、目標 b、成功確率 p の場合:
- p ≠ 1/2: 破産確率 = ((q/p)^a - (q/p)^b) / (1 - (q/p)^b)
- p = 1/2: 破産確率 = (b-a)/b

#### 6.4 マルチンゲール収束定理

**ドゥーブの収束定理**: {Xₙ} が下マルチンゲールで supₙ E[Xₙ⁺] < ∞ ⟹
X_∞ = lim Xₙ が a.s. に存在し、E[|X_∞|] < ∞

**L²マルチンゲール収束**: E[Xₙ²] ≤ C < ∞ なら Xₙ → X_∞ (a.s. かつ L²)

#### 6.5 ドゥーブの不等式

**最大値不等式**: {Xₙ} が非負の下マルチンゲールのとき:
P(max_{1≤k≤n} Xₖ ≥ λ) ≤ E[Xₙ] / λ

**Lᵖ 最大値不等式** (p > 1):
E[max_{1≤k≤n} |Xₖ|ᵖ] ≤ (p/(p-1))ᵖ E[|Xₙ|ᵖ]

#### 6.6 ドゥーブ分解

適合可積分過程 {Xₙ} = {Mₙ} + {Aₙ} と一意に分解:
- Mₙ: マルチンゲール (M₀ = X₀)
- Aₙ: 予測可能過程 (A₀ = 0)、Aₙ₊₁ - Aₙ は Fₙ-可測

#### 6.7 実務活用
- **オプション価格理論**: リスク中立測度下での割引株価はマルチンゲール
- **逐次解析**: SPRT（逐次確率比検定）の性能解析
- **ベッティング戦略**: マルチンゲール戦略が「公正なゲームでは勝てない」ことの数学的基盤
- **カルマンフィルタ**: イノベーション列はマルチンゲール

### 7. 更新過程

#### 7.1 定義

i.i.d. 非負確率変数 X₁, X₂, ...（到着間隔）に対し:
- Sₙ = Σᵢ₌₁ⁿ Xᵢ（n 回目の更新時刻）
- N(t) = max{n : Sₙ ≤ t}（時刻 t までの更新回数）

#### 7.2 更新関数と更新方程式

**更新関数**: m(t) = E[N(t)]

**更新方程式**: m(t) = F(t) + ∫₀ᵗ m(t-s) dF(s)

ここで F は到着間隔の分布関数。

#### 7.3 基本更新定理

E[X₁] = μ > 0 のとき:
- m(t)/t → 1/μ (t → ∞)
- (大数の法則) N(t)/t → 1/μ (a.s.)
- (中心極限定理) (N(t) - t/μ) / (σ√(t/μ³)) → N(0,1)

#### 7.4 残余寿命・超過寿命・経過時間

時刻 t において:
- **残余寿命（excess life）**: γ(t) = S_{N(t)+1} - t
- **経過時間（age）**: δ(t) = t - S_{N(t)}
- **全寿命（spread）**: β(t) = γ(t) + δ(t) = X_{N(t)+1}

**検査パラドックス**: 定常状態でランダムに観測すると、観測された間隔の期待値は
E[X²]/(2E[X]) > E[X]（ジェンセンの不等式より、分散 > 0 のとき厳密に大きい）

#### 7.5 実務活用
- **機器の交換戦略**: 予防保全の最適周期（age replacement / block replacement）
- **在庫補充**: 定期発注の最適サイクル
- **ソフトウェア信頼性**: バグ発見過程のモデリング

### 8. 待ち行列理論

#### 8.1 ケンドールの記法

A/B/c/K/N/D
- A: 到着過程（M=ポアソン, D=決定論的, G=一般分布, Eₖ=アーラン）
- B: サービス時間分布
- c: サーバー数
- K: システム容量（省略時 ∞）
- N: 客源数（省略時 ∞）
- D: 待ち行列規律（省略時 FCFS）

#### 8.2 M/M/1

到着 Poi(λ)、サービス Exp(μ)、サーバー1台。ρ = λ/μ < 1 で安定。

| 指標 | 公式 |
|---|---|
| システム内客数 L | ρ/(1-ρ) |
| 待ち行列内客数 Lq | ρ²/(1-ρ) |
| システム滞在時間 W | 1/(μ-λ) |
| 待ち時間 Wq | ρ/(μ-λ) |
| 空き確率 P₀ | 1-ρ |
| n人いる確率 Pₙ | (1-ρ)ρⁿ |

#### 8.3 M/M/c

サーバー c 台。ρ = λ/(cμ) < 1 で安定。

**アーランC公式**（待ち確率）:
C(c, a) = P(待ち) = (aᶜ/c!) / ((1-ρ)Σₖ₌₀^{c-1} aᵏ/k! + aᶜ/c!)
ただし a = λ/μ（提供負荷）

Lq = C(c,a) × ρ/(1-ρ)、Wq = Lq/λ

#### 8.4 M/G/1: ポラチェック・ヒンチンの公式

サービス時間が一般分布（平均 1/μ、分散 σ²）の場合:

Wq = λ(σ² + 1/μ²) / (2(1-ρ))

Lq = λ²(σ² + 1/μ²) / (2(1-ρ))

サービス時間の変動係数 Cs = σμ が大きいほど待ち時間増大。

#### 8.5 リトルの公式

L = λW （システム内平均客数 = 到着率 × 平均滞在時間）

驚くべき一般性: 到着過程やサービス時間分布に依存せず、定常状態で常に成立。
同様に Lq = λWq も成立。

#### 8.6 M/M/1/K（有限容量）

システム容量 K の場合（客の損失あり）:
- Pₙ = (1-ρ)ρⁿ / (1-ρ^{K+1})
- 損失確率 P_K = (1-ρ)ρᴷ / (1-ρ^{K+1})
- 実効到着率 λ_eff = λ(1-P_K)

#### 8.7 実務活用
- **システム性能設計**: Webサーバーのスレッド数・接続プール設計
- **コールセンター**: エージェント数の最適配置（アーランC公式）
- **ネットワーク**: バッファサイズの決定、QoS保証
- **製造ライン**: 工程間のバッファ在庫設計

## Part B: 統計的決定理論

### 9. 決定理論の枠組み

#### 9.1 基本構成要素

統計的決定問題は以下の4つ組で記述される:
- **パラメータ空間** Θ: 真のパラメータ θ ∈ Θ
- **標本空間** X: 観測データ x ∈ X
- **行動空間** A: 取り得る決定（推定値、検定結果、区間等）
- **損失関数** L(θ, a): θ が真のとき行動 a を取った場合の損失

#### 9.2 決定関数

**決定関数（決定規則）** δ: X → A は、データから行動への写像。
- 推定問題: δ(x) = θ̂（推定量）
- 検定問題: δ(x) ∈ {0, 1}（棄却/採択）
- 信頼集合: δ(x) ⊂ Θ

**確率化決定関数**: δ(x, ·) が A 上の確率分布（ランダムに行動を選択）。

#### 9.3 損失関数

| 名称 | L(θ, a) | 用途 |
|---|---|---|
| 二乗誤差損失 | (θ - a)² | 推定（平均的な偏差を重視） |
| 絶対誤差損失 | |θ - a| | 推定（外れ値にロバスト） |
| 0-1損失 | I(a ≠ θ) | 検定・分類 |
| 重み付き二乗誤差 | w(θ)(θ - a)² | θ の領域で重要度が異なる場合 |
| LINEX損失 | e^{c(a-θ)} - c(a-θ) - 1 | 非対称損失（過大/過小評価のコスト非対称） |
| エントロピー損失 | a/θ - log(a/θ) - 1 | スケールパラメータの推定 |

#### 9.4 リスク関数

R(θ, δ) = E_θ[L(θ, δ(X))] = ∫ L(θ, δ(x)) f(x|θ) dx

リスク関数は θ の関数であり、全ての θ で一様に最小なリスクを持つ決定関数は通常存在しない。

#### 9.5 許容性（Admissibility）

δ₁ が δ₂ を**優越（dominate）**: R(θ, δ₁) ≤ R(θ, δ₂) が全 θ で成立し、少なくとも1つの θ で厳密。

δ が**許容的（admissible）**: どの決定関数にも優越されない。

**スタインのパラドックス**: p ≥ 3 のとき、正規分布の平均ベクトルの推定で、
標本平均 X̄ は二乗誤差損失の下で**非許容的**。
ジェームス・スタイン推定量 δJS = (1 - (p-2)/(||X||²)) X が X̄ を優越する。

#### 9.6 実務活用
- **機械学習**: 損失関数の選択がモデルの振る舞いを決定（MSE vs MAE vs Huber）
- **医療**: 偽陽性と偽陰性のコストが非対称 → 非対称損失関数
- **保険**: LINEX損失で保険料算出（過少見積もりのペナルティ大）

### 10. ベイズ決定とミニマックス決定

#### 10.1 ベイズリスク

事前分布 π(θ) のもとでのベイズリスク:
r(π, δ) = E_π[R(θ, δ)] = ∫ R(θ, δ) π(θ) dθ

**ベイズ規則 δ_π**: r(π, δ_π) = inf_δ r(π, δ)

#### 10.2 ベイズ規則の導出

事後分布 π(θ|x) のもとでの事後期待損失を最小化:
δ_π(x) = argmin_a E[L(θ, a) | X = x] = argmin_a ∫ L(θ, a) π(θ|x) dθ

| 損失関数 | ベイズ推定量 |
|---|---|
| 二乗誤差 | 事後平均 E[θ|x] |
| 絶対誤差 | 事後中央値 |
| 0-1 損失 | 事後最頻値（MAP推定） |

#### 10.3 ミニマックス規則

δ* がミニマックス ⟺ sup_θ R(θ, δ*) = inf_δ sup_θ R(θ, δ)

最悪ケースのリスクを最小化する保守的アプローチ。

#### 10.4 ミニマックス定理と完備類定理

**ミニマックス定理**: ある条件のもとで
inf_δ sup_θ R(θ, δ) = sup_π inf_δ r(π, δ)

**完備類定理（ウォルドの定理）**: 適切な正則条件のもとで、ベイズ規則の集合は
本質的完備類（essentially complete class）を成す。
→ 最適な決定関数はベイズ規則の中に必ず存在する。

#### 10.5 ベイズ規則がミニマックスとなる条件

事前分布 π に対するベイズ規則 δ_π が「最も有利でない事前分布（least favorable prior）」に対する
ベイズ規則であり、かつリスク関数が θ に依存しない（定数リスク）とき、δ_π はミニマックス。

**例**: X ~ N(θ, 1) で θ ∈ R、二乗誤差損失。X̄ はベイズリスクが定数（1/n）→ ミニマックス。

#### 10.6 実務活用
- **ロバスト推定**: 分布の仮定に対する最悪ケース保証（ミニマックス）
- **ゲーム理論**: プレイヤーの最適混合戦略（ミニマックス戦略）
- **金融**: ポートフォリオのロバスト最適化（最悪シナリオでのリターン最大化）
- **A/Bテスト**: ベイズ的意思決定（事後損失最小化で早期停止判断）

### 11. 情報理論

#### 11.1 エントロピー

離散確率変数 X の**エントロピー**:
H(X) = -Σₓ p(x) log₂ p(x) （単位: ビット）

連続確率変数の**微分エントロピー**:
h(X) = -∫ f(x) log f(x) dx

性質:
- H(X) ≥ 0（離散）。微分エントロピーは負になりうる
- H(X) ≤ log|X|（離散一様分布で最大）
- H(X, Y) = H(X) + H(Y|X) = H(Y) + H(X|Y)（チェイン・ルール）
- H(X, Y) ≤ H(X) + H(Y)（等号は独立時のみ）

#### 11.2 条件付きエントロピーと相互情報量

**条件付きエントロピー**: H(X|Y) = -Σ_{x,y} p(x,y) log p(x|y)

**相互情報量**: I(X; Y) = H(X) - H(X|Y) = H(Y) - H(Y|X) = H(X) + H(Y) - H(X,Y)

性質:
- I(X; Y) ≥ 0（等号は X, Y 独立時のみ）
- I(X; Y) = I(Y; X)（対称）
- I(X; Y) ≤ min(H(X), H(Y))

#### 11.3 KLダイバージェンス（カルバック・ライブラー情報量）

D_KL(P || Q) = Σₓ p(x) log(p(x)/q(x)) = E_P[log(P/Q)]

性質:
- D_KL(P || Q) ≥ 0（ギブスの不等式。等号は P = Q のとき）
- **非対称**: D_KL(P || Q) ≠ D_KL(Q || P) 一般に
- 三角不等式を満たさない（距離ではない）
- 相互情報量との関係: I(X; Y) = D_KL(P_{XY} || P_X × P_Y)

#### 11.4 フィッシャー情報量との関係

**ド・ブルイの恒等式**: X が密度 f を持つとき、X + √t Z（Z は標準正規）の微分エントロピーについて
d/dt h(X + √t Z) = (1/2) J(X + √t Z)
ここで J はフィッシャー情報量。

**クラメール・ラオの不等式の情報理論的解釈**:
Var(θ̂) ≥ 1/I(θ) は、推定精度はデータの持つ情報量で制限されることを意味する。

#### 11.5 AICの情報理論的解釈

AIC = -2 log L(θ̂) + 2k

これは E[D_KL(f_true || f_θ̂)] の漸近的不偏推定量。
→ AIC最小化 ≈ 真の分布との KL ダイバージェンス最小化。

BIC = -2 log L(θ̂) + k log n はベイズ的なモデル選択基準。

#### 11.6 最大エントロピー原理

与えられた制約条件のもとで、エントロピーを最大化する分布を選ぶ。

| 制約 | 最大エントロピー分布 |
|---|---|
| 支持 [a,b] のみ | 一様分布 U(a,b) |
| 平均 μ, 分散 σ² | 正規分布 N(μ, σ²) |
| 支持 [0,∞), 平均 μ | 指数分布 Exp(1/μ) |
| 平均 μ, 分散 σ², 支持 [0,∞) | ラゲール型分布 |

#### 11.7 実務活用
- **特徴選択**: 相互情報量 I(X; Y) が大きい特徴量を優先選択
- **モデル選択**: AIC/BIC による予測性能とパラメータ数のトレードオフ
- **異常検知**: KLダイバージェンスでトレーニング時との分布の乖離を検出
- **通信路容量**: C = max_{P_X} I(X; Y) で通信の理論的上限を算出
- **深層学習**: 情報ボトルネック理論、VAEのELBO最適化

### 12. 因果推論

#### 12.1 潜在結果モデル（ルービンの因果モデル）

個体 i に対し:
- Yᵢ(1): 処理を受けた場合の潜在結果
- Yᵢ(0): 処理を受けなかった場合の潜在結果
- 個体レベルの因果効果: τᵢ = Yᵢ(1) - Yᵢ(0)

**因果推論の根本問題**: 同一個体で両方の潜在結果を同時に観測できない。

**SUTVA（安定単位処理値仮定）**: 他の個体の処理割当が自分の結果に影響しない。

#### 12.2 平均処理効果

| 効果 | 定義 | 意味 |
|---|---|---|
| ATE | E[Y(1) - Y(0)] | 母集団全体の平均効果 |
| ATT | E[Y(1) - Y(0) \| T=1] | 処理群に限った平均効果 |
| ATU | E[Y(1) - Y(0) \| T=0] | 非処理群に限った平均効果 |
| LATE | E[Y(1) - Y(0) \| complier] | 準拠者に限った局所平均効果 |
| CATE | E[Y(1) - Y(0) \| X=x] | 条件付き平均処理効果 |

#### 12.3 無交絡性と傾向スコア

**強い無視可能性（ignorability）**: {Y(0), Y(1)} ⊥ T | X

この仮定のもとで:
ATE = E_X[E[Y|T=1, X] - E[Y|T=0, X]]

**傾向スコア**: e(x) = P(T=1 | X=x)

**傾向スコア定理（ローゼンバウム・ルービン）**:
{Y(0), Y(1)} ⊥ T | X ⟹ {Y(0), Y(1)} ⊥ T | e(X)

#### 12.4 傾向スコアの利用方法

**マッチング**: 傾向スコアが近い処理群・対照群のペアを作って比較。
- 最近傍マッチング、キャリパーマッチング、カーネルマッチング

**IPTW（逆確率重み付け）**:
ATE = E[TY/e(X)] - E[(1-T)Y/(1-e(X))]

安定化重み: w = T/e(X) + (1-T)/(1-e(X)) を正規化して分散を軽減。

**二重ロバスト推定（AIPW）**: アウトカムモデルと傾向スコアモデルのどちらか一方が正しければ一致性を持つ。

#### 12.5 操作変数法（IV）

交絡がある場合でも因果効果を識別する手法。

**操作変数 Z の条件**:
1. 関連性: Cov(Z, T) ≠ 0（Z は T に影響）
2. 排他制約: Z → Y の直接効果なし（T を通じてのみ影響）
3. 独立性: Z は交絡因子と独立

**2段階最小二乗法（2SLS）**:
- 第1段階: T̂ = α + γZ + ε₁
- 第2段階: Y = β₀ + β₁T̂ + ε₂

β₁ が因果効果の推定値。ワルド推定量: β = Cov(Y,Z) / Cov(T,Z)

**弱操作変数問題**: F統計量 < 10 なら推定が不安定（Stock-Yogo基準）。

#### 12.6 差分の差分法（DID）

処理前後 × 処理/対照の2×2構造:

ATT = (Ȳ₁₁ - Ȳ₁₀) - (Ȳ₀₁ - Ȳ₀₀)
    = (処理群の変化) - (対照群の変化)

**平行トレンド仮定**: 処理がなければ両群は同じ時間変化を示した。

**回帰式**: Y = β₀ + β₁ T + β₂ Post + β₃ (T × Post) + ε
β₃ が DID 推定量。

**多時点DID**: イベントスタディ・デザイン、staggered adoption の場合は
最近の研究で従来の2方向固定効果が偏りうることが示されている（Callaway-Sant'Anna, Sun-Abraham法）。

#### 12.7 回帰不連続デザイン（RDD）

割当変数 X が閾値 c を超えるかどうかで処理が決まる:
- **シャープRDD**: T = I(X ≥ c)（完全に閾値で決定）
- **ファジーRDD**: P(T=1|X) が c で不連続（IVとして解釈）

**推定**: 閾値近傍での局所的因果効果:
τ_RD = lim_{x↓c} E[Y|X=x] - lim_{x↑c} E[Y|X=x]

局所多項式回帰で推定。**帯域幅の選択**が重要（Imbens-Kalyanaraman法など）。

**検証テスト**: McCraryの密度テスト（割当変数の密度が閾値で不連続でないことを確認）。

#### 12.8 構造的因果モデル（パールのdo計算）

**構造的因果モデル (SCM)**:
- 変数集合 V = {V₁, ..., Vₚ}
- 各変数に構造方程式: Vᵢ = fᵢ(pa(Vᵢ), Uᵢ)
- DAG（有向非巡回グラフ）で因果関係を表現

**介入（do演算子）**: do(X = x) は X の構造方程式を X = x に置き換える操作。
P(Y | do(X=x)) ≠ P(Y | X=x) 一般に（交絡がある場合）。

#### 12.9 DAGとd分離

**d分離**: DAG上でパス分析により条件付き独立性を判定。

パスがブロックされる条件:
- チェイン A → B → C: B で条件付けるとブロック
- フォーク A ← B → C: B で条件付けるとブロック
- 合流点（コライダー） A → B ← C: B で条件付け**ない**とブロック、条件付けると開通

**バックドア基準**: X → Y の因果効果を識別するには、X から Y への全てのバックドアパスを
ブロックする変数集合 Z で条件付ければよい:
P(Y | do(X=x)) = Σ_z P(Y | X=x, Z=z) P(Z=z)

**フロントドア基準**: 中間変数 M を用いた識別（バックドアが閉じられない場合の代替）。

#### 12.10 実務活用
- **政策評価**: 教育プログラム・雇用政策の効果測定（DID、RDD）
- **医療**: ランダム化できない場合の治療効果推定（傾向スコア、IV）
- **マーケティング**: 広告・キャンペーンの因果効果（uplift modeling、CATE推定）
- **テック**: プロダクト変更の因果効果（A/Bテストが不可能な場合のRDD、DID）
- **経済学**: 最低賃金の雇用効果（Card-Krueger DID）、兵役の所得効果（アンガー IV）
- **機械学習との融合**: 因果フォレスト（CATE推定）、Double ML（高次元交絡の調整）
