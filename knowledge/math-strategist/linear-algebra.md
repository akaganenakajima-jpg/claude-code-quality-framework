## 1. ベクトル空間

### 1.1 ベクトル空間の公理

体 F（通常 R）上のベクトル空間 V は以下の8公理を満たす:

**加法に関する4公理**:
1. 結合律: (u + v) + w = u + (v + w)
2. 交換律: u + v = v + u
3. 零元の存在: u + 0 = u
4. 逆元の存在: u + (-u) = 0

**スカラー倍に関する4公理**:
5. 分配律(ベクトル): c(u + v) = cu + cv
6. 分配律(スカラー): (c + d)u = cu + du
7. 結合律: c(du) = (cd)u
8. 単位元: 1u = u

### 1.2 部分空間

W ⊂ V が部分空間 ⟺ 以下の3条件を満たす:
1. 0 ∈ W（零ベクトルを含む）
2. u, v ∈ W ⟹ u + v ∈ W（加法に閉じる）
3. c ∈ F, u ∈ W ⟹ cu ∈ W（スカラー倍に閉じる）

**典型的な部分空間の例**:
- 連立方程式 Ax = 0 の解空間（零空間）
- 行列 A の列空間 Col(A) = {Ax | x ∈ R^n}
- 多項式空間の次数 n 以下の部分空間

### 1.3 線形独立・従属

ベクトル {v1, ..., vk} が**線形独立** ⟺ c1v1 + ... + ckvk = 0 が c1=...=ck=0 のみを解に持つ

**判定法**:
- 行列 [v1 ... vk] を行階段形に変形し、ピボット列の数 = k なら独立
- det([v1 ... vk]) ≠ 0 なら独立（正方の場合）

### 1.4 基底と次元

V の基底 B = {v1, ..., vn}:
- V を生成（V の任意の元が B の線形結合で表せる）
- 線形独立

**次元 dim(V)** = 基底のベクトル数（基底の選び方に依らず一定）

**次元公式**:
- dim(U + W) = dim(U) + dim(W) - dim(U ∩ W)

### 1.5 ランク

**ランク rank(A)** = A の列ベクトルの最大線形独立集合の大きさ

**重要定理**: 行ランク = 列ランク（証明: 行基本変形で行階段形にするとピボット数が一致）

**ランクの性質**:
- rank(A) ≤ min(m, n)（m×n行列の場合）
- rank(AB) ≤ min(rank(A), rank(B))
- rank(A'A) = rank(AA') = rank(A)
- rank(A + B) ≤ rank(A) + rank(B)

**ランク・零度定理（次元定理）**:
- rank(A) + dim(ker(A)) = n（列数）

### 1.6 零空間（カーネル）と像空間（レンジ）

**零空間**: ker(A) = {x ∈ R^n | Ax = 0}
- dim(ker(A)) = n - rank(A)（零度 = 列数 - ランク）

**像空間**: Im(A) = {Ax | x ∈ R^n} = Col(A)
- dim(Im(A)) = rank(A)

**実務活用**: 連立方程式の解の構造理解。特に機械学習の特徴量が冗長かどうかの判定に直結。

### 1.7 直和分解

V = U ⊕ W ⟺ V = U + W かつ U ∩ W = {0}

このとき任意の v ∈ V は v = u + w（u ∈ U, w ∈ W）と**一意に**分解される。

dim(V) = dim(U) + dim(W)

**実務活用**: PCAにおける主成分空間と残差空間の直和分解。信号とノイズの分離。

## 2. 行列の基本演算と性質

### 2.1 行列の積

(AB)_{ij} = Σ_k a_{ik} b_{kj}

**性質**:
- 結合律: (AB)C = A(BC)
- 分配律: A(B+C) = AB + AC
- **非可換**: AB ≠ BA（一般に）
- (AB)' = B'A'
- (AB)^{-1} = B^{-1}A^{-1}（共に正則のとき）

**計算量**: m×n と n×p の積 → O(mnp)

### 2.2 転置

(A')_{ij} = A_{ji}

**性質**:
- (A')' = A
- (A+B)' = A' + B'
- (cA)' = cA'
- (AB)' = B'A'

**対称行列**: A = A'（対角線で対称）
**交代行列**: A = -A'（対角成分は0）

任意の正方行列は対称行列と交代行列の和に分解できる:
A = (A+A')/2 + (A-A')/2

### 2.3 逆行列

n次正方行列 A が正則（可逆） ⟺ A^{-1} が存在して AA^{-1} = A^{-1}A = I

**正則行列の等価条件**（重要 — 全て同値）:
1. det(A) ≠ 0
2. rank(A) = n
3. ker(A) = {0}
4. A の列ベクトルが線形独立
5. A の固有値が全て 0 でない
6. Ax = b が任意の b に対して一意解を持つ

**2×2逆行列**:
A = [[a,b],[c,d]] のとき A^{-1} = (1/det(A))[[d,-b],[-c,a]]
det(A) = ad - bc

**逆行列の性質**:
- (A^{-1})^{-1} = A
- (AB)^{-1} = B^{-1}A^{-1}
- (A')^{-1} = (A^{-1})'
- det(A^{-1}) = 1/det(A)

### 2.4 ブロック行列

行列を小行列のブロックに分割して演算:

[[A, B], [C, D]] × [[E], [F]] = [[AE+BF], [CE+DF]]

**シューアの補行列**: M = [[A,B],[C,D]] に対して
- S = D - CA^{-1}B（A が正則のとき）
- det(M) = det(A)det(S)

**実務活用**: 大規模行列の効率的な計算。共分散行列のブロック構造を利用した条件付き分布の導出。

### 2.5 行列式

**定義（余因子展開）**:
det(A) = Σ_{j=1}^n (-1)^{i+j} a_{ij} M_{ij}（第i行で展開）

M_{ij}: (i,j)小行列式（i行j列を除いた部分行列の行列式）

**性質**:
- det(AB) = det(A)det(B)
- det(A') = det(A)
- det(cA) = c^n det(A)（n次行列）
- 行の交換 → 符号反転
- 行のスカラー倍 → 行列式もスカラー倍
- 行の加法 → 行列式不変

**3×3行列式（サラスの公式）**:
det([[a,b,c],[d,e,f],[g,h,i]]) = aei + bfg + cdh - ceg - bdi - afh

### 2.6 クラメルの公式

Ax = b の解（A が正則のとき）:
x_i = det(A_i) / det(A)

A_i: A の第i列を b で置換した行列

**注意**: 計算量 O(n! × n) で非効率。理論的・小規模問題向け。

**実務活用**: 2×2, 3×3の手計算。パラメータの解析解の導出。

## 3. 連立方程式

### 3.1 ガウスの消去法

Ax = b を拡大係数行列 [A|b] の行基本変形で行階段形に変換して解く。

**行基本変形**:
1. 行の交換
2. 行のスカラー倍（0以外）
3. ある行に別の行のスカラー倍を加える

**計算量**: O(n³/3)（前進消去）+ O(n²)（後退代入）

### 3.2 LU分解

A = LU（L: 下三角、U: 上三角）

**手順**: ガウスの消去法の乗数を L に記録
**ピボット選択**: PA = LU（P: 置換行列、数値安定性のため）

Ax = b の解法:
1. Ly = Pb を前進代入で解く
2. Ux = y を後退代入で解く

**利点**: 右辺 b が変わっても分解をやり直す必要がない

**実務活用**: 連立方程式を繰り返し解く場合（同じ係数行列で異なる右辺）。

### 3.3 解の存在と一意性

Ax = b に対して:
- **唯一解**: rank(A) = rank([A|b]) = n
- **不能（解なし）**: rank(A) < rank([A|b])
- **不定（無限解）**: rank(A) = rank([A|b]) < n
  - 自由度 = n - rank(A)

### 3.4 最小二乗解

Ax = b が解を持たないとき、||Ax - b||² を最小化する x を求める。

**法方程式（正規方程式）**:
A'Ax̂ = A'b

A'A が正則ならば: x̂ = (A'A)^{-1}A'b

**幾何学的解釈**: x̂ は b の Col(A) への直交射影 Pb = A(A'A)^{-1}A'b を実現する。

**実務活用**: 線形回帰の基礎。特徴量行列 X と目標ベクトル y に対して β̂ = (X'X)^{-1}X'y。

### 3.5 ムーア・ペンローズ擬逆行列

A⁺ は以下の4条件を満たす唯一の行列:
1. AA⁺A = A
2. A⁺AA⁺ = A⁺
3. (AA⁺)' = AA⁺
4. (A⁺A)' = A⁺A

**SVD経由の計算**: A = UΣV' ならば A⁺ = VΣ⁺U'
（Σ⁺ は Σ の非零対角要素を逆数にしたもの）

**性質**:
- A が正則なら A⁺ = A^{-1}
- 最小二乗解 x̂ = A⁺b は ||x|| も最小化する（最小ノルム最小二乗解）

**実務活用**: rank落ちの場合やp>nの場合の回帰。正則化前の理論的解。

## 4. 固有値・固有ベクトル

### 4.1 固有値問題

Av = λv（v ≠ 0）

λ: 固有値、v: 固有ベクトル

**意味**: 行列 A はベクトル v の方向を変えず、λ倍だけスケーリングする。

### 4.2 特性多項式

det(A - λI) = 0 の根が固有値

**2×2行列**: λ² - tr(A)λ + det(A) = 0
- tr(A) = a + d（対角和 = 固有値の和）
- det(A) = ad - bc（= 固有値の積）

**3×3行列**: λ³ - tr(A)λ² + (余因子の和)λ - det(A) = 0

### 4.3 固有空間

V_λ = {v | Av = λv} = ker(A - λI)

**代数的重複度**: 特性多項式における λ の重複度
**幾何学的重複度**: dim(V_λ) = dim(ker(A - λI))

常に: 幾何学的重複度 ≤ 代数的重複度

### 4.4 対角化

A = PDP^{-1}（P: 固有ベクトルを列に並べた行列、D: 固有値の対角行列）

**対角化可能条件**: 全ての固有値について幾何学的重複度 = 代数的重複度

**対角化の利点**:
- A^k = PD^kP^{-1}（べき乗が容易）
- exp(A) = P exp(D) P^{-1}

**対角化不可能な場合**: ジョルダン標準形 A = PJP^{-1}
- J はジョルダンブロックの対角行列
- ジョルダンブロック: 対角に固有値、上副対角に1

### 4.5 対称行列の性質

**実対称行列**（A = A'）の固有値・固有ベクトルには特別な性質がある:

1. **全ての固有値は実数**
2. **異なる固有値の固有ベクトルは直交する**
3. **必ず対角化可能**（幾何学的重複度 = 代数的重複度）
4. **直交行列で対角化可能**: A = QΛQ'（Q'Q = QQ' = I）

**実務活用**: 共分散行列（対称半正定値）のPCAへの応用。

### 4.6 スペクトル分解

対称行列 A のスペクトル分解:
A = λ1 q1q1' + λ2 q2q2' + ... + λn qnqn'

各 qiqi' はランク1の射影行列。

**意味**: 行列を固有値×固有方向の射影の重ね合わせとして表現。

### 4.7 特異値分解 (SVD)

任意の m×n 行列 A に対して:
A = UΣV'

- U: m×m 直交行列（左特異ベクトル）
- Σ: m×n 対角行列（特異値 σ1 ≥ σ2 ≥ ... ≥ 0）
- V: n×n 直交行列（右特異ベクトル）

**特異値と固有値の関係**:
- σi² = λi(A'A) = λi(AA')
- U の列: AA' の固有ベクトル
- V の列: A'A の固有ベクトル

**幾何学的意味**: 線形変換 A は「回転(V')→ 伸縮(Σ) → 回転(U)」の合成

**低ランク近似（エッカート・ヤングの定理）**:
rank-k 近似 A_k = Σ_{i=1}^k σi ui vi' はフロベニウスノルムで最良:
||A - A_k||_F = √(σ_{k+1}² + ... + σ_r²)

**PCAとの関係**:
- データ行列 X（中心化済み）のSVD: X = UΣV'
- 主成分スコア: XV = UΣ
- 共分散行列: (1/(n-1))X'X = (1/(n-1))VΣ²V'
- V の列が主成分方向

**実務活用**:
- **推薦システム**: ユーザー×アイテム行列の低ランク近似で欠損値を補完
- **画像圧縮**: 上位k個の特異値だけで画像を再構成（k << min(m,n)で大幅圧縮）
- **次元削減**: 高次元データを低次元に射影（ノイズ除去効果あり）
- **潜在意味解析 (LSA)**: 文書×単語行列のSVDでトピック抽出

## 5. 正定値行列

### 5.1 定義

n次実対称行列 A が**正定値** ⟺ 全ての x ≠ 0 に対して x'Ax > 0

**正定値の等価条件**（全て同値）:
1. 全ての固有値が正: λi > 0
2. 全ての主小行列式（首座小行列式）が正
3. x'Ax > 0（全 x ≠ 0）
4. コレスキー分解 A = LL' が存在（L: 下三角、対角正）
5. A = B'B となる正則行列 B が存在

### 5.2 半正定値

A が**半正定値** ⟺ 全ての x に対して x'Ax ≥ 0

**等価条件**:
- 全ての固有値が非負: λi ≥ 0
- A = B'B となる行列 B が存在（B は正則でなくてもよい）

### 5.3 コレスキー分解

正定値行列 A = LL'（L: 正の対角要素を持つ下三角行列）

**計算量**: O(n³/3)（LU分解の約半分）

**アルゴリズム**:
```
l_{jj} = √(a_{jj} - Σ_{k=1}^{j-1} l_{jk}²)
l_{ij} = (a_{ij} - Σ_{k=1}^{j-1} l_{ik}l_{jk}) / l_{jj}  (i > j)
```

**実務活用**: 正定値連立方程式の高速解法。多変量正規分布からのサンプリング（x = Lz, z~N(0,I)）。

### 5.4 共分散行列は半正定値

Σ = E[(X-μ)(X-μ)'] に対して:
x'Σx = E[(x'(X-μ))²] ≥ 0

等号は x'(X-μ) = 0 a.s. のとき。

**実務活用**: 共分散行列の推定で半正定値性が崩れる場合（サンプル数<次元数）、最近傍半正定値行列への射影が必要。

### 5.5 カーネル関数と正定値性

カーネル関数 K(x, y) が正定値 ⟺ 任意の {x1,...,xn} に対してグラム行列 K_{ij} = K(xi, xj) が半正定値

**マーサーの定理**: 正定値カーネルは再生核ヒルベルト空間の内積に対応

**代表的な正定値カーネル**:
- 線形カーネル: K(x,y) = x'y
- 多項式カーネル: K(x,y) = (x'y + c)^d
- RBF（ガウシアン）カーネル: K(x,y) = exp(-||x-y||²/(2σ²))

**実務活用**: SVMのカーネルトリック。ガウス過程回帰の共分散関数設計。

## 6. 行列微分

### 6.1 スカラーのベクトル微分（勾配ベクトル）

f: R^n → R に対して:
∇f = ∂f/∂x = [∂f/∂x1, ..., ∂f/∂xn]'

### 6.2 基本公式

| 関数 f(x) | 勾配 ∂f/∂x |
|-----------|------------|
| a'x | a |
| x'a | a |
| x'x | 2x |
| x'Ax（A対称） | 2Ax |
| x'Ax（A一般） | (A+A')x |
| b'Ax | A'b |

### 6.3 スカラーの行列微分

f: R^{m×n} → R に対して ∂f/∂X は同じサイズの行列:
(∂f/∂X)_{ij} = ∂f/∂X_{ij}

| 関数 f(X) | ∂f/∂X |
|-----------|--------|
| tr(AX) | A' |
| tr(X'A) | A |
| tr(X'AX) | (A+A')X |
| tr(AXB) | A'B' |（これは ∂/∂X ではなく要確認）
| log det(X)（X正定値） | X^{-1} |

### 6.4 ヘッセ行列

H(f) = ∂²f/∂x∂x' は n×n 対称行列:
H_{ij} = ∂²f/(∂xi ∂xj)

- H が正定値 → f は狭義凸（極小点は大域最小）
- H が半正定値 → f は凸
- H が不定 → 鞍点

### 6.5 連鎖律（行列版）

y = f(x), z = g(y) のとき:
∂z/∂x = (∂y/∂x)' (∂z/∂y)

**実務活用**: ニューラルネットワークの誤差逆伝播法（バックプロパゲーション）の数学的基盤。各層の勾配を連鎖律で効率的に計算。

## 7. テンソル

### 7.1 定義

テンソルは多次元配列の一般化:
- 0階テンソル: スカラー
- 1階テンソル: ベクトル
- 2階テンソル: 行列
- 3階以上: 高階テンソル

n1 × n2 × ... × nk のk階テンソル T の要素: T_{i1, i2, ..., ik}

### 7.2 テンソル演算

**モード積**: テンソル T とモード-n に沿った行列 U の積
(T ×_n U)_{...i_{n-1}, j, i_{n+1}...} = Σ_{i_n} T_{...i_n...} U_{j, i_n}

**テンソル内積**: <A, B> = Σ_{i1,...,ik} A_{i1...ik} B_{i1...ik}

**フロベニウスノルム**: ||T||_F = √<T, T>

### 7.3 テンソル分解

**CP分解（CANDECOMP/PARAFAC）**:
T ≈ Σ_{r=1}^R a_r ⊗ b_r ⊗ c_r（ランクR近似）

外積の和に分解。各成分は解釈しやすい。

**タッカー分解**:
T ≈ G ×_1 A ×_2 B ×_3 C

G: コアテンソル、A,B,C: 因子行列
SVDの高階版。PCAの多次元版であるMPCA（多線形主成分分析）に対応。

**テンソルトレイン（TT分解）**:
各要素を行列の積で表現。高階テンソルの効率的な表現。

### 7.4 テンソルの実務活用

- **深層学習**: CNNの重みテンソル（入力チャネル×出力チャネル×高さ×幅）
- **自然言語処理**: 単語×文書×時間のテンソル分解でトピックの時間変化を捉える
- **推薦システム**: ユーザー×アイテム×コンテキストの3階テンソル
- **信号処理**: 多チャネル多周波数信号のブラインド分離
- **量子化学**: 分子軌道の多体相互作用テンソル
- **モデル圧縮**: ニューラルネットワークの重みテンソルを低ランク近似で圧縮

## 付録: 重要な行列の種類と性質

| 行列の種類 | 定義 | 主要性質 |
|-----------|------|---------|
| 対称行列 | A = A' | 固有値は実数、直交対角化可能 |
| 直交行列 | Q'Q = I | 列ベクトルが正規直交、det=±1、||Qx||=||x|| |
| 正定値行列 | x'Ax > 0 | 固有値全正、コレスキー分解可能 |
| 冪等行列 | A² = A | 固有値は0か1、射影行列 |
| ニルポテント行列 | A^k = 0 | 固有値は全て0 |
| 正規行列 | AA' = A'A | ユニタリ対角化可能 |
| 確率行列 | 各行の和=1, 全要素≥0 | マルコフ連鎖の遷移行列 |

## 付録: 計算の tips

**固有値の手計算（2×2）**:
A = [[a,b],[c,d]] のとき
λ = (a+d)/2 ± √((a-d)²/4 + bc)

**行列式の手計算（3×3）**: サラスの公式を使う

**逆行列の手計算（2×2）**: ad-bc で割って対角交換・副対角符号反転
