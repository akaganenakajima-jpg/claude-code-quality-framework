## 1. 確率の数学的基礎

### 1.1 確率空間

確率空間 (Ω, F, P):
- Ω: 標本空間（全ての可能な結果の集合）
- F: 事象の族（σ加法族）
- P: 確率測度（P(Ω)=1, 可算加法性）

**確率変数**: X: Ω → R の可測関数

**確率分布**: P(X ∈ A) = P({ω | X(ω) ∈ A})

### 1.2 条件付き確率とベイズの定理

**条件付き確率**: P(A|B) = P(A∩B)/P(B)

**乗法定理**: P(A∩B) = P(A|B)P(B) = P(B|A)P(A)

**全確率の法則**: P(A) = Σ P(A|Bi)P(Bi)（{Bi} が Ω の分割）

**ベイズの定理**:
P(Bi|A) = P(A|Bi)P(Bi) / Σ_j P(A|Bj)P(Bj)

- P(Bi): 事前確率
- P(A|Bi): 尤度
- P(Bi|A): 事後確率

**連続版**: p(θ|x) = p(x|θ)p(θ) / p(x) = p(x|θ)p(θ) / ∫p(x|θ)p(θ)dθ

**実務活用**: ベイズ分類器、ベイズ最適化、A/Bテストのベイズ解析、スパムフィルター。

### 1.3 期待値と分散

**期待値**: E[X] = Σ xi P(X=xi)（離散）, E[X] = ∫xf(x)dx（連続）

**期待値の線形性**（常に成立、独立不要）:
E[aX + bY] = aE[X] + bE[Y]

**分散**: Var(X) = E[(X-μ)²] = E[X²] - (E[X])²

**分散の性質**:
- Var(aX + b) = a²Var(X)
- Var(X + Y) = Var(X) + Var(Y) + 2Cov(X,Y)
- **独立なら**: Var(X + Y) = Var(X) + Var(Y)

### 1.4 共分散と相関係数

**共分散**: Cov(X,Y) = E[(X-μX)(Y-μY)] = E[XY] - E[X]E[Y]

**性質**:
- Cov(X,X) = Var(X)
- Cov(aX+b, cY+d) = ac Cov(X,Y)
- 独立 → Cov(X,Y) = 0（逆は不成立）

**相関係数**: ρ(X,Y) = Cov(X,Y) / (σX σY)

**性質**:
- -1 ≤ ρ ≤ 1
- |ρ| = 1 ⟺ Y = aX + b（完全線形関係）
- ρ = 0 は無相関（非線形な依存関係は検出できない）

**共分散行列**: Σ = E[(X-μ)(X-μ)']
- 対称行列
- 半正定値（→ linear-algebra.md §5.4）
- 対角要素 = 各変数の分散
- 非対角要素 = 共分散

**実務活用**: ポートフォリオのリスク計算、PCA（共分散行列の固有値分解）、多変量正規分布のパラメータ。

### 1.5 モーメント

**k次モーメント**: E[X^k]
**k次中心モーメント**: E[(X-μ)^k]

**歪度（Skewness）**: γ1 = E[(X-μ)³] / σ³
- γ1 > 0: 右裾が長い（正の歪み）
- γ1 < 0: 左裾が長い（負の歪み）
- γ1 = 0: 対称（正規分布等）

**尖度（Kurtosis）**: γ2 = E[(X-μ)⁴] / σ⁴
- 正規分布: γ2 = 3
- 超過尖度: γ2 - 3（正規分布で0）
- γ2 > 3: 裾が重い（外れ値が多い）
- γ2 < 3: 裾が軽い

**モーメント母関数**: M(t) = E[e^{tX}]
- M^{(k)}(0) = E[X^k]
- 分布を一意に特定（存在すれば）
- 独立なら M_{X+Y}(t) = M_X(t) M_Y(t)

**実務活用**: データの分布形状の把握。金融リスク管理（尖度が大きい→テールリスク大）。正規性の検定。

### 1.6 主要な確率分布とDS数学での役割

| 分布 | パラメータ | 平均 | 分散 | DS数学での重要性 |
|------|-----------|------|------|-----------------|
| ベルヌーイ | p | p | p(1-p) | 二値分類の基礎 |
| 二項 | n, p | np | np(1-p) | 成功回数、信頼区間 |
| ポアソン | λ | λ | λ | 稀な事象のモデル |
| 幾何 | p | 1/p | (1-p)/p² | 初成功までの試行 |
| 正規 | μ, σ² | μ | σ² | 中心極限定理、回帰誤差 |
| 指数 | λ | 1/λ | 1/λ² | 待ち時間、無記憶性 |
| ガンマ | α, β | α/β | α/β² | ベイズ事前、待ち時間合計 |

| ディリクレ | α | αi/Σα | — | トピックモデルの事前 |

## 2. 確率不等式

### 2.1 マルコフの不等式

X ≥ 0, a > 0 に対して:
P(X ≥ a) ≤ E[X] / a

**証明**: E[X] = ∫_0^∞ x f(x)dx ≥ ∫_a^∞ x f(x)dx ≥ a ∫_a^∞ f(x)dx = a P(X≥a)

**用途**: 他の不等式の導出の基盤。分布の形状を仮定せずに上界を与える。

### 2.2 チェビシェフの不等式

P(|X - μ| ≥ kσ) ≤ 1/k²

**証明**: マルコフの不等式を (X-μ)² に適用

**具体的な値**:
- k=2: P(|X-μ| ≥ 2σ) ≤ 0.25
- k=3: P(|X-μ| ≥ 3σ) ≤ 0.111
- 正規分布では: k=2 → 0.046, k=3 → 0.003（チェビシェフよりずっと小さい）

**実務活用**: 分布に依らない信頼区間の構成。外れ値の定量的基準。

### 2.3 イェンセンの不等式

φ が凸関数なら: φ(E[X]) ≤ E[φ(X)]
φ が凹関数なら: φ(E[X]) ≥ E[φ(X)]

**例**:
- φ(x) = x²（凸）: (E[X])² ≤ E[X²] → Var(X) ≥ 0
- φ(x) = log(x)（凹）: log(E[X]) ≥ E[log(X)]
- φ(x) = e^x（凸）: e^{E[X]} ≤ E[e^X]

**実務活用**:
- **ELBO（変分下界）の導出**: log p(x) = log ∫ p(x,z)dz = log E_q[p(x,z)/q(z)] ≥ E_q[log(p(x,z)/q(z))]
- **EM法の理論的基盤**: E-stepでの下界の構成
- **KLダイバージェンスの非負性**: KL(p||q) ≥ 0

### 2.4 ヘフディングの不等式

X1, ..., Xn が独立、ai ≤ Xi ≤ bi のとき:
P(|X̄ - E[X̄]| ≥ t) ≤ 2 exp(-2n²t² / Σ(bi-ai)²)

特に Xi ∈ [0,1] なら:
P(|X̄ - μ| ≥ t) ≤ 2 exp(-2nt²)

**チェビシェフとの比較**: 指数的に減衰（チェビシェフは多項式的）。

**実務活用**: PAC学習理論の基盤。サンプルサイズの決定。バンディット問題のUCBアルゴリズム。

### 2.5 チェルノフ限界

P(X ≥ a) ≤ min_{t>0} e^{-ta} E[e^{tX}] = min_{t>0} e^{-ta} M(t)

モーメント母関数を利用した最も鋭い指数的上界。

**二項分布への適用**: X ~ Bin(n, p) のとき
P(X ≥ (1+δ)np) ≤ (e^δ / (1+δ)^{1+δ})^{np}

**実務活用**: 通信理論の誤り確率の上界。ランダムアルゴリズムの失敗確率の評価。

### 2.6 集中不等式の機械学習への応用

**汎化誤差の上界**:
真のリスク R(h) と経験リスク R̂(h) の差の上界:

P(sup_{h∈H} |R̂(h) - R(h)| ≥ ε) ≤ 2|H| exp(-2nε²)

（有限仮説クラスの場合。ヘフディングの不等式＋合併上界）

**VC次元との関係**: 仮説クラスが無限の場合:
P(sup_{h∈H} |R̂(h) - R(h)| ≥ ε) ≤ 4 S_H(2n) exp(-nε²/8)

S_H(n): 成長関数、VC次元 d で O(n^d) に制御

**サンプル複雑度**: 汎化誤差を ε 以内に抑えるために必要なサンプル数:
n = O(d log(1/ε) + log(1/δ)) / ε²

**実務活用**: モデル選択の理論的根拠。過学習の定量的理解。正則化の必要性の理論的説明。

## 3. 情報理論

### 3.1 エントロピー

**離散エントロピー**: H(X) = -Σ p(x) log p(x)

**性質**:
- H(X) ≥ 0
- H(X) ≤ log|X|（一様分布で最大）
- H(X) = 0 ⟺ X が確定的
- log の底: 2（ビット）、e（ナット）、10（ディジット）

**連続版（微分エントロピー）**: h(X) = -∫ f(x) log f(x) dx
- 微分エントロピーは負になりうる
- 分散 σ² を固定すると正規分布が最大エントロピー: h(N(μ,σ²)) = (1/2)log(2πeσ²)

### 3.2 条件付きエントロピー

H(Y|X) = -Σ_{x,y} p(x,y) log p(y|x) = Σ_x p(x) H(Y|X=x)

**連鎖律**: H(X,Y) = H(X) + H(Y|X) = H(Y) + H(X|Y)

**性質**: H(Y|X) ≤ H(Y)（条件付けで不確実性は減少）
等号は X, Y が独立のとき。

### 3.3 相互情報量

I(X;Y) = H(X) - H(X|Y) = H(Y) - H(Y|X) = H(X) + H(Y) - H(X,Y)

= Σ_{x,y} p(x,y) log(p(x,y)/(p(x)p(y)))

= KL(p(x,y) || p(x)p(y))

**性質**:
- I(X;Y) ≥ 0
- I(X;Y) = 0 ⟺ X, Y が独立
- I(X;Y) = I(Y;X)（対称）
- I(X;X) = H(X)

**実務活用**: 特徴量選択（目的変数との相互情報量が大きい特徴量を選ぶ）。

### 3.4 KLダイバージェンス

KL(p||q) = Σ p(x) log(p(x)/q(x)) = E_p[log(p/q)]

**性質**:
- KL(p||q) ≥ 0（ギブスの不等式。イェンセンの不等式から導出）
- KL(p||q) = 0 ⟺ p = q
- **非対称**: KL(p||q) ≠ KL(q||p)
- 三角不等式を満たさない → 距離の公理を満たさない

**正規分布間のKL**:
KL(N(μ1,σ1²) || N(μ2,σ2²)) = log(σ2/σ1) + (σ1² + (μ1-μ2)²)/(2σ2²) - 1/2

**多変量正規**:
KL(N(μ1,Σ1) || N(μ2,Σ2)) = (1/2)[tr(Σ2^{-1}Σ1) + (μ2-μ1)'Σ2^{-1}(μ2-μ1) - n + log(det(Σ2)/det(Σ1))]

**Forward KL vs Reverse KL**:
- Forward KL(p||q): qがpの全域をカバー（モード平均化、変分推論でq→平均場近似）
- Reverse KL(q||p): qがpのモードの1つに集中（モード探索）

**実務活用**: VAEの損失関数（再構成誤差 + KLダイバージェンス）。方策最適化。分布間の類似度。

### 3.5 JSダイバージェンス

JS(p||q) = (1/2)KL(p||M) + (1/2)KL(q||M)　（M = (p+q)/2）

**性質**:
- 対称: JS(p||q) = JS(q||p)
- 0 ≤ JS ≤ log2
- √JS は距離の公理を満たす（計量）

**実務活用**: GAN（Generative Adversarial Networks）の理論解析。KLの対称化版。

### 3.6 クロスエントロピー

H(p, q) = -Σ p(x) log q(x) = H(p) + KL(p||q)

**分類損失関数としてのクロスエントロピー**:
L = -Σ_{i=1}^C yi log ŷi

- 二値分類: L = -[y log ŷ + (1-y) log(1-ŷ)]
- p = 真の分布（one-hot）、q = モデルの予測確率
- クロスエントロピー最小化 = KLダイバージェンス最小化（H(p)は定数のため）
- = 最尤推定と等価

**実務活用**: ロジスティック回帰・ニューラルネットワークの損失関数。分類モデルの学習。

### 3.7 最大エントロピー原理

既知の制約（期待値等）の下でエントロピーを最大化する分布を選ぶ。

**結果**:
- 制約なし: 一様分布
- 平均制約（E[X]=μ, X≥0）: 指数分布
- 平均・分散制約: 正規分布

**指数型分布族との関係**:
最大エントロピー分布は指数型分布族に属する:
p(x) = exp(Σ θi fi(x) - A(θ))

**実務活用**: 最大エントロピーモデル（ロジスティック回帰の別名）。情報が限られた状況での確率モデル選択。

## 4. ランダム行列理論の基礎

### 4.1 高次元統計の課題（p > n 問題）

次元 p がサンプル数 n を超えるとき:
- 標本共分散行列 S = (1/n)X'X は特異（rank ≤ n < p）
- 固有値分布が真の共分散行列と大きく乖離
- 最大固有値の過大推定、最小固有値の過小推定（0になる）

### 4.2 マルチェンコ・パスツール則

p/n → γ（0 < γ < ∞）のとき、白色ノイズ（Σ=I）の標本共分散行列の固有値分布は:

f(λ) = (1/(2πγλ))√((λ+-λ)(λ-λ-))

支持区間: [λ-, λ+] = [(1-√γ)², (1+√γ)²]

γ > 1 のとき、p-n 個の零固有値が追加で現れる。

**意味**: ノイズのみのデータでも固有値は [λ-, λ+] に広がる。この範囲を超える固有値だけが真の信号。

### 4.3 信号とノイズの分離

**バルクエッジ**: λ+ = (1+√γ)² を超える固有値 = 信号成分の候補

**BBP相転移（Baik-Ben Arous-Peche）**:
真の固有値 l が 1 + √γ を超えるとき、標本固有値が λ+ から飛び出す。
l ≤ 1 + √γ のとき、標本固有値はバルクに埋もれ検出不可能。

**実務活用**: 高次元PCAでの主成分数の決定。ノイズフロアの推定。金融データの因子モデル。

### 4.4 共分散行列の正則化

p > n の場合の対策:
- **縮小推定（Shrinkage）**: Ŝ = αS + (1-α)I（Ledoit-Wolf推定量）
  - α の最適値: 解析的に求まる
- **スパース推定（Graphical Lasso）**: L1正則化で逆共分散行列を推定
- **因子モデル**: Σ ≈ LL' + Ψ（低ランク+対角）

**実務活用**: 高次元データのポートフォリオ最適化。遺伝子発現データの解析。画像認識での共分散推定。

## 付録: 確率不等式の使い分けガイド

| 不等式 | 条件 | 上界の形 | 鋭さ | 用途 |
|--------|------|---------|------|------|
| マルコフ | X≥0 | O(1/a) | 粗い | 理論の基盤 |
| チェビシェフ | 分散有限 | O(1/k²) | 粗い | 分布不問の信頼区間 |
| ヘフディング | 独立・有界 | exp(-nt²) | 中程度 | 汎化誤差、バンディット |
| チェルノフ | MGF存在 | exp(-nI(a)) | 鋭い | 誤り確率の精密評価 |
| ベルンシュタイン | 独立・有界 | exp(-nt²/(σ²+t)) | 鋭い | 分散も考慮 |

## 付録: DS数学で問われる典型的な計算問題

1. **エントロピー計算**: H(X) = -Σ pi log pi を具体的な分布で計算
2. **KL計算**: 2つの正規分布間のKLダイバージェンスを公式で計算
3. **ベイズ更新**: 事前Beta(α,β) + 二項データ → 事後Beta(α+k, β+n-k)
4. **相互情報量**: 同時分布表から I(X;Y) を計算
5. **確率不等式の適用**: n個のサンプルで誤差ε以内に収まる確率の下界
