# 微積分・最適化 — DS数学ストラテジスト上級リファレンス

> 機械学習の「学習」は本質的に最適化問題。微積分はその言語であり、最適化は核心技術。

---

## 1. 多変数微分

### 1.1 偏微分

f(x1, ..., xn) の xi に関する偏微分:
∂f/∂xi = lim_{h→0} [f(x1,...,xi+h,...,xn) - f(x1,...,xn)] / h

他の変数を定数として扱い、1変数の微分を行う。

### 1.2 全微分

df = (∂f/∂x1)dx1 + (∂f/∂x2)dx2 + ... + (∂f/∂xn)dxn = ∇f' dx

全微分は各変数の微小変化に対する f の変化量の1次近似。

**全微分と偏微分の関係**:
z = f(x(t), y(t)) のとき dz/dt = (∂f/∂x)(dx/dt) + (∂f/∂y)(dy/dt)

### 1.3 方向微分

方向 u（||u||=1）への方向微分:
D_u f(x) = ∇f(x)' u = |∇f| cos θ

- 勾配方向（θ=0）で最大: D_u f = |∇f|
- 反勾配方向（θ=π）で最小: D_u f = -|∇f|
- 勾配に直交する方向で 0

**実務活用**: 勾配降下法は「最も急な降下方向」= -∇f の方向に進む。

### 1.4 勾配ベクトルとヘッセ行列

**勾配ベクトル**:
∇f = [∂f/∂x1, ∂f/∂x2, ..., ∂f/∂xn]'

**ヘッセ行列**:
H_{ij} = ∂²f/(∂xi ∂xj)

f が2回連続微分可能なら H は対称（シュワルツの定理: ∂²f/∂xi∂xj = ∂²f/∂xj∂xi）

**実務活用**: ∇f は損失関数の勾配（学習の方向）、H は曲率情報（学習率の適応に利用）。

### 1.5 テイラー展開

**1次近似（線形近似）**:
f(x+h) ≈ f(x) + ∇f(x)' h

**2次近似（二次近似）**:
f(x+h) ≈ f(x) + ∇f(x)' h + (1/2) h' H(x) h

**実務活用**:
- 1次近似: 勾配降下法の理論的根拠
- 2次近似: ニュートン法の理論的根拠。損失関数の局所的形状の把握。

### 1.6 陰関数定理

F(x, y) = 0 で ∂F/∂y ≠ 0 ならば、y = g(x) が局所的に存在し:
dy/dx = -(∂F/∂x) / (∂F/∂y)

**多変数版**: F: R^{n+m} → R^m, det(∂F/∂y) ≠ 0 ならば y = g(x) が局所的に存在

**実務活用**: 制約条件下でのパラメータの感度分析。

### 1.7 ヤコビ行列とヤコビアン

f: R^n → R^m に対するヤコビ行列 J:
J_{ij} = ∂fi/∂xj（m×n行列）

**ヤコビアン**: J が正方（m=n）のとき det(J)

**変数変換での利用**:
(x, y) → (r, θ) のとき dxdy = |det(J)| drdθ

極座標変換: x = rcosθ, y = rsinθ
J = [[cosθ, -rsinθ], [sinθ, rcosθ]]
|det(J)| = r
→ dxdy = r dr dθ

**実務活用**: 確率分布の変数変換。ニューラルネットワークの正規化フロー（Normalizing Flows）。

## 2. 多変数積分

### 2.1 重積分

∫∫_D f(x,y) dxdy

**累次積分（フビニの定理）**:
∫∫_D f(x,y) dxdy = ∫_a^b [∫_{g1(x)}^{g2(x)} f(x,y) dy] dx

積分順序の交換が可能（f が連続、D が有界閉領域のとき）。

### 2.2 変数変換

(x,y) → (u,v) の変換で:
∫∫ f(x,y) dxdy = ∫∫ f(x(u,v), y(u,v)) |∂(x,y)/∂(u,v)| dudv

**極座標**: dxdy = r drdθ
**球座標**: dxdydz = r²sinφ drdθdφ

### 2.3 ガウス積分

∫_{-∞}^{∞} exp(-x²) dx = √π

**一般形**:
∫_{-∞}^{∞} exp(-ax²) dx = √(π/a)　（a > 0）

**導出**: I² = ∫∫ exp(-(x²+y²)) dxdy に極座標変換して r について積分

**多変数ガウス積分**:
∫ exp(-(1/2)x'Ax) dx = (2π)^{n/2} / √det(A)　（A正定値）

**実務活用**: 正規分布の正規化定数の導出。ベイズ推論での周辺尤度の計算。

### 2.4 ガンマ関数

Γ(s) = ∫_0^∞ t^{s-1} e^{-t} dt　（s > 0）

**性質**:
- Γ(s+1) = s Γ(s)（再帰関係）
- Γ(n+1) = n!（自然数）
- Γ(1/2) = √π
- Γ(1) = 1

**スターリングの近似**: n! ≈ √(2πn)(n/e)^n

**実務活用**: ガンマ分布、t分布、F分布、ディリクレ分布の正規化定数。

### 2.5 ベータ関数

B(a, b) = ∫_0^1 t^{a-1}(1-t)^{b-1} dt = Γ(a)Γ(b) / Γ(a+b)

**性質**:
- B(a, b) = B(b, a)（対称）
- B(1, 1) = 1

**実務活用**: ベータ分布の正規化定数。ベイズ推論における事前分布（二項分布の共役事前）。

## 3. 無制約最適化

### 3.1 極値の条件

**必要条件（1次）**: 極値点 x* で ∇f(x*) = 0（停留点）

**十分条件（2次）**: 停留点 x* で
- H(x*) が正定値 → **極小**
- H(x*) が負定値 → **極大**
- H(x*) が不定（正と負の固有値を持つ）→ **鞍点**
- H(x*) が半正定値/半負定値 → 判定不能（高次の項が必要）

**2変数の場合**: H = [[f_xx, f_xy], [f_xy, f_yy]]
- D = f_xx f_yy - f_xy² > 0 かつ f_xx > 0 → 極小
- D > 0 かつ f_xx < 0 → 極大
- D < 0 → 鞍点
- D = 0 → 判定不能

### 3.2 勾配降下法

x_{k+1} = x_k - α ∇f(x_k)

α: 学習率（ステップサイズ）

**収束条件**:
- f が凸かつ L-リプシッツ連続な勾配を持つとき
- 0 < α < 2/L で収束
- α = 1/L で最速（条件数が悪いと遅い）
- 収束速度: O(1/k)（凸）、O(ρ^k)（強凸、ρ = (κ-1)/(κ+1)、κ=条件数）

**学習率の選択**:
- 固定: 小さすぎると遅い、大きすぎると発散
- 減衰: α_k = α_0/k（理論的収束保証）
- ライン探索: α_k = argmin_α f(x_k - α∇f(x_k))
- Armijo条件/Wolfe条件による不正確なライン探索

### 3.3 ニュートン法

x_{k+1} = x_k - H(x_k)^{-1} ∇f(x_k)

**2次近似の最小化**に基づく: f(x+h) ≈ f(x) + ∇f'h + (1/2)h'Hh の最小化 → h = -H^{-1}∇f

**収束速度**: 2次収束（十分近いと ||x_{k+1}-x*|| ≤ C||x_k-x*||²）

**問題点**:
- H の計算コスト O(n²)、逆行列 O(n³)
- H が正定値でないと降下方向にならない
- 初期点が遠いと収束しない

### 3.4 準ニュートン法

H^{-1} を近似行列 B_k で置換: x_{k+1} = x_k - α_k B_k ∇f(x_k)

**BFGS（Broyden-Fletcher-Goldfarb-Shanno）**:
勾配情報のみで B_k を逐次更新（セカント条件 B_{k+1} s_k = y_k を満たす）

s_k = x_{k+1} - x_k, y_k = ∇f(x_{k+1}) - ∇f(x_k)

**L-BFGS（Limited-memory BFGS）**:
直近 m 回分の {s_k, y_k} のみ保持（メモリ O(mn)）。大規模問題向き。

**収束速度**: 超線形収束

**実務活用**: ロジスティック回帰、CRFなど中規模凸最適化の標準手法。

### 3.5 確率的勾配降下法 (SGD)

x_{k+1} = x_k - α_k ∇f_i(x_k)

全データの勾配 ∇f = (1/n)Σ∇f_i の代わりに、ランダムサンプル i の勾配を使用。

**ミニバッチSGD**: B 個のサンプルの平均勾配を使用
x_{k+1} = x_k - α_k (1/|B|) Σ_{i∈B} ∇f_i(x_k)

**分散**: E[||∇f_i - ∇f||²] = σ²（ノイズの分散）
- バッチサイズ B → 分散 σ²/B に減少

**モメンタム**:
v_{k+1} = μ v_k + ∇f(x_k)
x_{k+1} = x_k - α v_{k+1}

慣性項 μv_k が振動を抑制し、谷底方向への移動を加速。

**Nesterov加速勾配法**:
v_{k+1} = μ v_k + ∇f(x_k - αμv_k)
x_{k+1} = x_k - α v_{k+1}

「先読み」して勾配を計算。収束速度 O(1/k²)（凸関数で最適）。

**Adam（Adaptive Moment Estimation）**:
m_k = β1 m_{k-1} + (1-β1) g_k　　（1次モーメント推定）
v_k = β2 v_{k-1} + (1-β2) g_k²　　（2次モーメント推定）
m̂_k = m_k / (1-β1^k)　　（バイアス補正）
v̂_k = v_k / (1-β2^k)
x_{k+1} = x_k - α m̂_k / (√v̂_k + ε)

デフォルト: β1=0.9, β2=0.999, ε=1e-8

パラメータごとに適応的な学習率。スパースな勾配に強い。

**実務活用**: 深層学習の学習はほぼ全てSGD系。Adam が汎用デフォルト。大規模データに対してフルバッチ勾配降下は非現実的。

### 3.6 収束速度の比較

| 手法 | 収束速度 | 1ステップコスト | 適用場面 |
|------|---------|---------------|---------|
| 勾配降下法 | O(1/k) 凸, 線形 強凸 | O(n) | 小〜中規模、凸 |
| ニュートン法 | 2次 | O(n³) | 小規模、高精度 |
| BFGS | 超線形 | O(n²) | 中規模 |
| L-BFGS | 超線形 | O(mn) | 大規模 |
| SGD | O(1/√k) | O(B×d) | 大規模データ |
| Adam | 実用的に高速 | O(d) | 深層学習 |

## 4. 制約付き最適化

### 4.1 ラグランジュの未定乗数法

minimize f(x) subject to g_i(x) = 0 (i=1,...,m)

**ラグランジュ関数**: L(x, λ) = f(x) + Σ λi gi(x)

**必要条件**: ∇_x L = 0 かつ g_i(x) = 0

具体的には:
∂f/∂xj + Σ λi ∂gi/∂xj = 0 (j=1,...,n)
gi(x) = 0 (i=1,...,m)

**例**: f(x,y) = x² + y² を x + y = 1 の下で最小化
L = x² + y² + λ(x+y-1)
∂L/∂x = 2x + λ = 0 → x = -λ/2
∂L/∂y = 2y + λ = 0 → y = -λ/2
x + y = 1 → -λ = 1 → λ = -1
解: (x,y) = (1/2, 1/2), f = 1/2

**実務活用**: 等式制約付きの最適化全般。エントロピー最大化、正規化制約下のパラメータ推定。

### 4.2 KKT条件

minimize f(x) subject to g_i(x) ≤ 0, h_j(x) = 0

**KKT（Karush-Kuhn-Tucker）条件**（必要条件）:
1. **停留条件**: ∇f + Σ μi ∇gi + Σ λj ∇hj = 0
2. **主実行可能性**: gi(x) ≤ 0, hj(x) = 0
3. **双対実行可能性**: μi ≥ 0
4. **相補性条件**: μi gi(x) = 0（各 i について）

相補性条件の意味: μi > 0 なら gi(x) = 0（制約がアクティブ）

**凸最適化ではKKT条件は十分条件でもある**（制約想定の下で）。

### 4.3 双対問題

**ラグランジュ双対関数**:
g(λ, μ) = inf_x L(x, λ, μ)

**双対問題**: maximize g(λ, μ) subject to μ ≥ 0

**弱双対性**: 常に g(λ*, μ*) ≤ f(x*)（双対最適値 ≤ 主最適値）
**強双対性**: g(λ*, μ*) = f(x*)（スレーターの条件下で成立。凸計画で内点が存在すれば十分）

**双対ギャップ**: f(x) - g(λ, μ) ≥ 0（収束判定に使用）

### 4.4 凸最適化

**凸関数の定義**:
f(θx + (1-θ)y) ≤ θf(x) + (1-θ)f(y)　（0 ≤ θ ≤ 1）

**凸関数の判定**:
- 1次条件: f(y) ≥ f(x) + ∇f(x)'(y-x)（接線が下からの支え）
- 2次条件: H(x) が半正定値（全ての x で）

**凸計画の大域最適性**: 凸関数の局所最小 = 大域最小

**凸関数の例**:
- 線形関数、アフィン関数
- 二次関数 x'Ax + b'x + c（A半正定値）
- ノルム ||x||
- exp(x), -log(x)（x>0）
- max(x1, ..., xn)

**凸性の保存操作**:
- 非負重み付き和: Σ αi fi（αi ≥ 0, fi 凸）は凸
- 点ごとの最大: max(f1,...,fk) は凸
- アフィン合成: f(Ax+b) は凸（f が凸のとき）

### 4.5 線形計画法

minimize c'x subject to Ax ≤ b, x ≥ 0

**標準形**: minimize c'x subject to Ax = b, x ≥ 0

**単体法**: 実行可能領域（凸多面体）の頂点を巡回して最適解を探索
- 最悪 O(2^n) だが実用的には高速
- ピボット選択: 最大係数法（Dantzig）等

**内点法**: 実行可能領域の内部を通って最適解に収束
- 多項式時間 O(n^{3.5})
- バリア関数: f(x) + (1/t) Σ(-log(xi))

**双対定理**: 主問題が最適解を持つ ⟺ 双対問題が最適解を持つ、かつ最適値は等しい

**実務活用**: スケジューリング、輸送問題、ネットワークフロー。

### 4.6 二次計画法

minimize (1/2)x'Qx + c'x subject to Ax ≤ b

Q が半正定値なら凸二次計画（大域最適解が効率的に求まる）。

**実務活用**: SVMの双対問題、ポートフォリオ最適化（平均分散モデル）。

### 4.7 SVMの双対問題

**主問題**: minimize (1/2)||w||² subject to yi(w'xi + b) ≥ 1

**双対問題**: maximize Σαi - (1/2)ΣΣ αiαj yiyj xi'xj
subject to αi ≥ 0, Σαi yi = 0

**KKT相補性**: αi[yi(w'xi + b) - 1] = 0
→ αi > 0 のサンプルがサポートベクター

**カーネルトリック**: xi'xj → K(xi, xj) に置換（特徴空間での内積を直接計算）

**実務活用**: 高次元データの分類。カーネル法による非線形分離。

### 4.8 ポートフォリオ最適化（マーコウィッツ・モデル）

minimize (1/2)w'Σw subject to μ'w = r（目標収益率）, 1'w = 1（予算制約）

Σ: 共分散行列（半正定値）、μ: 期待収益ベクトル

ラグランジュ乗数法で解析解が得られる:
w* = Σ^{-1}(λ1 μ + λ2 1)

効率的フロンティア: 目標収益率 r を変えて得られる最小分散ポートフォリオの集合

**実務活用**: 金融工学の基盤。リスク・リターンのトレードオフ最適化。

## 5. 変分法

### 5.1 汎関数

汎関数: 関数を入力として実数を返す写像
J[y] = ∫_a^b F(x, y(x), y'(x)) dx

### 5.2 オイラー・ラグランジュ方程式

J[y] を最小化する y(x) の必要条件:
∂F/∂y - d/dx(∂F/∂y') = 0

**例**: 2点間の最短経路
J[y] = ∫ √(1 + y'²) dx
∂F/∂y = 0, ∂F/∂y' = y'/√(1+y'²)
→ y'/√(1+y'²) = const → y = ax + b（直線）

### 5.3 変分法の拡張

**多変数**: F(x, y1, ..., yn, y1', ..., yn') に対して各 yi について EL方程式
**高階微分**: F(x, y, y', y'') に対して ∂F/∂y - d/dx(∂F/∂y') + d²/dx²(∂F/∂y'') = 0
**制約付き**: イソペリメトリック問題 → ラグランジュ乗数を導入

**実務活用**: 最適制御理論（ロケット軌道最適化等）。測地線の計算。変分推論（ELBO最大化）はこの離散版。

## 6. 情報幾何の基礎

### 6.1 統計的多様体

パラメトリック確率分布の族 {p(x|θ)} はパラメータ空間 Θ 上の多様体を成す。

### 6.2 フィッシャー情報行列

F_{ij}(θ) = E[∂log p(x|θ)/∂θi × ∂log p(x|θ)/∂θj]

= -E[∂²log p(x|θ)/(∂θi ∂θj)]（正則条件下で等しい）

**性質**:
- 半正定値
- リパラメトリゼーション不変（テンソルとして変換）
- クラメール・ラオの下界: Var(θ̂) ≥ F^{-1}/n

**フィッシャー計量**: ds² = Σ F_{ij} dθi dθj（統計的多様体上のリーマン計量）

### 6.3 KLダイバージェンス

KL(p||q) = ∫ p(x) log(p(x)/q(x)) dx = E_p[log(p/q)]

**性質**:
- KL(p||q) ≥ 0（ギブスの不等式）
- KL(p||q) = 0 ⟺ p = q（a.e.）
- **非対称**: KL(p||q) ≠ KL(q||p)（距離ではない）
- **三角不等式を満たさない**

**テイラー展開との関係**:
KL(p(x|θ)||p(x|θ+dθ)) ≈ (1/2) dθ' F(θ) dθ

→ KLダイバージェンスの局所的な形がフィッシャー計量

### 6.4 自然勾配法

通常の勾配降下: θ_{k+1} = θ_k - α ∇L(θ_k)
自然勾配降下: θ_{k+1} = θ_k - α F(θ_k)^{-1} ∇L(θ_k)

**動機**: パラメータ空間のユークリッド距離ではなく、分布間のKLダイバージェンスに基づく最急降下。

**利点**: パラメトリゼーションに依存しない（リパラメトリゼーション不変）。条件数の悪い問題で高速収束。

**実務活用**:
- **深層学習**: KFAC（Kronecker-Factored Approximate Curvature）による近似自然勾配
- **強化学習**: TRPO（Trust Region Policy Optimization）、PPO の理論的根拠
- **変分推論**: 自然勾配VIで指数型分布族の効率的学習

## 付録: 最適化手法の選択ガイド

| 問題の性質 | 推奨手法 | 理由 |
|-----------|---------|------|
| 凸・小規模（n<1000） | ニュートン法/BFGS | 2次/超線形収束 |
| 凸・大規模 | L-BFGS | メモリ効率 |
| 大規模データ・深層学習 | Adam/SGD+モメンタム | 確率的手法が現実的 |
| 線形制約 | 線形計画法（内点法） | 多項式時間保証 |
| 二次目的+線形制約 | 二次計画法 | SVM等 |
| 等式制約 | ラグランジュ乗数法 | 解析解が得られる場合 |
| 不等式制約 | KKT+内点法 | 凸なら大域最適 |
| 非凸・離散 | 焼きなまし/遺伝的アルゴリズム | → applied-math.md §4 |

## 付録: よく使う微分公式

| 関数 | 微分 |
|------|------|
| x^n | nx^{n-1} |
| e^{ax} | ae^{ax} |
| ln(x) | 1/x |
| sin(x) | cos(x) |
