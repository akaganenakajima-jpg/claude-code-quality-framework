## 1. グラフ理論

### 1.1 グラフの基本

**グラフ G = (V, E)**: 頂点集合 V と辺集合 E

**種類**:
- 無向グラフ: E ⊂ {{u,v} | u,v ∈ V}
- 有向グラフ: E ⊂ {(u,v) | u,v ∈ V}
- 重み付きグラフ: 各辺に重み w(e) が付与

**隣接行列 A**: A_{ij} = 1 if (i,j) ∈ E, else 0
- 無向グラフ: A は対称
- 有向グラフ: A は非対称

**次数**: deg(v) = v に接続する辺の数
- 有向: 入次数 deg⁻(v) + 出次数 deg⁺(v)
- 握手定理: Σ deg(v) = 2|E|

**経路と連結性**:
- 経路: 辺で繋がった頂点の列
- 連結: 任意の2頂点間に経路が存在
- 連結成分: 極大連結部分グラフ

**木（ツリー）**: 連結かつ閉路を持たないグラフ
- |E| = |V| - 1
- 任意の2頂点間の経路が一意

### 1.2 最短経路

**ダイクストラ法**: 非負重みの単一始点最短経路
- 計算量: O(|V|² )（素朴）、O((|V|+|E|)log|V|)（ヒープ使用）
- 貪欲法: 未確定頂点の中から最小距離を選んで確定

**ベルマン・フォード法**: 負の重みも可（負閉路検出も可能）
- 計算量: O(|V||E|)
- |V|-1 回の緩和操作

**フロイド・ワーシャル法**: 全点対最短経路
- 計算量: O(|V|³)
- 動的計画法: d[i][j] = min(d[i][j], d[i][k]+d[k][j])

**実務活用**: 経路探索、ネットワーク最適化、社会ネットワーク分析（中心性の計算）。

### 1.3 最小全域木

**クラスカル法**: 辺を重み昇順にソートし、閉路を作らない辺を追加
- 計算量: O(|E|log|E|)
- Union-Find で閉路判定

**プリム法**: ある頂点から開始し、最小重みの辺で隣接頂点を追加
- 計算量: O(|E|log|V|)（ヒープ使用）

**実務活用**: ネットワーク設計（通信網の最小コスト構築）。クラスタリング（MST-based clustering）。

### 1.4 PageRank

ウェブページの重要度スコア:
r(i) = (1-d)/N + d Σ_{j→i} r(j)/deg⁺(j)

d: 減衰係数（通常0.85）、N: 全ページ数

**行列表現**: r = (1-d)/N × 1 + d × M r
- M: 遷移確率行列（M_{ij} = 1/deg⁺(j) if j→i）
- べき乗法で固有ベクトルを求める

**マルコフ連鎖との関係**:
- ランダムサーファーモデル: 確率 d でリンクを辿り、1-d でランダムジャンプ
- 定常分布 = PageRank ベクトル
- エルゴード性: d < 1 により既約・非周期が保証される

**収束**: べき乗法の収束速度は第2固有値 |λ2| ≤ d に支配される

**実務活用**: 検索エンジン順位付け。ソーシャルネットワークの影響力分析。論文引用ネットワーク。

### 1.5 スペクトラルグラフ理論

**ラプラシアン行列**: L = D - A（D: 次数対角行列、A: 隣接行列）

**正規化ラプラシアン**: L_sym = D^{-1/2} L D^{-1/2} = I - D^{-1/2}AD^{-1/2}

**ラプラシアンの性質**:
- 半正定値（x'Lx = Σ_{(i,j)∈E} (xi-xj)² ≥ 0）
- 最小固有値 = 0（固有ベクトルは定数ベクトル）
- 0固有値の重複度 = 連結成分数
- 第2最小固有値 λ2（フィードラー値）: グラフの連結度の指標

**スペクトラルクラスタリング**:
1. L の固有値分解
2. 下位 k 個の固有ベクトルを特徴量として使用
3. k-means でクラスタリング

**チーガーの不等式**: λ2/2 ≤ h(G) ≤ √(2λ2)
h(G): グラフのチーガー定数（最小カット比）

**実務活用**: コミュニティ検出。画像セグメンテーション。グラフニューラルネットワーク（GNN）の理論基盤。

## 2. 数値計算

### 2.1 浮動小数点演算

**IEEE 754 倍精度**:
(-1)^s × 1.f × 2^{e-1023}
- 符号: 1ビット、指数: 11ビット、仮数: 52ビット
- 精度: 約15-16桁の有効数字
- 範囲: 約 ±10^{308}

**機械イプシロン**: ε_mach ≈ 2.2 × 10^{-16}（1 + ε_mach > 1 となる最小の ε）

**丸め誤差**: fl(a ⊕ b) = (a ⊕ b)(1 + δ)、|δ| ≤ ε_mach

**桁落ち（Catastrophic Cancellation）**:
近い値の引き算で有効桁数が大幅に減少
例: √(x²+1) - x（x が大きいとき）→ 1/(√(x²+1)+x) に変形して回避

**情報落ち**: 絶対値の差が大きい数の加算で小さい方の情報が消失

**オーバーフロー/アンダーフロー**:
- log-sum-exp トリック: log(Σ e^{xi}) = max(x) + log(Σ e^{xi - max(x)})
- softmax の数値安定化に必須

**実務活用**: 機械学習の実装で数値不安定性はバグの温床。損失関数の数値安定な計算。

### 2.2 数値微分

**前進差分**: f'(x) ≈ (f(x+h) - f(x)) / h, 誤差 O(h)

**後退差分**: f'(x) ≈ (f(x) - f(x-h)) / h, 誤差 O(h)

**中心差分**: f'(x) ≈ (f(x+h) - f(x-h)) / (2h), 誤差 O(h²)

**2階微分**: f''(x) ≈ (f(x+h) - 2f(x) + f(x-h)) / h², 誤差 O(h²)

**最適な h の選択**: 打ち切り誤差と丸め誤差のトレードオフ
- 前進/後退差分: h ≈ √ε_mach ≈ 10^{-8}
- 中心差分: h ≈ ε_mach^{1/3} ≈ 10^{-5}

**自動微分（AD）との違い**:
- 数値微分: 近似、丸め誤差あり
- 自動微分: 厳密、計算グラフを利用
- 記号微分: 厳密、式が複雑化しうる

**実務活用**: 勾配の検証（gradient checking）。自動微分が利用できない場合のフォールバック。

### 2.3 数値積分

**台形則**: ∫_a^b f(x)dx ≈ h/2 [f(a) + 2f(x1) + ... + 2f(x_{n-1}) + f(b)]
- 誤差: O(h²)

**シンプソン則**: 2次多項式で近似
∫_a^b f(x)dx ≈ h/3 [f(x0) + 4f(x1) + 2f(x2) + 4f(x3) + ... + f(xn)]
- n は偶数
- 誤差: O(h⁴)（3次多項式まで厳密）

**ガウス求積**: 最適な評価点（ガウス点）と重みで積分
- n点ガウス・ルジャンドル求積: 2n-1次多項式まで厳密
- 台形則やシンプソン則より少ない評価点で高精度

**モンテカルロ積分**: → §4.1

**実務活用**: ベイズ推論の周辺尤度計算。累積分布関数の数値計算。

### 2.4 常微分方程式の数値解法

dy/dt = f(t, y), y(t0) = y0

**オイラー法**: y_{n+1} = y_n + h f(t_n, y_n)
- 1次精度: 局所誤差 O(h²)、大域誤差 O(h)
- 安定性制約: h < 2/|λ|（λ は系の固有値）

**改良オイラー法（ホイン法）**: 2次精度
k1 = f(t_n, y_n)
k2 = f(t_n+h, y_n+hk1)
y_{n+1} = y_n + h/2 (k1 + k2)

**4次ルンゲ・クッタ法（RK4）**: 4次精度（最も広く使用）
k1 = f(t_n, y_n)
k2 = f(t_n+h/2, y_n+hk1/2)
k3 = f(t_n+h/2, y_n+hk2/2)
k4 = f(t_n+h, y_n+hk3)
y_{n+1} = y_n + h/6 (k1 + 2k2 + 2k3 + k4)

**陰的オイラー法**: y_{n+1} = y_n + h f(t_{n+1}, y_{n+1})
- 硬い方程式（stiff equation）に対して安定
- 各ステップで非線形方程式を解く必要あり

**実務活用**: 物理シミュレーション、疫学モデル（SIR等）、金融工学（ブラック・ショールズ方程式）。

### 2.5 連立方程式の反復解法

Ax = b を反復的に解く（大規模疎行列向け）。

**ヤコビ法**:
x_i^{(k+1)} = (b_i - Σ_{j≠i} a_{ij} x_j^{(k)}) / a_{ii}

**ガウス・ザイデル法**:
x_i^{(k+1)} = (b_i - Σ_{j<i} a_{ij} x_j^{(k+1)} - Σ_{j>i} a_{ij} x_j^{(k)}) / a_{ii}

最新の値を即座に使用 → ヤコビ法より速く収束

**収束条件**: A が対角優位（|a_{ii}| > Σ_{j≠i} |a_{ij}|）なら両方とも収束

**共役勾配法（CG）**: 正定値対称行列の場合の最速反復法
- 理論上 n 回以内に収束
- 実用的には前処理で条件数を改善

**実務活用**: 偏微分方程式の離散化後の大規模連立方程式。グラフラプラシアンの方程式。

## 3. フーリエ解析

### 3.1 フーリエ級数

周期 T の関数 f(t) のフーリエ級数展開:
f(t) = a0/2 + Σ_{n=1}^∞ [an cos(2πnt/T) + bn sin(2πnt/T)]

係数:
a0 = (2/T) ∫_0^T f(t) dt
an = (2/T) ∫_0^T f(t) cos(2πnt/T) dt
bn = (2/T) ∫_0^T f(t) sin(2πnt/T) dt

**複素形式**: f(t) = Σ_{n=-∞}^∞ cn e^{i2πnt/T}
cn = (1/T) ∫_0^T f(t) e^{-i2πnt/T} dt

**収束**: f が区分的に滑らかなら各点収束（不連続点ではギブス現象）

### 3.2 フーリエ変換

F(ω) = ∫_{-∞}^∞ f(t) e^{-iωt} dt

逆変換: f(t) = (1/2π) ∫_{-∞}^∞ F(ω) e^{iωt} dω

**性質**:
| 時間域 | 周波数域 |
|--------|---------|
| f(t-a) | e^{-iaω} F(ω)（時間シフト） |
| e^{iat} f(t) | F(ω-a)（周波数シフト） |
| f(at) | (1/|a|)F(ω/a)（スケーリング） |
| f'(t) | iω F(ω)（微分→乗算） |
| f*g (畳み込み) | F·G（畳み込み定理） |

**パーセバルの等式**: ∫|f(t)|² dt = (1/2π)∫|F(ω)|² dω（エネルギー保存）

### 3.3 離散フーリエ変換 (DFT) と FFT

**DFT**: X[k] = Σ_{n=0}^{N-1} x[n] e^{-i2πkn/N}　（k = 0,...,N-1）

**逆DFT**: x[n] = (1/N) Σ_{k=0}^{N-1} X[k] e^{i2πkn/N}

**FFT（高速フーリエ変換）**: DFTを O(N²) → O(N log N) で計算
- クーリー・テューキーのアルゴリズム（分割統治法）
- N = 2^m のときに最も効率的

**周波数分解能**: Δf = 1/(NΔt) = fs/N（fs: サンプリング周波数）

**ナイキスト周波数**: fs/2（これ以上の周波数は折り返し = エイリアシング）

### 3.4 畳み込み定理

**時間域の畳み込み**: (f*g)(t) = ∫ f(τ)g(t-τ)dτ

**フーリエ変換との関係**:
F[f*g] = F[f] · F[g]（畳み込み → 各点乗算）
F[f·g] = (1/2π) F[f] * F[g]（乗算 → 畳み込み）

**実務活用**:
- **CNN**: 畳み込み層はフィルターとの畳み込み演算。FFTで高速化可能。
- **信号処理**: フィルタリング（低域通過、高域通過、帯域通過）
- **画像処理**: ぼかし、エッジ検出、鮮鋭化
- **時系列解析**: パワースペクトル密度の推定（ピリオドグラム）
- **音声処理**: スペクトログラム、メル周波数ケプストラム係数 (MFCC)

## 4. 確率的アルゴリズム

### 4.1 モンテカルロ法

**MC積分**: I = ∫ f(x)p(x)dx ≈ (1/N) Σ f(xi)　（xi ~ p(x)）

誤差: O(1/√N)（次元に依存しない → 高次元で特に有利）

**重点サンプリング（Importance Sampling）**:
I = ∫ f(x)p(x)dx = ∫ f(x)(p(x)/q(x))q(x)dx ≈ (1/N) Σ f(xi)w(xi)

w(xi) = p(xi)/q(xi): 重要度重み、xi ~ q(x)

最適な提案分布: q*(x) ∝ |f(x)|p(x)

**分散削減手法**:
- 対照変量法（Control Variates）
- 層化抽出法（Stratified Sampling）
- アンチセティック変量法

**実務活用**: 金融工学のオプション価格付け。ベイズ推論の周辺尤度推定。物理シミュレーション。

### 4.2 MCMC（マルコフ連鎖モンテカルロ）

目標分布 π(x) からのサンプリング（直接サンプリングが困難な場合）。

**メトロポリス・ヘイスティングス法**:
1. 提案分布 q(x'|x) から候補 x' を生成
2. 受理確率 α = min(1, π(x')q(x|x') / (π(x)q(x'|x))) を計算
3. 確率 α で x' を受理、1-α で現在の x を維持

**ギブスサンプリング**:
各変数を他の変数を固定して条件付き分布からサンプリング:
x_i^{(t+1)} ~ p(xi | x1^{(t+1)}, ..., x_{i-1}^{(t+1)}, x_{i+1}^{(t)}, ..., xn^{(t)})

MH法の特殊ケース（受理確率は常に1）。

**収束診断**:
- バーンイン期間の除去
- トレースプロット
- ゲルマン・ルービン統計量 (R̂ < 1.1)
- 自己相関の確認

**ハミルトニアンモンテカルロ（HMC）**:
勾配情報を利用して効率的にサンプリング。高次元で特に有効。

**実務活用**: ベイズ推論の事後分布サンプリング。統計力学のシミュレーション。Stan/PyMC等のPPL内部。

### 4.3 シミュレーティッドアニーリング（SA）

組合せ最適化問題の近似解法:
1. 初期解 x0、初期温度 T0
2. 近傍解 x' を生成
3. ΔE = f(x') - f(x) を計算
4. ΔE < 0 なら受理。ΔE ≥ 0 なら確率 exp(-ΔE/T) で受理
5. 温度 T を徐々に下げる（冷却スケジュール）

**理論的保証**: 冷却スケジュール T(k) = C/log(k) で大域最適解に収束（ただし実用的には遅い）

**冷却スケジュール**:
- 幾何冷却: T_{k+1} = αT_k（α ∈ [0.9, 0.99]）
- 対数冷却: T_k = C/log(k+1)
- 適応的冷却: 受理率に基づいて調整

**実務活用**: 巡回セールスマン問題、VLSIチップ配置、スケジューリング。

### 4.4 遺伝的アルゴリズム（GA）

生物の進化を模倣した最適化手法:
1. **初期化**: ランダムに個体群（解の集合）を生成
2. **適合度評価**: 各個体の目的関数値を計算
3. **選択**: 適合度に基づいて親個体を選ぶ（ルーレット選択、トーナメント選択）
4. **交叉**: 2つの親から子を生成（一点交叉、二点交叉、一様交叉）
5. **突然変異**: 一定確率で遺伝子を変化（探索の多様性維持）
6. **世代交代**: 新しい個体群で置き換え
7. 2-6 を繰り返す

**パラメータ**: 個体群サイズ、交叉率（0.6-0.9）、突然変異率（0.001-0.01）

**実務活用**: 構造最適化、ニューラルネットワークのアーキテクチャ探索（NAS）、特徴量選択。

### 4.5 粒子群最適化（PSO）

鳥の群れの行動を模倣:
v_i^{t+1} = w v_i^t + c1 r1 (pbest_i - x_i^t) + c2 r2 (gbest - x_i^t)
x_i^{t+1} = x_i^t + v_i^{t+1}

- pbest: 個体の歴代最良位置
- gbest: 群れ全体の歴代最良位置
- w: 慣性係数、c1,c2: 加速係数、r1,r2: 乱数[0,1]

**実務活用**: ニューラルネットワークの重み最適化、関数近似、ロボティクス。

## 5. 微分方程式とデータサイエンス

### 5.1 常微分方程式（ODE）

**1階線形ODE**: dy/dt + p(t)y = q(t)
解: y = e^{-∫p dt}[∫q e^{∫p dt} dt + C]

**定数係数2階ODE**: ay'' + by' + cy = 0
特性方程式: ar² + br + c = 0 の根に応じて:
- 異なる実根 r1, r2: y = C1 e^{r1t} + C2 e^{r2t}
- 重根 r: y = (C1 + C2t) e^{rt}
- 複素根 α ± βi: y = e^{αt}(C1 cos βt + C2 sin βt)

**連立ODE**: dx/dt = Ax
解: x(t) = e^{At} x(0) = P diag(e^{λi t}) P^{-1} x(0)

### 5.2 SIRモデル（疫学）

dS/dt = -βSI
dI/dt = βSI - γI
dR/dt = γI

- S: 感受性者、I: 感染者、R: 回復者
- β: 感染率、γ: 回復率
- 基本再生産数: R0 = β/γ
- R0 > 1: 感染拡大、R0 < 1: 感染収束

**実務活用**: COVID-19の感染予測。ワクチン接種率の閾値推定（集団免疫: 1 - 1/R0）。

### 5.3 偏微分方程式の基礎

**熱方程式**: ∂u/∂t = κ ∂²u/∂x²（拡散現象）
**波動方程式**: ∂²u/∂t² = c² ∂²u/∂x²（振動現象）
**ラプラス方程式**: ∂²u/∂x² + ∂²u/∂y² = 0（定常状態）

**数値解法**:
- 有限差分法（FDM）: 微分を差分で近似
- 有限要素法（FEM）: 弱形式＋基底関数で離散化
- 有限体積法（FVM）: 保存則に基づく

**実務活用**: 物理シミュレーション、画像の拡散フィルタ（異方性拡散によるノイズ除去）。

### 5.4 Neural ODE

**概念**: 残差ネットワーク（ResNet）の連続極限
- ResNet: h_{t+1} = h_t + f(h_t, θ_t)（離散）
- Neural ODE: dh/dt = f(h(t), t, θ)（連続）

**利点**:
- メモリ効率: adjoint法でO(1)メモリ（層数に依存しない）
- 適応的な計算量: ODEソルバーが精度に応じてステップ数を調整
- 連続時間モデル: 不等間隔の時系列データに自然に対応

**adjoint法による勾配計算**:
損失 L の θ に関する勾配を逆方向のODEを解いて計算（バックプロパゲーションの連続版）

**実務活用**: 不規則サンプリングの時系列データ（医療記録等）。正規化フローの連続版（Continuous Normalizing Flows）。生成モデル。

### 5.5 拡散モデル（Diffusion Models）

**前向き過程（ノイズ追加）**:
dx = -½β(t)x dt + √β(t) dW（確率微分方程式、SDE）

**逆向き過程（ノイズ除去）**:
dx = [-½β(t)x - β(t)∇log p_t(x)] dt + √β(t) dW̄

スコア関数 ∇log p_t(x) をニューラルネットワークで学習。

**実務活用**: 画像生成（Stable Diffusion等）。分子設計。音声合成。

## 付録: 計算量の比較（主要アルゴリズム）

| アルゴリズム | 時間計算量 | 空間計算量 | 備考 |
|-------------|-----------|-----------|------|
| 行列乗算 | O(n³) [O(n^{2.37}) 理論] | O(n²) | ストラッセン O(n^{2.81}) |
| 固有値分解 | O(n³) | O(n²) | QRアルゴリズム |
| SVD | O(mn²) (m≥n) | O(mn) | 切り詰めSVD: O(mnk) |
| FFT | O(N log N) | O(N) | DFTの O(N²) から高速化 |
| ダイクストラ | O((V+E)log V) | O(V) | ヒープ使用時 |
| ガウス消去 | O(n³) | O(n²) | LU分解も同様 |
| CG法 | O(n√κ) | O(n) | κ: 条件数 |
| RK4 | O(n×steps) | O(n) | 4回の関数評価/ステップ |

## 付録: 確率的手法の選択ガイド

| 問題 | 推奨手法 | 理由 |
|------|---------|------|
| 高次元積分 | モンテカルロ | 次元の呪いを回避 |
| 事後分布サンプリング | MCMC (HMC) | 勾配利用で高次元に強い |
| 離散組合せ最適化 | SA / GA | 大域的探索 |
| 連続非凸最適化 | PSO / SA | 局所解回避 |
| 大域最適化（低次元） | ベイズ最適化 | サンプル効率が高い |
| ブラックボックス最適化 | GA / CMA-ES | 勾配不要 |
