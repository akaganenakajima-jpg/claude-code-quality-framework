## Apple HIG 連携ガイド

`apple-hig-designer` skill 使用時、心理学的根拠を添えることでデザイン判断の精度が上がる。

| HIG原則 | 対応する心理学法則 | 参照 |
|---|---|---|
| Clarity（明瞭さ） | 認知負荷, 視覚的階層, 段階的開示 | `information.md` §1, §3, §2 |
| Deference（控えめさ） | 選択的注意, バナーブラインドネス | `information.md` §4, §5 |
| Depth（深み） | スキューモーフィズム, プライミング | `impression.md` §9, §6 |
| Feedback（フィードバック） | ドハティ閾値, 労働の錯覚 | `information.md` §8, `impression.md` §5 |
| Consistency（一貫性） | 期待バイアス, デフォルトバイアス | `bias.md` §2, `information.md` §7 |
| Direct Manipulation | 目標勾配効果, ユーザーデライト | `motivation.md` §1, `impression.md` §7 |
