# /5s — 作業開始前の5Sチェック

新しいタスクを始める前に、作業環境を整える。

## 実行内容

### 1. 片付け — 前タスクの残骸チェック
以下のコマンドを実行する:
- Windows: `tasklist | grep -i "node\|python\|wrangler"` でゾンビプロセスを確認
- 不要なバックグラウンドプロセスがあれば報告する

### 2. 現状確認 — 今どこにいるか
以下のコマンドを実行する:
- `git status` — ブランチ名・未コミット変更を確認
- `pwd` — 作業ディレクトリを確認
- 未コミットの変更があれば報告する

### 3. 健康チェック — リソース状況
以下のコマンドを実行する:
- Windows: `wmic cpu get loadpercentage`, `wmic OS get FreePhysicalMemory`, `wmic logicaldisk get size,freespace`
- 異常値（CPU>50%持続、メモリ>80%、ディスク>90%）があれば報告する

### 4. ルール確認 — 設定の最新化
- プロジェクトの CLAUDE.md が読み込まれているか確認
- memory ファイルの最終更新日を確認

### 5. 手順開始
- TodoWrite でタスクリストを作成する
- 必要なら `tasks/` フォルダにタスクファイルを作成する

## 結果報告

全チェック結果を一覧で報告する。異常があれば対処を提案する。
